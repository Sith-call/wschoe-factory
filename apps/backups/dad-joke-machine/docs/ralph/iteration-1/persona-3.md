# 페르소나 3 피드백: 이대학생 (22세, 대학생)

## 화면별 피드백

### 홈 화면 (초기 상태)
- "아재개그 자판기" 제목 + "눌러봐!" 버튼 하나만 있는 심플한 구성 — 깔끔하고 좋음
- 근데 처음 들어오면 화면이 너무 비어 있음. 버튼 하나에 여백이 너무 많아서 "이게 뭐지?" 싶음
- "오늘 0개 봄" 하단 텍스트는 귀여움. 동기 부여가 됨
- 즐겨찾기 아이콘이 우측 상단에 있는데 아이콘이 좀 작아서 처음에 뭔지 몰랐음

### 개그 출제 화면
- "두구두구..." 로딩 애니메이션 — 기대감 들어서 좋음
- 질문 + "탭해서 답 보기" 구성이 직관적. 탭하면 바로 답 나오는 건 만족
- "건너뛰기" 버튼이 있어서 재미없는 문제는 스킵 가능 — 과팅에서 쓸 때 편할 듯
- 근데 개그 카드 영역이 화면 중앙에 너무 작게 몰려 있음. 카드를 좀 더 크게 만들면 좋겠음

### 답 확인 화면
- 답이 나올 때 "즐겨찾기 추가" 별표 아이콘 + "다음!" 버튼 구성 좋음
- 근데 즐겨찾기 추가 눌렀을 때 시각적 피드백이 부족함 — 별이 채워지거나, 토스트 메시지가 뜨거나 해야 "저장됐구나" 알 수 있을 텐데
- "오늘 N개 봄" 카운터가 올라가는 건 소소한 재미

### 즐겨찾기 페이지
- 저장한 개그가 리스트로 잘 나옴. 삭제 버튼도 있음
- 근데 즐겨찾기가 1개밖에 없을 때 화면이 너무 허전함
- 뒤로가기 버튼이 좌측 상단에 있어서 내비게이션은 직관적

## 개선 제안 (우선순위별)

### 1순위 (필수)
1. **개그 수가 너무 적음 (체감)** — 친구들이랑 쓰려면 최소 100개는 있어야 하는데, 빠르게 몇 개 뽑다 보면 반복될 것 같음. 카테고리별 분류(언어유희, 상황극, 영어개그 등)가 있으면 과팅에서 "영어개그만 돌려!" 이렇게 쓸 수 있음
2. **즐겨찾기 추가 시 피드백 없음** — 별 눌렀는데 아무 반응이 없어서 저장된 건지 모르겠음. 별이 채워지는 애니메이션이나 "저장!" 토스트 필요
3. **첫 화면이 너무 밋밋함** — 버튼 하나만 덩그러니 있으면 앱을 처음 깐 사람은 "이게 뭔데?" 싶음. 한 줄 설명이나 예시 개그 하나 보여주면 좋겠음

### 2순위 (있으면 좋음)
4. **공유 기능** — 카카오톡이나 인스타 스토리로 개그를 공유할 수 있으면 과팅 후에도 "아까 그거 뭐였지" 하면서 다시 볼 수 있음. 개그 카드를 이미지로 저장하는 것도 괜찮
5. **다크모드** — 밤에 자기 전에 쓸 때 눈 아픔. 요즘 앱은 다크모드 기본인데
6. **개그 난이도/유형 태그** — "이건 어르신용", "이건 MZ용" 같은 태그가 있으면 상황에 맞게 골라 쓸 수 있음

### 3순위 (재미 요소)
7. **개그 랭킹/투표** — 다른 사람들이 재밌다고 투표한 개그를 볼 수 있으면 좋겠음
8. **연속 재생 모드** — 탭 안 하고 자동으로 슬라이드쇼처럼 넘어가는 모드. 과팅에서 폰 세워놓고 쓸 때
9. **오늘의 개그** — 매일 추천 개그 1개 푸시알림으로 보내주면 습관적으로 들어올 듯

## 만족도 점수: 52/100

## 만족도 근거

**좋은 점 (+)**
- 깔끔하고 광고 없는 UI — 이건 진짜 좋음 (+15)
- 탭해서 답 보는 인터랙션이 직관적 (+10)
- 즐겨찾기 기능 있음 (+5)
- 로딩 애니메이션 "두구두구..." 귀여움 (+5)
- 건너뛰기 기능으로 빠르게 넘길 수 있음 (+5)

**아쉬운 점 (-)**
- 첫 화면이 너무 비어 있어서 앱의 매력을 못 느낌 (-10)
- 개그 종류/카테고리 분류가 없어서 원하는 유형만 골라 볼 수 없음 (-10)
- 즐겨찾기 추가 시 피드백 없음 — UX 기본이 빠져 있음 (-8)
- 공유 기능 없음 — 혼자만 보는 앱이면 금방 지루해짐 (-8)
- 전체적으로 콘텐츠(개그)의 양과 다양성이 부족해 보임 (-7)
- 다크모드 미지원 — 밤에 쓰기 불편 (-5)
- 개그 카드 영역이 화면 대비 너무 작음 — 모바일 최적화 부족 (-5)

**한줄 총평:** 기본기는 있는데, 친구들한테 "이 앱 써봐" 하고 추천하기엔 아직 콘텐츠랑 기능이 부족함. 공유 기능이랑 카테고리만 추가돼도 과팅 필수앱이 될 수 있을 것 같음.
