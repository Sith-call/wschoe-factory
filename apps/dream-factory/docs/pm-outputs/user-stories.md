# 꿈 공장 (Dream Factory) -- 유저 스토리

## Epic 1: 꿈 기록 (Dream Logging)

### US-01: 앱 인트로 및 진입

**AS** 사용자
**I WANT** 앱의 목적을 이해하고 꿈 기록을 시작하거나 과거 꿈을 탐색할 수 있기를
**SO THAT** 별도 설명 없이 핵심 기능에 바로 접근할 수 있다

#### 인수 조건
- [ ] "어젯밤 무슨 꿈 꿨어?" 헤드라인 + 몽환적 별/구름 일러스트
- [ ] "꿈 기록하기" CTA 버튼 -> SceneBuilderScreen 이동
- [ ] "꿈 갤러리" 텍스트 링크 -> GalleryScreen 이동
- [ ] 최근 기록한 꿈 미니 카드 1장 프리뷰 (있을 경우)
- [ ] 데모 모드: 첫 방문 시에도 DB/로그인 없이 즉시 사용
- [ ] 하단 "이번 주 꿈 통계" 요약 뱃지 (기록 수, 주요 키워드)

**Screen:** IntroScreen

---

### US-02: 꿈 장면 구성 (Scene Builder)

**AS** 사용자
**I WANT** 꿈의 핵심 요소(장소, 날씨, 인물, 오브젝트)를 시각적으로 조합할 수 있기를
**SO THAT** 텍스트 없이도 꿈의 분위기를 빠르게 입력할 수 있다

#### 인수 조건
- [ ] 4단계 탭 UI: 장소 -> 날씨 -> 인물 -> 오브젝트
- [ ] **장소** (단일 선택, 8개): 숲, 바다, 도시, 학교, 집, 하늘, 지하, 미지의 장소
- [ ] **날씨** (단일 선택, 6개): 맑음, 흐림, 비, 눈, 안개, 폭풍
- [ ] **인물** (멀티셀렉트 1~3명, 7개): 나, 가족, 연인, 친구, 낯선사람, 유명인, 동물
- [ ] **오브젝트** (멀티셀렉트 1~3개, 8개): 물, 불, 거울, 열쇠, 계단, 시계, 꽃, 문
- [ ] 각 요소는 아이콘 카드 형태, 선택 시 인디고 테두리 + 글로우 효과
- [ ] 상단 "꿈 조각 미리보기" 영역: 선택한 요소들이 실시간으로 아이콘 조합 표시
- [ ] 프로그레스 바: Step 1/3
- [ ] 최소 장소 + 날씨 선택 필수, 나머지는 선택

**Screen:** SceneBuilderScreen

### US-03: 꿈 감정 및 선명도 입력

**AS** 사용자
**I WANT** 꿈에서 느낀 감정과 선명도를 기록할 수 있기를
**SO THAT** 감정 패턴과 꿈의 질을 함께 분석받을 수 있다

#### 인수 조건
- [ ] **꿈 감정** (멀티셀렉트 1~2개, 8개): 평화, 공포, 혼란, 즐거움, 슬픔, 분노, 놀라움, 그리움
- [ ] **선명도 슬라이더** (5단계): 1-흐릿 ~ 5-생생함
- [ ] **한줄 메모** (선택, 80자): "꿈에서 특별히 기억나는 것이 있다면?"
- [ ] 프로그레스 바: Step 2/3
- [ ] 뒤로가기 -> SceneBuilder (선택 상태 유지)

**Screen:** EmotionScreen

---

### US-04: 꿈 해석 로딩

**AS** 사용자
**I WANT** 꿈 데이터가 분석되는 과정을 시각적으로 확인할 수 있기를
**SO THAT** 결과에 대한 기대감과 몰입감을 느낄 수 있다

#### 인수 조건
- [ ] 다크 남색 배경 (deep indigo-black) + 별 파티클 애니메이션
- [ ] 중앙 수정구슬 아이콘 회전 애니메이션
- [ ] 텍스트 순차 전환: "꿈의 조각을 모으는 중..." -> "상징을 해독하는 중..." -> "당신만의 꿈 해석을 완성하는 중..."
- [ ] 3초 후 자동으로 InterpretationScreen 전환
- [ ] 프로그레스 바 없음 (전체 화면 몰입)

**Screen:** AnalysisScreen

## Epic 2: 꿈 해석 (Dream Interpretation)

### US-05: 꿈 해석 카드 결과

**AS** 사용자
**I WANT** 내 꿈 요소 조합에 기반한 해석 카드를 받아볼 수 있기를
**SO THAT** 꿈의 의미를 재미있게 이해하고 공유할 수 있다

#### 인수 조건
- [ ] 꿈 해석 카드 (타로 카드 스타일): 그라디언트 배경 + 상징 아이콘 조합
- [ ] **카드 제목**: 꿈 유형명 동적 생성 (예: "안개 속 열쇠의 꿈", "폭풍의 바다에서 만난 낯선이")
- [ ] **핵심 키워드 3개**: 장소+오브젝트+감정 조합 기반 (예: #변화 #무의식 #갈망)
- [ ] **해석 텍스트** (3~4문장): 조합 매핑 테이블 기반 생성
- [ ] **상징 해설 영역**: 선택한 오브젝트별 1줄 상징 의미 (예: "열쇠 - 새로운 기회의 문")
- [ ] **운세 한마디**: 랜덤 긍정 메시지 (20개 풀)
- [ ] "카드 저장하기" -> localStorage에 저장 + GalleryScreen 이동 옵션
- [ ] "공유하기" 버튼 -> ShareScreen 이동
- [ ] "다시 꾸기" 버튼 -> IntroScreen 복귀

**Screen:** InterpretationScreen

---

### US-06: 꿈 카드 공유

**AS** 사용자
**I WANT** 내 꿈 해석 카드를 SNS 공유 가능한 형태로 볼 수 있기를
**SO THAT** 친구들과 꿈 이야기를 나누고 앱을 알릴 수 있다

#### 인수 조건
- [ ] 공유 전용 카드 뷰: 430x540px 고정 비율 (SNS 최적화)
- [ ] 카드 구성: 꿈 유형명 + 키워드 3개 + 핵심 해석 1문장 + 앱 워터마크
- [ ] 배경 그라디언트: 꿈 감정에 따라 다른 컬러 (평화=보라-청, 공포=진회색-남색, 즐거움=핑크-오렌지 등)
- [ ] "이미지로 저장" 버튼 (html2canvas 또는 유사)
- [ ] "카카오톡 공유" 버튼
- [ ] "링크 복사" 버튼
- [ ] 뒤로가기 -> InterpretationScreen

**Screen:** ShareScreen

## Epic 3: 꿈 갤러리 및 패턴 분석 (Gallery & Patterns)

### US-07: 꿈 갤러리

**AS** 사용자
**I WANT** 지금까지 기록한 꿈들을 카드 갤러리 형태로 탐색할 수 있기를
**SO THAT** 나의 꿈 컬렉션을 시각적으로 감상하고 특정 꿈을 다시 볼 수 있다

#### 인수 조건
- [ ] 2열 그리드 갤러리: 각 카드에 꿈 유형명 + 날짜 + 감정 이모지 + 미니 그라디언트
- [ ] 날짜 역순 정렬 (최신순)
- [ ] 카드 탭 -> 해당 꿈의 InterpretationScreen (읽기 모드)
- [ ] 빈 상태: "아직 기록한 꿈이 없어요. 첫 번째 꿈을 기록해보세요!" + CTA
- [ ] 상단 필터 칩: 전체 / 이번 주 / 감정별 (평화, 공포 등)
- [ ] 데모 모드: 시드 데이터 5개 꿈 카드 제공

**Screen:** GalleryScreen

---

### US-08: 꿈 패턴 대시보드

**AS** 사용자
**I WANT** 내 꿈의 반복 패턴과 통계를 시각적으로 확인할 수 있기를
**SO THAT** 무의식의 경향을 이해하고 자기 탐색에 활용할 수 있다

#### 인수 조건
- [ ] GalleryScreen 상단 탭 전환: "갤러리" | "패턴 분석"
- [ ] **꿈 빈도 차트**: 최근 14일 기록 여부 도트 캘린더 (기록일=인디고, 미기록=회색)
- [ ] **자주 나오는 상징 TOP 5**: 가로 바 차트 (오브젝트+장소 통합 집계)
- [ ] **감정 분포 도넛 차트**: 전체 기록 기준 감정 비율
- [ ] **꿈 선명도 추이**: 최근 7회 기록의 선명도 라인 차트
- [ ] **패턴 인사이트 카드**: 조건부 메시지 (예: "최근 '물' 상징이 자주 등장해요. 감정적 변화의 시기일 수 있어요.")
- [ ] 데이터 부족 시: "5개 이상 기록하면 패턴을 분석해드려요" 안내
- [ ] 데모 모드: 시드 데이터 기반 통계 표시

**Screen:** PatternScreen

---

### US-09: 꿈 일기 확장 메모

**AS** 사용자
**I WANT** 특정 꿈에 대해 깨어난 후 추가 메모를 남길 수 있기를
**SO THAT** 시간이 지나도 꿈의 맥락과 느낌을 잊지 않을 수 있다

#### 인수 조건
- [ ] InterpretationScreen(읽기 모드)에서 "메모 추가" 버튼
- [ ] 메모 입력 바텀시트: 텍스트 영역 (200자 제한)
- [ ] 기존 메모가 있으면 수정 모드로 열림
- [ ] 저장 시 localStorage 업데이트 + 성공 토스트
- [ ] GalleryScreen 카드에 메모 유무 아이콘 표시

**Screen:** InterpretationScreen (메모 바텀시트)

## Epic 4: 시스템 (System)

### US-10: 데모 모드 및 해석 로직

**AS** 개발자/시연자
**I WANT** DB 없이 완전한 서비스 플로우를 체험할 수 있기를
**SO THAT** 백엔드 구축 전에도 앱의 핵심 가치를 검증할 수 있다

#### 인수 조건
- [ ] 모든 데이터 localStorage 기반 (서버 통신 없음)
- [ ] **꿈 해석 매핑 테이블**: 장소(8) x 날씨(6) x 주요감정(8) 조합으로 해석 텍스트 매핑
- [ ] **카드 제목 생성 규칙**: `[날씨 수식어] + [장소] + [오브젝트/인물 기반 서술]` (예: "비 내리는 숲에서 찾은 열쇠")
- [ ] **키워드 매핑**: 각 오브젝트/장소에 고정 키워드 1개, 감정에 키워드 1개 -> 3개 조합
- [ ] **상징 해설 풀**: 오브젝트 8개 x 3가지 해설 변형 = 24개
- [ ] **운세 메시지 풀**: 긍정 메시지 20개 (랜덤 선택)
- [ ] **시드 데이터**: 5일치 샘플 꿈 기록 (다양한 조합)
- [ ] **패턴 인사이트 규칙**: 빈도 기반 조건문 (특정 상징 3회 이상 등장 시 메시지 트리거)

**Screen:** 해당 없음 (전체 앱 로직)

## 스크린 플로우

```
                          [1. IntroScreen]
                         /                \
              [꿈 기록하기]              [꿈 갤러리]
                    |                        |
          [2. SceneBuilderScreen]    [7. GalleryScreen]
           장소->날씨->인물->오브젝트   |           |
                    |              [갤러리]    [패턴 분석]
          [3. EmotionScreen]          |      [8. PatternScreen]
           감정+선명도+메모            |
                    |           [카드 탭]
          [4. AnalysisScreen]         |
           로딩 3초                   |
                    |                 |
          [5. InterpretationScreen] <-+
           해석 카드 + 상징 해설
           |         |        |
      [저장하기] [공유하기] [다시 꾸기]
           |         |        |
    [7. Gallery] [6. Share] [1. Intro]
                     |
              [뒤로] -> [5. Interpretation]
```

**조건부 플로우:**
- SceneBuilder: 장소+날씨 필수, 인물+오브젝트는 선택 (최소 1개씩 권장)
- SceneBuilder 탭 전환: 이전 탭 선택값 유지
- EmotionScreen: 감정 최소 1개 필수, 선명도 기본값 3
- Analysis: 자동 전환 (사용자 액션 없음)
- InterpretationScreen: 신규 모드(기록 직후) vs 읽기 모드(갤러리에서 진입)
- GalleryScreen: 필터에 따른 조건부 렌더링
- PatternScreen: 기록 5개 미만 시 제한된 분석 표시
