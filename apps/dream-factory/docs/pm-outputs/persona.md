## 성격 & 라이프스타일

서연이는 감수성이 풍부하고 자기 탐색에 관심이 많다. 회사에서는 SNS 콘텐츠 기획을 담당하면서도, 퇴근 후에는 타로 유튜브를 보거나 MBTI 관련 밈을 친구들에게 공유하는 게 낙이다. "나"에 대해 알아가는 걸 좋아하지만, 너무 무겁거나 학술적인 건 싫어한다. 가볍고 예쁘게 포장된 자기 탐색 콘텐츠를 선호한다.

꿈을 꽤 자주 꾸는 편이고, 아침에 일어나면 카톡 단톡방에 "야 나 어젯밤에 진짜 이상한 꿈 꿨어ㅋㅋ"라고 올리는 타입. 그런데 텍스트로 쓰려고 하면 기억이 흐려져서 포기하는 경우가 많다.

## 테크 활용도

- **높은 편** — 새 앱을 잘 깔아보지만, 2~3일 안에 매력 없으면 바로 삭제
- 앱스토어 리뷰를 꼼꼼히 읽는 편은 아니고, 인스타 스토리에서 본 걸 바로 깔아봄
- UI가 못생기면 아무리 좋아도 안 씀 ("감성이 없으면 싫어")

## 매일 쓰는 앱

- 카카오톡, 인스타그램, 유튜브
- 당근마켓, 쏘카
- 핀터레스트 (무드보드 수집)
- 포레스트 (집중 타이머)
- 가끔: 타로/운세 앱 (한 달에 2~3번, 설치→삭제 반복)

## 이 앱을 깔아보는 이유

- 인스타에서 누가 올린 꿈 해석 카드가 예뻐서 "이거 뭐야?" 하고 링크 타고 들어옴
- 타로 카드 느낌이라 관심이 감
- 꿈을 텍스트로 쓰지 않아도 되는 게 매력적
- "내 무의식이 뭘 말하는지" 궁금함

## 삭제하는 이유

- 3번 써봤는데 해석이 다 비슷비슷하면 → "뻔하네" → 삭제
- 첫 화면에서 뭘 해야 할지 모르겠으면 → 5초 안에 이탈
- 공유 카드 디자인이 기대 이하면 → 공유 안 하고 흥미 잃음
- 꿈 기록하는 데 3분 이상 걸리면 → "이건 아침에 못 해" → 삭제
- 패턴 분석이 데이터 쌓기 전에는 텅 비어 보이면 → 허무함
