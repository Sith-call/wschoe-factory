## 성격 & 라이프스타일

하은이는 예술적 감수성이 깊고, 일상 속 모든 것에서 창작 소재를 찾는 타입이다. 졸업 전시 준비를 하면서 "나만의 고유한 시각 언어"를 고민하고 있고, 꿈에서 본 장면을 스케치북에 러프하게 그려두는 습관이 있다. 초현실주의(달리, 마그리트) 팬이고, "꿈이야말로 가장 정직한 창작 소스"라고 믿는다.

친구가 적은 편은 아닌데, 깊은 관계를 선호해서 소수와 오래 사귄다. SNS는 인스타 위주지만 올리는 것보다 아카이빙 목적이 더 크다 — 스토리에 자기 작업 과정을 올리고, 피드에는 완성작만 올린다. 타로/오라클 카드에도 관심이 있지만 "맞추는 것"보다 "영감을 주는 것"에 가치를 둔다.

최근 꿈을 텍스트로 기록하는 것에 한계를 느끼고 있다. 꿈의 색감, 공간감, 분위기를 글로는 담을 수 없어서 항상 아쉽다.

## 테크 활용도

- **크리에이티브 유저** — 앱을 도구로 판단하기보다 "이 앱이 주는 경험의 질"로 평가
- 미적 완성도에 매우 예민 — 폰트 커닝, 색상 조합, 여백 비율까지 눈에 들어옴
- 작업에 도움이 되면 유료 앱도 기꺼이 구매하지만, 무의미하게 예쁘기만 한 앱은 금방 삭제
- 비주얼 아카이빙에 집착 — 스크린샷을 폴더별로 정리하고, 레퍼런스로 활용

## 이 앱을 깔아보는 이유

- 인스타 디자이너 친구가 "꿈을 비주얼로 기록하는 앱 신기하다"고 스토리에 올린 걸 보고 관심
- 꿈의 시각적 요소(장소, 날씨, 오브젝트)를 아이콘으로 선택하는 방식이 자기 작업 과정(무드보드 → 키워드 → 비주얼)과 비슷해서 끌림
- 졸업 전시 컨셉을 "꿈의 풍경"으로 잡고 있어서, 꿈 기록 → 패턴 발견 → 작업 소재로 활용하고 싶음
- 해석 카드의 비주얼 퀄리티가 레퍼런스로 쓸 만한지 확인하고 싶음

## 삭제하는 이유

- 해석 카드 비주얼이 "그냥 그라디언트 + 아이콘"이면 → "이걸 레퍼런스라고 저장하긴..." → 흥미 상실
- 컬러 팔레트가 보라색 일변도면 → "색감 다양성이 없네" → 시각적 영감을 못 얻음
- 꿈 기록 과정이 창작의 촉매가 아니라 그냥 선택지 고르기면 → "스케치북이 낫다" → 이탈
- 갤러리가 시간순 나열이면 → "테마별/색감별 모아보기가 안 돼?" → 아카이빙 욕구 미충족
- 패턴 분석이 숫자/차트 위주면 → "나한테는 시각적 맵이 더 와닿는데" → 감성 불일치
