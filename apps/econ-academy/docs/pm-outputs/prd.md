# PRD: 경제 아카데미 (Econ Academy)

## 1. 문제 정의

**"경제 용어를 체계적으로 배울 곳이 없다."**

현재 경제학 학습의 문제:
- 유튜브/블로그: 파편화된 설명, 용어 간 연결 부재
- 교과서: 분량 과다, 피드백 없음, 재미 없음
- 기존 앱: 단순 사전식 나열, 선행 개념 무시, 체계적 경로 없음

핵심 인사이트: 경제 용어는 **의존성 그래프**다. "수요탄력성"을 이해하려면 "수요"를 먼저 알아야 하고, "수요"를 알려면 "시장"과 "가격"을 먼저 알아야 한다. 이 순서를 무시한 학습은 효과가 없다.

## 2. 타겟 유저

| 세그먼트 | 특징 | 핵심 니즈 |
|---------|------|----------|
| 경제학 입문 대학생 | 경제학원론 수강, 용어 암기 필요 | 시험 대비, 빠른 복습 |
| 직장인 (뉴스 독해) | 경제 뉴스를 이해하고 싶음 | 일상 적용, 짧은 학습 |
| 공시생 | 경제학 과목 준비 | 체계적 암기, 반복 학습 |

**1차 타겟**: 경제학원론 수강 대학생 — 100개 용어가 한 학기 커리큘럼과 일치

## 3. 핵심 기능

### 3.1 용어 카드 시스템 (100개)

각 용어 = 하나의 카드:
- **앞면**: 용어 이름 + 한 줄 정의
- **뒷면**: 상세 설명 (2-3문단) + 시각 자료 (다이어그램/그래프)
- **관련 용어 링크**: 선행/후행 용어 연결

### 3.2 학습 경로 (의존성 그래프)

10개 카테고리, 각 10개 용어. 카테고리 간 + 카테고리 내 의존성:

**잠금 규칙**: 카테고리 N을 열려면 선행 카테고리의 용어 중 최소 7개를 "읽기 완료" 이상으로 달성해야 함.

### 3.3 3단계 마스터리

각 용어별 3단계 진행:

```
[1] 읽기 (Read)     → 카드 앞뒤를 읽으면 완료. 기본 이해.
[2] 퀴즈 (Quiz)     → 3문항 퀴즈 통과 (2/3 이상 정답). 지식 확인.
[3] 실험 (Lab)      → 인터랙티브 시뮬레이션 완료 (해당 용어만). 깊은 이해.
```

- 읽기를 통과해야 퀴즈 해금
- 퀴즈를 통과해야 실험 해금 (실험이 있는 용어만)
- 실험이 없는 용어는 퀴즈 통과 = 마스터

### 3.4 퀴즈 시스템

각 용어별 3문항:
- **1문항**: 정의 선택 (4지선다)
- **2문항**: 관계 판단 ("GDP가 증가하면 경제성장률은?" — 참/거짓 또는 선택)
- **3문항**: 적용 시나리오 ("A 상황에서 어떤 개념이 적용되는가?")

오답 시 해설 표시. 재도전 가능 (문제 순서 셔플).

### 3.5 인터랙티브 실험실 (핵심 20개 용어)

### 3.6 진도 추적

- **대시보드**: 전체 100개 중 마스터 수, 카테고리별 진행률
- **오늘의 학습**: 복습 필요한 용어 (마지막 학습 후 일정 시간 경과) + 다음 추천 용어
- **스트릭**: 연속 학습 일수

### 3.7 간격 반복 (Spaced Repetition)

- 읽기 완료 후 1일, 3일, 7일, 14일, 30일 간격으로 복습 알림
- 복습 = 퀴즈 재풀이 (이미 통과한 퀴즈도 간격 반복 대상)
- 복습 미응시 시 마스터리 단계 유지하되, 대시보드에 "복습 필요" 표시

## 4. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | React 18 + TypeScript | 컴포넌트 기반, 타입 안전 |
| 빌드 | Vite | 빠른 HMR, 가벼움 |
| 스타일링 | Tailwind CSS (CDN) | 빌드 없는 유틸리티 CSS |
| 상태 관리 | React Context + useReducer | 외부 라이브러리 불필요 |
| 데이터 저장 | localStorage | 서버 없음, 즉시 동작 |
| 차트/시각화 | SVG 직접 렌더링 | 의존성 최소화 |
| 배포 | 정적 호스팅 (Vite build) | 서버 불필요 |

## 5. 데모 모드

- 서버 없음. 모든 데이터는 localStorage에 저장.
- 첫 방문 시 100개 용어 데이터 자동 로드 (하드코딩된 JSON)
- 진도/마스터리/복습 일정 모두 localStorage 관리
- 데이터 초기화 버튼 제공 (Settings)

## 6. 비기능 요구사항

- **성능**: 첫 로드 2초 이내, 카드 전환 100ms 이내
- **접근성**: 모바일 우선 반응형 (375px ~ 1440px)
- **오프라인**: localStorage 기반이므로 최초 로드 후 오프라인 사용 가능
- **데이터 크기**: 100개 용어 JSON < 200KB

## 9. 성공 기준

1. 100개 용어 전체가 카드+퀴즈로 학습 가능
2. 20개 핵심 용어에 인터랙티브 실험 제공
3. 의존성 기반 학습 경로가 정상 작동 (잠금/해금)
4. 간격 반복으로 복습 유도
5. Ralph Loop 페르소나 전원 80%+ 만족도 달성
