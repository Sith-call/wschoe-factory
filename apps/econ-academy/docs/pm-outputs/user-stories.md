# 유저 스토리: 경제 아카데미

## 온보딩

### US-01: 첫 방문 대시보드
**As a** 처음 방문한 사용자
**I want to** 앱의 전체 학습 구조를 한눈에 파악하고 싶다
**So that** 100개 용어를 어떻게 배워나갈지 감을 잡을 수 있다

**수락 조건**:
- 대시보드에 전체 진도 요약 표시 (0/100 마스터)
- 10개 카테고리 목록이 보이고, 첫 번째 카테고리만 해금 상태
- "학습 시작" 버튼이 첫 번째 카테고리의 첫 용어로 안내

### US-02: 카테고리 탐색
**As a** 학습자
**I want to** 10개 카테고리와 각 카테고리의 잠금/해금 상태를 확인하고 싶다
**So that** 어디까지 진행했고 다음에 뭘 배울지 알 수 있다

**수락 조건**:
- 해금된 카테고리: 진행률 표시 (예: 3/10 마스터)
- 잠금된 카테고리: 잠금 아이콘 + 해금 조건 표시 ("경제학 기초 원리 7개 이상 읽기 완료 필요")
- 카테고리 탭 시 해당 용어 목록으로 이동

---

## 용어 학습

### US-03: 용어 목록 확인
**As a** 학습자
**I want to** 카테고리 내 10개 용어의 마스터리 상태를 보고 싶다
**So that** 어떤 용어를 공부할지 선택할 수 있다

**수락 조건**:
- 각 용어 옆에 마스터리 단계 아이콘 (미학습 / 읽기완료 / 퀴즈통과 / 마스터)
- 의존성 순서대로 정렬 (위에서 아래로)
- 선행 용어 미완료 시 잠금 표시

### US-04: 용어 카드 읽기
**As a** 학습자
**I want to** 용어 카드의 앞면(한 줄 정의)과 뒷면(상세 설명 + 시각자료)을 보고 싶다
**So that** 용어의 기본 개념을 이해할 수 있다

**수락 조건**:
- 카드 앞면: 용어 이름 + 한 줄 정의
- 카드 뒷면: 2-3문단 설명 + 다이어그램/그래프
- 카드를 끝까지 스크롤하면 "읽기 완료" 버튼 활성화
- 읽기 완료 후 마스터리 1단계로 업데이트

### US-05: 관련 용어 탐색
**As a** 학습자
**I want to** 현재 용어와 연결된 선행/후행 용어를 확인하고 싶다
**So that** 개념 간 관계를 이해하고 다음 학습으로 넘어갈 수 있다

**수락 조건**:
- 카드 하단에 "관련 용어" 섹션
- 선행 용어 (이 용어를 이해하려면 필요한 것)
- 후행 용어 (이 용어를 알면 배울 수 있는 것)
- 해금된 용어는 탭하면 해당 카드로 이동

## 퀴즈

### US-06: 용어 퀴즈 풀기
**As a** 읽기를 완료한 학습자
**I want to** 3문항 퀴즈로 이해도를 확인하고 싶다
**So that** 용어를 진짜 이해했는지 검증할 수 있다

**수락 조건**:
- 읽기 완료 후 퀴즈 버튼 활성화
- 3문항: 정의 선택, 관계 판단, 시나리오 적용
- 2/3 이상 정답 시 퀴즈 통과
- 오답 시 해설 표시

### US-07: 퀴즈 재도전
**As a** 퀴즈에서 탈락한 학습자
**I want to** 바로 재도전하고 싶다
**So that** 틀린 부분을 고쳐서 통과할 수 있다

**수락 조건**:
- 탈락 후 "다시 풀기" 버튼 표시
- 재도전 시 문제 순서 셔플
- 재도전 횟수 제한 없음

---

## 실험실

### US-08: 인터랙티브 실험 진행
**As a** 퀴즈를 통과한 학습자
**I want to** 인터랙티브 시뮬레이션으로 개념을 체험하고 싶다
**So that** 이론을 넘어 직관적으로 이해할 수 있다

**수락 조건**:
- 퀴즈 통과 후 실험 버튼 활성화 (해당 용어에 실험이 있는 경우)
- 슬라이더/드래그 등으로 변수 조절
- 결과가 실시간으로 시각적 변화
- 실험 완료 조건 달성 시 마스터 3단계 부여

### US-09: 실험 없는 용어 마스터
**As a** 퀴즈를 통과한 학습자 (실험 없는 용어)
**I want to** 퀴즈 통과만으로 마스터 인정받고 싶다
**So that** 모든 용어를 공평하게 마스터할 수 있다

**수락 조건**:
- 실험이 없는 80개 용어는 퀴즈 통과 = 마스터
- 마스터 달성 시 축하 피드백

## 진도 추적

### US-10: 대시보드에서 진도 확인
**As a** 학습자
**I want to** 전체 100개 용어 중 몇 개를 마스터했는지 한눈에 보고 싶다
**So that** 학습 동기를 유지할 수 있다

**수락 조건**:
- 전체 마스터 수 / 100 표시
- 카테고리별 진행률 바
- 최근 학습한 용어 3개 표시

### US-11: 학습 스트릭 확인
**As a** 매일 학습하는 사용자
**I want to** 연속 학습 일수를 확인하고 싶다
**So that** 꾸준히 학습할 동기가 생긴다

**수락 조건**:
- 대시보드에 연속 학습 일수 표시
- 오늘 학습 완료 시 스트릭 +1

### US-12: 상세 진도 통계
**As a** 학습 중반의 사용자
**I want to** 카테고리별, 단계별 상세 통계를 보고 싶다
**So that** 약한 영역을 파악하고 집중할 수 있다

**수락 조건**:
- Progress 화면에서 카테고리별 마스터리 분포 (읽기/퀴즈/마스터)
- 퀴즈 정답률 통계
- 가장 많이 틀린 용어 TOP 5

## 복습 (간격 반복)

### US-13: 오늘의 복습 목록
**As a** 이전에 학습한 용어가 있는 사용자
**I want to** 오늘 복습해야 할 용어 목록을 보고 싶다
**So that** 잊기 전에 복습할 수 있다

**수락 조건**:
- 대시보드 "오늘의 복습" 섹션에 복습 대상 용어 표시
- 복습 간격: 1일, 3일, 7일, 14일, 30일
- 복습 = 퀴즈 재풀이

### US-14: 복습 완료
**As a** 복습을 마친 학습자
**I want to** 복습 완료 표시가 되고 다음 복습 일정을 알고 싶다
**So that** 꾸준한 간격 반복을 유지할 수 있다

**수락 조건**:
- 복습 퀴즈 통과 시 다음 복습 간격으로 예약
- 복습 퀴즈 실패 시 간격 리셋 (1일 후 다시)
- "복습 필요" 뱃지 제거

## 설정 및 기타

### US-15: 데이터 초기화
**As a** 처음부터 다시 시작하고 싶은 사용자
**I want to** 모든 학습 데이터를 초기화하고 싶다
**So that** 깨끗한 상태에서 다시 학습할 수 있다

**수락 조건**:
- Settings 화면에 "데이터 초기화" 버튼
- 확인 모달 ("정말 초기화하시겠습니까?")
- 초기화 후 대시보드로 이동 (0/100 상태)

### US-16: 다음 추천 용어
**As a** 어떤 용어를 공부할지 모르는 학습자
**I want to** 앱이 다음에 학습할 용어를 추천해주길 원한다
**So that** 최적의 순서로 학습할 수 있다

**수락 조건**:
- 대시보드 "다음 학습" 카드에 추천 용어 1개 표시
- 추천 로직: 해금된 용어 중 마스터리가 가장 낮은 것 우선
- 탭하면 해당 용어 카드로 이동

### US-17: 카테고리 해금 알림
**As a** 선행 카테고리를 거의 완료한 학습자
**I want to** 새 카테고리가 해금되었을 때 알림을 받고 싶다
**So that** 새로운 영역을 탐색할 수 있다

**수락 조건**:
- 카테고리 해금 조건 달성 시 토스트 알림 ("시장과 가격 카테고리가 해금되었습니다!")
- 대시보드에 "새로 해금" 뱃지 표시

### US-18: 용어 검색
**As a** 특정 용어를 빠르게 찾고 싶은 사용자
**I want to** 용어 이름으로 검색하고 싶다
**So that** 카테고리를 탐색하지 않고도 원하는 용어에 접근할 수 있다

**수락 조건**:
- 상단 검색 바에 용어 이름 입력
- 실시간 자동완성 (2글자 이상)
- 잠금된 용어도 검색 결과에 표시 (단, 탭 시 "선행 용어 학습 필요" 안내)
