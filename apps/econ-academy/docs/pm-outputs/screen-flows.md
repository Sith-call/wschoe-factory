# 화면 설계: 경제 아카데미

## 화면 목록

| # | 화면 ID | 화면 이름 | 설명 |
|---|---------|----------|------|
| 1 | Home | 대시보드 | 진도 요약 + 오늘의 학습/복습 |
| 2 | CategoryList | 카테고리 목록 | 10개 카테고리 + 잠금/해금 상태 |
| 3 | TermList | 용어 목록 | 카테고리 내 10개 용어 + 마스터리 |
| 4 | TermCard | 용어 카드 | 정의 + 설명 + 시각자료 |
| 5 | Quiz | 퀴즈 | 3문항 퀴즈 + 결과 |
| 6 | Lab | 실험실 | 인터랙티브 시뮬레이션 |
| 7 | Progress | 진도 통계 | 전체 마스터리 통계 |
| 8 | Search | 검색 | 용어 검색 + 자동완성 |
| 9 | Settings | 설정 | 데이터 초기화 등 |

## 화면 전환 플로우

```
                    ┌─────────┐
                    │  Home   │
                    └────┬────┘
                         │
              ┌──────────┼──────────┐
              │          │          │
     ┌────────▼──┐  ┌────▼────┐  ┌─▼────────┐
     │CategoryList│  │Progress │  │ Settings │
     └────────┬──┘  └─────────┘  └──────────┘
              │
     ┌────────▼──┐
     │ TermList   │
     └────────┬──┘
              │
     ┌────────▼──┐
     │ TermCard   │──────────┐
     └────────┬──┘          │
              │             │
     ┌────────▼──┐  ┌───────▼──┐
     │   Quiz    │  │  Search  │
     └────────┬──┘  └──────────┘
              │
     ┌────────▼──┐
     │    Lab    │
     └──────────┘
```

**핵심 플로우**: Home → CategoryList → TermList → TermCard → Quiz → Lab

**보조 플로우**:
- Home → 오늘의 복습 → Quiz (복습 모드)
- Home → 다음 추천 → TermCard
- CategoryList → Search (상단 검색 아이콘)
- 어디서든 하단 탭으로 Home / CategoryList / Progress / Settings 이동

## 네비게이션 구조

**하단 탭 바 (항상 표시)**:
| 탭 | 화면 | 아이콘 |
|---|------|-------|
| 홈 | Home | 집 모양 |
| 카테고리 | CategoryList | 그리드 모양 |
| 진도 | Progress | 차트 모양 |
| 설정 | Settings | 톱니바퀴 |

**상단 바**:
- 뒤로 가기 (← ) — TermList, TermCard, Quiz, Lab, Search
- 검색 아이콘 — CategoryList 화면 우상단

## 반응형 브레이크포인트

| 구간 | 너비 | 레이아웃 |
|------|------|---------|
| 모바일 | 375px - 767px | 단일 컬럼, 하단 탭 |
| 태블릿 | 768px - 1023px | 2컬럼 그리드 (카테고리, 용어 목록) |
| 데스크톱 | 1024px+ | 3컬럼 (사이드바 + 메인 + 상세) |

모바일 우선 설계. 태블릿/데스크톱은 공간 활용만 변경하고 기능은 동일.
