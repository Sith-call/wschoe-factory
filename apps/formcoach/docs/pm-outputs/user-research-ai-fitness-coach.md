# User Research Report: AI 기반 개인 피트니스 코치 앱

**날짜**: 2026-03-18
**데이터 소스**: 탐색적 리서치 (제품 설명 기반 가설 — 인터뷰/설문으로 검증 필요)
**신뢰도**: 🟡 가설 수준 — 정성/정량 리서치로 검증 권장

---

## Executive Summary

20-40대 헬스장 초보~중급자는 "무엇을 해야 할지 모르는 막막함", "자세가 맞는지 불안함", "혼자 하면 작심삼일" 세 가지 핵심 고충을 공유한다. 이들은 PT 비용 부담(월 40-80만원)과 유튜브/앱의 비개인화된 콘텐츠 사이에서 **"합리적 가격의 개인화된 가이드"**라는 미충족 수요를 갖고 있다. 가장 높은 가치 세그먼트는 "목표 지향 자기개발러"이며, 가장 높은 성장 세그먼트는 "재시작 다이어터"이다. 고객 여정에서 가장 큰 이탈 지점은 온보딩 직후(1-2주차)와 무료→유료 전환 시점이다.

---

## 페르소나 (Personas)

### 페르소나 1: 김지현 (28세, 직장인) — "운동하고 싶은데 뭘 해야 할지 모르겠어요"

| 항목 | 내용 |
|------|------|
| **누구** | 마케팅 회사 3년차, 주 5일 근무, 퇴근 후 또는 점심시간에 운동 시도 중. 헬스장 등록 3개월째, 러닝머신만 30분 뛰다 옴 |
| **Primary JTBD** | "헬스장에 갔을 때, 내 수준에 맞는 효과적인 운동 루틴을 알고 싶다. 그래야 시간 낭비 없이 원하는 몸을 만들 수 있으니까." |
| **핵심 고충 (Pains)** | ① 기구 사용법을 몰라 민폐 끼칠까 두려움 ② 유튜브로 찾아도 "나한테 맞는 건지" 확신 없음 ③ PT는 월 40만원+, 부담스러움 |
| **핵심 이득 (Gains)** | "오늘 이거 하세요"라고 명확히 알려주는 것, 운동 후 성취감, 체형 변화가 눈에 보이는 것 |
| **행동 패턴** | 인스타그램에서 운동 릴스 저장 → 헬스장에서 따라하기 시도 → 포기 → 다시 검색 반복. 앱 다운로드 잘 하지만 2주 내 삭제 |
| **기술 친화도** | 높음 (앱 적극 사용, 새로운 앱에 거부감 없음) |
| **추정 비율** | 전체 타겟의 ~40% |

**핵심 인사이트**: 이 페르소나는 "정보 과부하" 상태이다. 콘텐츠는 넘치지만 자기에게 맞는 것을 골라주는 "큐레이션"을 원한다. AI 운동 계획의 핵심 가치가 여기에 있다.

### 페르소나 2: 박성훈 (34세, IT 개발자) — "효율적으로 운동하고 싶은데, 자세가 맞는 건지 불안해요"

| 항목 | 내용 |
|------|------|
| **누구** | 스타트업 백엔드 개발자, 데이터 기반 의사결정 선호. 헬스장 1년+, 혼자 웨이트 트레이닝 하지만 정체기 느끼는 중 |
| **Primary JTBD** | "웨이트 트레이닝을 할 때, 내 자세가 정확한지 실시간으로 확인하고 싶다. 그래야 부상 없이 효율적으로 근력을 키울 수 있으니까." |
| **핵심 고충 (Pains)** | ① 데드리프트/스쿼트 자세가 맞는지 불안 (허리 부상 걱정) ② 혼자 하니까 피드백이 없어 발전이 느림 ③ 운동 기록을 엑셀/노션에 수동 입력, 번거로움 |
| **핵심 이득 (Gains)** | 데이터로 진행 상황 확인, 정확한 자세로 하는 확신, 효율적인 분할 루틴 |
| **행동 패턴** | 운동 앱 2-3개 병행 사용 (기록용 + 타이머 + 루틴). 유튜브 "올바른 데드리프트 자세" 반복 시청. 데이터/그래프에 동기부여 받음 |
| **기술 친화도** | 매우 높음 (개발자, 새로운 기술에 호기심) |
| **추정 비율** | 전체 타겟의 ~25% |

**핵심 인사이트**: 이 페르소나는 **카메라 자세 교정의 얼리 어답터**가 될 가능성이 높다. 기술에 대한 관용도가 높고, 자세 교정의 가치를 가장 강하게 느낀다. 다만, 기술 정확도에 대한 기대치도 높다.

### 페르소나 3: 이수연 (32세, 육아맘) — "아이 재우고 나면 시간이 없어요, 집에서 빠르게 하고 싶어요"

| 항목 | 내용 |
|------|------|
| **누구** | 3살 아이 엄마, 출산 후 체중 회복이 목표. 헬스장 갈 시간이 없어 홈트레이닝 선호. 과거 PT 경험 있으나 현재 여건상 불가 |
| **Primary JTBD** | "아이가 잠든 30-40분 동안, 내 목표(체중 감량)에 맞는 홈트레이닝을 하고 싶다. 그래야 육아 중에도 건강을 관리할 수 있으니까." |
| **핵심 고충 (Pains)** | ① 시간이 불규칙하고 짧음 (20-40분) ② 홈트 자세가 맞는지 피드백 받을 방법이 없음 ③ 혼자 하면 의지가 약해짐, 동기부여 필요 |
| **핵심 이득 (Gains)** | 짧은 시간에 효과적인 운동, 자세 확인으로 안전한 운동, 눈에 보이는 체중/체형 변화 |
| **행동 패턴** | 유튜브 홈트 영상 따라하기 → 같은 영상 반복하다 지루해짐 → 새 영상 탐색 → 루틴 깨짐. 카카오톡 육아맘 그룹에서 운동 정보 공유 |
| **기술 친화도** | 중간 (앱 사용하지만 복잡한 설정은 싫어함) |
| **추정 비율** | 전체 타겟의 ~20% |

**핵심 인사이트**: 이 페르소나는 **시간 유연성**이 핵심이다. "30분 운동", "20분 운동" 등 시간 기반 계획 생성이 결정적 기능이 된다. 또한 소셜 커뮤니티(카카오톡 그룹 연동)가 리텐션에 크게 기여할 수 있다.

### 행동 기반 세그먼트

| 세그먼트 | 규모 (추정) | Primary JTBD | 제품 적합도 | 지불 의향 | 성장성 |
|---------|------------|-------------|-----------|----------|--------|
| **🎯 목표 지향 자기개발러** | 30% | 체계적이고 효율적인 운동으로 가시적 성과 달성 | 🟢 매우 높음 | 🟢 높음 (월 14,900원+) | 🟡 중간 |
| **🔄 재시작 다이어터** | 35% | 작심삼일을 극복하고 이번에는 꾸준히 운동하기 | 🟢 높음 | 🟡 중간 (월 9,900원) | 🟢 높음 |
| **🏠 시간 부족 홈트러** | 20% | 제한된 시간/공간에서 최대 효과의 운동하기 | 🟡 중간 | 🟡 중간 (월 9,900원) | 🟡 중간 |

### 세그먼트 상세

#### 🎯 목표 지향 자기개발러 — **최고 가치 세그먼트**
- **특성**: 명확한 목표(여름 전 -5kg, 벤치프레스 80kg 등), 계획적, 데이터 좋아함
- **페르소나 매핑**: 박성훈 (70%), 김지현 (30%)
- **핵심 기능 수요**: 적응형 운동 계획, 진행 상황 대시보드, 자세 교정
- **전환 트리거**: "내 데이터를 분석해서 최적 계획을 만들어준다" → 즉시 시도
- **이탈 트리거**: AI 추천이 너무 쉽거나 비현실적일 때

#### 🔄 재시작 다이어터 — **최고 성장 세그먼트**
- **특성**: 운동 시도 횟수 3회+, 매번 2주 내 중단, 동기부여가 핵심 니즈
- **페르소나 매핑**: 김지현 (60%), 이수연 (40%)
- **핵심 기능 수요**: 게이미피케이션, AI 음성 격려, 쉬운 시작(10분 운동부터)
- **전환 트리거**: "이번에는 다를 거야" 순간 + 친구 추천
- **이탈 트리거**: 3일 연속 미운동 시 "이미 망했다" 심리

#### 🏠 시간 부족 홈트러
- **특성**: 육아/야근으로 시간 제약, 홈트레이닝 선호, 기구 최소 필요
- **페르소나 매핑**: 이수연 (90%), 김지현 (10%)
- **핵심 기능 수요**: 시간 기반 운동 계획(15/20/30분), 기구 없는 운동, 조용한 운동 옵션
- **전환 트리거**: "내 상황(시간/장소)에 맞춰준다"
- **이탈 트리거**: 카메라 설치 번거로움, 운동 소음 문제

#### 💪 자세 집착 중급자
- **특성**: 웨이트 1년+, 자세 교정에 높은 가치 부여, 기꺼이 프리미엄 지불
- **페르소나 매핑**: 박성훈 (100%)
- **핵심 기능 수요**: 실시간 자세 교정, 운동별 상세 폼 분석, 진행 통계
- **전환 트리거**: "카메라로 실시간 자세 교정" 데모 영상
- **이탈 트리거**: 자세 인식 정확도 미달

 😐      🤔      😊/😰     💪/😫      🎉      📢

### 상세 여정

| 단계 | 터치포인트 | 감정 | 고충 (Pain Points) | 기회 (Opportunities) |

| **온보딩** (Onboarding) | 앱 설치, 체력 테스트, 목표 설정, 첫 운동 | 😊→😰 설렘에서 "생각보다 복잡하네"로 전환 가능 | ① 체력 테스트가 길거나 어려우면 이탈 ② 카메라 설정이 번거로움 ③ 첫 운동이 너무 쉽거나 어려움 | ✅ **Aha Moment**: 첫 운동에서 "자세 좋아요!" 피드백 → 즉각적 가치 경험 ✅ 온보딩 3분 이내 완료 + 바로 첫 운동 시작 |
| **활성 사용 (1-2주)** | 매일 운동 계획 확인, 운동 수행, 자세 피드백 | 💪→😫 "신기하다!"에서 "귀찮다"로 전이 위험 | ① ⚠️ **최대 이탈 구간**: 3-7일차에 루틴 깨짐 ② AI 추천이 반복적이면 흥미 감소 ③ 카메라 설치 매번 하기 번거움 | ✅ 3일 연속 완료 시 강력한 보상 (게이미피케이션) ✅ "오늘 건너뛰어도 괜찮아요, 내일 계획 조정했어요" 적응형 메시지 |
| **활성 사용 (3-4주)** | 진행 리포트 확인, 난이도 상향, 새 운동 추가 | 💪 "변화가 보이기 시작한다" 확신 | ① 무료 체험 종료 → 유료 전환 결정 ② 정체기 느끼면 의욕 저하 | ✅ **Aha Moment #2**: 첫 주간 리포트에서 "이번 주 스쿼트 무게 5kg 증가" 가시적 성과 ✅ ⚠️ **무료→유료 전환 시점**: 3주간 데이터 축적 후 "이 데이터를 유지하려면" |
| **확장** (Expansion) | 프리미엄 기능, 식단 연동, 친구 초대 | 🎉 "이 앱 없이 운동 못 해" 의존성 형성 | ① 프리미엄 기능의 추가 가치가 명확하지 않으면 전환 안 됨 ② 콘텐츠 업데이트가 느리면 지루함 | ✅ 친구 초대 시 1개월 무료 (바이럴 루프) ✅ 월간 챌린지로 지속 참여 유도 |
| **옹호** (Advocacy) | SNS 공유, 리뷰 작성, 친구 추천 | 📢 "이거 진짜 좋아" 자발적 추천 | ① 공유할 만한 "자랑거리"가 없으면 추천 안 함 | ✅ "3개월 변화 리포트" 인스타그램 공유 템플릿 ✅ 운동 기록 카드 (예쁜 디자인) SNS 공유 기능 |

### 핵심 이탈 지점 (Critical Drop-off Points)

```
1위: 🔴 온보딩 → 활성 사용 1주차 (추정 이탈률: 50-60%)
     원인: "첫 운동 경험"이 기대에 미치지 못함, 카메라 설정 포기
     대응: 카메라 없이도 기본 기능 사용 가능하게 설계

2위: 🔴 활성 사용 2-3주차 무료→유료 전환 (추정 전환율: 5-15%)
     원인: 무료 대안 풍부, 지불 가치 불확실
     대응: 3주간 축적된 개인 데이터의 "손실 회피" 활용

3위: 🟡 활성 사용 1-2개월 장기 리텐션 (추정 이탈률: 30-40%/월)
     원인: 운동 루틴이 반복적, 목표 달성 후 이탈
     대응: 새로운 목표 자동 제안, 소셜 챌린지
```

## 핵심 인사이트 (Key Insights)

### 1. "정보 과부하가 문제이지, 정보 부족이 아니다"
운동 콘텐츠는 유튜브/인스타에 넘친다. 사용자의 진짜 고충은 **"내 상황에 맞는 것을 골라주는 것"**이다. AI의 가치는 '새로운 운동'이 아니라 '내게 맞는 운동의 큐레이션'에 있다.

### 2. "자세 교정은 Want이 아니라 Need — 하지만 편의성이 관건"
자세 교정의 가치는 높지만, "매번 카메라를 세팅하는 번거로움"이 사용 빈도를 결정한다. **카메라 없이도 핵심 가치(운동 계획)를 경험할 수 있어야** 하며, 자세 교정은 "보너스 기능"으로 포지셔닝해야 이탈을 막을 수 있다.

### 3. "첫 운동의 'Aha Moment'가 리텐션을 결정한다"
온보딩에서 활성 사용으로의 전환은 **첫 운동 경험의 질**에 달려 있다. "자세 좋아요!", "이 운동이 당신에게 맞는 이유는..." 같은 즉각적 개인화 피드백이 "이 앱은 다르다"는 인식을 만든다.

### 4. "PT 비용 대비 가격 프레이밍이 전환의 핵심"
타겟 사용자는 PT(회당 5-8만원)를 "비싸지만 효과적"으로 인식한다. "PT 1회 비용 < 한 달 구독"이라는 프레이밍이 지불 결정의 가장 강력한 넛지가 될 수 있다.

### 5. "세그먼트별 진입 전략이 달라야 한다"
- **목표 지향 자기개발러**: 데이터/성과 기반 마케팅 ("AI가 분석한 최적 루틴")
- **재시작 다이어터**: 감성/동기부여 마케팅 ("이번에는 포기하지 않는 운동")
- **시간 부족 홈트러**: 편의성 마케팅 ("20분이면 충분해요")

## 권장 사항 (Recommendations)

### 즉시 실행 (Week 1-2)
1. **페르소나 검증 인터뷰 20명 실시** — 위 3개 페르소나의 JTBD와 고충이 실제로 존재하는지 확인
2. **"목표 지향 자기개발러"를 비치헤드 세그먼트로 설정** — 가장 높은 지불 의향 + 제품 적합도 + 얼리 어답터 가능성

### 단기 (Week 3-4)
3. **카메라 자세 교정을 "핵심 기능"이 아닌 "프리미엄 부가 기능"으로 포지셔닝** — 카메라 없이도 핵심 가치를 경험할 수 있게 설계
4. **첫 운동 경험 최적화에 집중** — 온보딩 3분 이내 + 첫 운동에서 즉각적 개인화 피드백

### 중기 (Month 2-3)
5. **3일/7일/14일 리텐션 게이미피케이션 시스템 설계** — 재시작 다이어터의 "작심삼일" 패턴에 직접 대응
6. **시간 기반 운동 계획 옵션 추가** — 홈트러 세그먼트 수용 (15분/20분/30분/45분)
