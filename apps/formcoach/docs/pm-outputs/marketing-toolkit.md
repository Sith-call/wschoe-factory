### 추천: **폼코치 (FormCoach)**
- 핵심 차별점(자세 교정)을 이름에서 바로 전달
- 짧고 발음하기 쉬우며, 앱스토어 검색에 유리
- 글로벌 확장 시에도 그대로 사용 가능
- "폼코치가 봐줄게" 같은 자연스러운 마케팅 문구 파생 가능

**차순위 추천: 자세히 (Jasehi)** — 한국 시장 한정이라면 언어유희가 강력한 브랜딩 요소

## 2. 마케팅 캠페인 아이디어

### 캠페인 1: "자세가 답이다" 챌린지

| 항목 | 내용 |
|------|------|
| **채널** | 인스타그램 릴스 / 틱톡 |
| **메시징** | "당신의 스쿼트, 몇 점일까요?" — AI 자세 점수를 공유하는 바이럴 챌린지 |
| **구체적 실행** | 사용자가 앱으로 스쿼트/데드리프트 자세를 촬영 → AI가 100점 만점으로 채점 → 점수 카드를 SNS에 공유 → 친구 태그하여 챌린지 전파 |
| **참여 근거** | 게이미피케이션 + 경쟁심리 + SNS 인증 문화 활용. 운동 초보자의 "내 자세 맞나?" 불안감을 재미있게 전환 |
| **노력** | 중 (자세 점수 카드 UI 개발 + SNS 공유 기능) |
| **기대 효과** | 높음 — 바이럴 계수 1.5 이상 목표, 앱 다운로드 직접 유도 |

### 캠페인 2: "PT비 100만원 아끼는 법" 콘텐츠 시리즈

| 항목 | 내용 |
|------|------|
| **채널** | 유튜브 / 네이버 블로그 |
| **메시징** | "PT 12회 = 100만원. 같은 효과를 월 9,900원에." — 비용 비교 콘텐츠 |
| **구체적 실행** | 실제 PT 트레이너와 AI 앱을 동시에 사용하는 4주 실험 콘텐츠. 자세 교정 정확도, 운동 효과, 비용을 객관 비교. 유튜버 1-2명과 협업 |
| **참여 근거** | "돈 아끼는 꿀팁" 콘텐츠는 직장인 타겟에게 강력한 클릭 유인. 실험 포맷으로 신뢰도 확보 |
| **노력** | 고 (4주 촬영 + 편집 + 유튜버 협업비) |
| **기대 효과** | 높음 — SEO 장기 유입 + 브랜드 신뢰도 구축 |

### 캠페인 3: "월요일 헬스장 가기 싫을 때" 공감 마케팅

| 항목 | 내용 |
|------|------|
| **채널** | 블라인드 / 직장인 커뮤니티 / 인스타그램 |
| **메시징** | 직장인의 운동 포기 순간을 유머러스하게 포착 → "폼코치가 옆에서 봐줄게" 솔루션 제시 |
| **구체적 실행** | 직장인 공감 밈 시리즈 제작: "헬스장 3개월 등록 → 3번 감", "러닝머신만 타다 옴", "유튜브 보고 따라했는데 허리 아픔" → 각 상황에 대한 앱 솔루션 연결 |
| **참여 근거** | 공감 콘텐츠는 공유율이 높고, 직장인 타겟의 핵심 페인포인트를 유머로 전환 |
| **노력** | 저 (밈/카드뉴스 디자인) |
| **기대 효과** | 중 — 브랜드 인지도 + 앱스토어 검색 유도 |

### 캠페인 4: "3주 변화 일기" UGC 캠페인

| 항목 | 내용 |
|------|------|
| **채널** | 인스타그램 / 앱 내 커뮤니티 |
| **메시징** | "3주면 자세가 바뀝니다. 당신의 변화를 기록하세요." |
| **구체적 실행** | 앱이 자동으로 1주/2주/3주 자세 비교 영상을 생성 → 사용자가 SNS에 공유 → 베스트 변화 사례에 3개월 무료 구독 시상. 해시태그: #폼코치3주챌린지 |
| **참여 근거** | Before/After 콘텐츠는 피트니스 분야에서 가장 높은 참여율. 자동 생성으로 사용자 부담 최소화 |
| **노력** | 중 (Before/After 비교 영상 자동 생성 기능 개발) |
| **기대 효과** | 높음 — 실 사용자 증언이 최고의 마케팅. 리텐션 + 바이럴 동시 달성 |

### 캠페인 5: "헬스장 앞 게릴라" 오프라인-온라인 연결

| 항목 | 내용 |
|------|------|
| **채널** | 오프라인 (서울 주요 헬스장 밀집 지역) → 온라인 전환 |
| **메시징** | "오늘 운동, AI가 봐드릴까요? QR 찍고 무료 체험" |
| **구체적 실행** | 강남/홍대/판교 헬스장 밀집 지역에 A형 간판 + QR 포스터 설치. "스쿼트 자세 무료 체크" 이벤트 부스를 주말 2-3회 운영. 현장에서 앱 시연 후 7일 무료 코드 배포 |
| **참여 근거** | 헬스장을 실제로 다니는 타겟에게 직접 도달. 체험이 최고의 설득 수단 |
| **노력** | 중 (부스 운영 인력 + QR 포스터 제작) |
| **기대 효과** | 중 — 직접 도달 범위는 제한적이나, 전환율 매우 높음 (체험 → 설치 30%+) |

- 모든 경쟁사 대비 유효한 포지셔닝 (NTC, Fitbod, 유튜브 모두 "보여주기만" 함)
