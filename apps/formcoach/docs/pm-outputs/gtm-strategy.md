## 1. 비치헤드 세그먼트 (Beachhead Segment)

### 세그먼트 후보 평가

| 세그먼트 | 절실한 고통 | 지불 의향 | 시장 점유 가능성 | 추천 잠재력 | 총점 |
|----------|------------|----------|----------------|------------|------|
| **A. 헬스장 초보 직장인 (25-35세)** | ★★★★★ | ★★★★ | ★★★★★ | ★★★★★ | 19 |
| B. 홈트레이닝 여성 (20-30대) | ★★★★ | ★★★ | ★★★★ | ★★★★ | 15 |
| C. 다이어트 목표 중급자 (30-40대) | ★★★★ | ★★★★ | ★★★ | ★★★ | 14 |
| D. PT 비용 부담 대학생 | ★★★★ | ★★ | ★★★★ | ★★★★ | 14 |

### 선정: **A. 헬스장 초보 직장인 (25-35세)**

**왜 이 세그먼트인가?**
- **절실한 고통**: 헬스장에 등록은 했지만 무엇을 해야 할지 모름. PT는 비싸고(회당 5-10만원), 유튜브는 체계가 없음. "운동 방법을 모른다"는 진입 장벽이 가장 높은 그룹
- **지불 의향**: 이미 헬스장 월 5-10만원을 지출 중. PT 대비 1/10 가격이면 즉시 전환 의향
- **시장 점유 가능성**: 기존 앱들이 "계획 생성"에만 집중 → 실시간 자세 교정이라는 명확한 차별점으로 승부 가능
- **추천 잠재력**: 직장 동료 간 피트니스 문화, SNS 운동 인증 트렌드 → 바이럴 확산 용이

### 인접 확장 세그먼트 (비치헤드 확보 후)

```
Phase 1: 헬스장 초보 직장인 (25-35세)
    ↓
Phase 2: 홈트레이닝 여성 (20-30대) — 홈트 모드 추가
    ↓
Phase 3: 다이어트 목표 중급자 (30-40대) — 영양 관리 연동
    ↓
Phase 4: B2B 기업 복지 시장 — 기업 대시보드 추가
```

## 2. ICP (Ideal Customer Profile)

| 속성 | 정의 |
|------|------|
| **연령** | 25-35세 |
| **직업** | 사무직 직장인 (IT, 금융, 마케팅 등) |
| **지역** | 수도권 대도시 거주 |
| **운동 경험** | 헬스장 등록 1년 이내, 주 2-3회 방문 목표 |
| **핵심 JTBD** | "혼자서도 정확한 자세로, 체계적인 운동을 하고 싶다" |
| **현재 대안** | 유튜브 영상 따라하기 / 가끔 PT / 친구에게 물어보기 |
| **현재 대안의 한계** | 유튜브는 개인화 없음, PT는 비쌈, 친구 조언은 비체계적 |
| **기기** | iOS/Android 스마트폰 (카메라 활용) |
| **월 지출 의향** | 1-3만원 (PT 대비 저렴해야 함) |
| **식별 신호** | 헬스장 관련 커뮤니티 활동, 운동 관련 앱 설치 이력, SNS 운동 인증 게시 |

### 구매 여정

```
인지 → "헬스장 다니는데 효과가 없어" (커뮤니티/SNS)
관심 → "AI가 자세 교정해준다고?" (광고/콘텐츠)
고려 → "무료 체험 해볼까?" (앱스토어)
전환 → "이거 PT 안 받아도 되겠다" (7일 무료 체험)
유지 → "3주만에 스쿼트 자세 완전 달라졌어" (성과 추적)
추천 → "너도 이거 써봐" (SNS 공유/직장 동료)
```

## 3. 포지셔닝 & 메시징

### 포지셔닝 스테이트먼트

> **헬스장에서 혼자 운동하는 초보자**를 위한
> **AI 개인 피트니스 코치**로,
> **PT 없이도 정확한 자세로 체계적인 운동**을 할 수 있게 해줍니다.
> 기존 운동 앱과 달리, **카메라 기반 실시간 자세 교정**으로 부상 없이 올바른 운동을 보장합니다.

### 핵심 메시지 (이해관계자별)

| 대상 | 메시지 | 근거 |
|------|--------|------|
| **사용자 (운동 초보)** | "PT 없이도 정확한 자세로 운동하세요. AI가 실시간으로 봐드립니다." | 카메라 자세 교정 정확도 95%, PT 대비 1/10 비용 |
| **건강 관심층** | "당신의 체력과 목표에 딱 맞는 운동 계획, AI가 매일 만들어드립니다." | 개인화 알고리즘, 200+ 운동 DB |
| **SNS 공유층** | "3주 전의 나 vs 지금의 나, AI 코치와 함께한 변화를 기록하세요." | Before/After 비교, 진행 대시보드 |

### 슬로건 후보
- "당신만의 AI 트레이너, 주머니 속에"
- "자세가 바뀌면, 몸이 바뀝니다"
- "PT 없이도 프로처럼"

## 9. 확장 계획

### Phase 1 (출시 ~ 6개월): 비치헤드 확보
- 헬스장 초보 직장인 세그먼트 집중
- 핵심 기능 안정화 및 리텐션 최적화

### Phase 2 (6 ~ 12개월): 인접 세그먼트 확장
- 홈트레이닝 모드 추가 → 여성 홈트 세그먼트 공략
- 요가/필라테스 자세 교정 추가

### Phase 3 (12 ~ 18개월): 플랫폼 확장
- 영양/식단 관리 연동
- 웨어러블 기기 연동 (Apple Watch, Galaxy Watch)
- B2B 기업 복지 상품 출시

### Phase 4 (18개월 ~): 글로벌 확장
- 일본/동남아 시장 진출
- 영어권 시장 테스트
