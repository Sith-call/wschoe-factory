# 페르소나 피드백: 김하은

| 항목 | 내용 |
|------|------|
| 이름 | 김하은 |
| 나이 | 23세 |
| 직업 | 디자인 전공 대학생 |
| 동기 | "내 색 감각이 상위 몇 %인지 궁금해" |
| 특성 | 앱 사용에 익숙, 예쁜 디자인에 민감, 공유를 많이 함 |

---

## 화면별 피드백

### 1. 인트로 화면

- 전체적으로 깔끔하고 미니멀. 배경 `#fafafa`, 보라 `#7c3aed` 포인트 컬러 조합이 나쁘지 않다.
- 눈 아이콘 + "컬러 감각 테스트" 타이틀 + "다른 색 하나를 찾아보세요" 서브 카피가 명확하다. 앱이 뭔지 바로 알 수 있다.
- "내 최고 기록" 영역이 있어서 재방문 시 이전 점수를 바로 확인할 수 있다.
- **아쉬운 점**: 인트로 화면이 너무 밋밋하다. 컬러 감각 테스트인데 인트로에 색감이 없다. 타일 미리보기나 컬러풀한 시각 요소가 있으면 훨씬 매력적일 것 같다. 인스타 스토리에 올릴 때 첫 화면이 예뻐야 관심을 끄는데, 지금은 텍스트만 있어서 스크린샷 찍고 싶은 느낌이 아니다.
- "시작하기" 버튼이 화면 하단에 있어 엄지 영역에 잘 맞는다.
- "기록 보기"가 텍스트 링크(underline)인데, 다른 CTA와 시각적 위계가 잘 구분된다.

### 2. 게임 화면

- 다크 배경(`#1e1b2e`)으로 전환되는데, 인트로(라이트)와 대비가 확실하다. 게임 몰입감을 위한 좋은 선택.
- HUD 구성(레벨, 타이머, 프로그레스 바, 라운드 표시)이 군더더기 없이 정리되어 있다.
- 타일 색상 차이가 레벨 1에서는 확연하게 보이고, 레벨이 올라갈수록 미세해진다. 그리드도 2x2 -> 3x3 -> 4x4로 커지면서 난이도가 자연스럽게 올라간다.
- 타이머가 10초로 설정되어 있고, 남은 시간에 따라 색상이 바뀐다(회색 -> 노랑 -> 빨강). 시각적 긴장감을 잘 준다.
- "다른 색 타일을 찾으세요" 가이드 텍스트가 1라운드에서만 나오는 것은 적절하다.
- **아쉬운 점**: 정답을 맞혔을 때 피드백이 너무 빠르다(100ms). 맞았다는 쾌감을 느낄 틈이 없다. 짧은 "딩!" 같은 효과음이나 타일이 반짝이는 애니메이션이 있으면 좋겠다.
- **아쉬운 점**: 오답 시 흔들림 애니메이션(shake)은 있지만, 어떤 타일이 정답인지 잠깐이라도 보여주지 않는다. 타임아웃 시에는 정답 타일에 빨간 ring이 뜨는데, 오답 시에도 비슷한 힌트가 있으면 학습 효과가 있다.
- 현재 점수(누적)가 게임 중에는 보이지 않는다. 실시간 점수를 보면 더 몰입할 수 있을 것 같다.

### 3. 게임 오버 화면 (GameOverScreen)

- "시간 초과!" 또는 "틀렸습니다!" + "Round N에서 탈락" 메시지가 간결하다.
- 2초 후 자동으로 결과 화면으로 전환된다. "결과 보기" 버튼도 있어서 기다리기 싫으면 바로 넘길 수 있다.
- **참고**: 20라운드를 모두 클리어하면 이 화면을 건너뛰고 바로 결과로 간다. 올클리어 시 축하 연출이 있으면 더 뿌듯할 것 같다.

### 4. 결과 화면

- 점수 카운트업 애니메이션(0 -> 최종 점수, ease-out 800ms)이 있다. 점수가 올라가는 걸 보는 재미가 있다.
- 등급 뱃지(S+/S/A/B+/B/C/D/F)와 칭호("초인의 눈", "프로 디자이너급" 등)가 순차적으로 나타나는 연출이 좋다.
- "상위 N%" 퍼센타일 표시가 있어서 — 이게 내가 제일 궁금했던 정보다. "상위 1%"라고 나오면 공유하고 싶어진다.
- "NEW BEST! 이전보다 +N점" 배너가 보라색 배경으로 눈에 띈다.
- 통계(클리어 레벨, 평균 반응, 총 라운드)가 깔끔하게 정리되어 있다.
- **아쉬운 점**: "상위 N%"가 정적 벨 커브 시뮬레이션(mean 3000, stddev 1500)이다. 실제 유저 데이터가 아니라서 진정성이 떨어진다. 하지만 MVP에서는 이해할 수 있다.
- **아쉬운 점**: 등급이 F("색맹은 아니죠...?")인 경우, 칭호가 좀 무례하게 느껴질 수 있다. 실제로 색약인 유저에게는 불쾌할 수 있다.
- 버튼 우선순위("다시 도전" primary > "결과 공유" secondary > "기록 보기" tertiary)가 적절하다.
- "이 테스트는 의학적 진단이 아닙니다" 면책 문구가 있다.

### 5. 공유 카드 (ShareCard)

- 바텀 시트 형태로 올라오는 모달. 오버레이 클릭으로 닫기 가능.
- 공유 카드 디자인: 다크 배경(`#1e1b2e`)에 등급 뱃지, "상위 N%", 점수, "당신의 눈은 몇 등급?" CTA 텍스트. 비율 4:5로 인스타 피드에 적합하다.
- "이미지 저장"은 Canvas API로 직접 그리는 방식인데, DOM 캡처가 아니라 수동으로 그리기 때문에 실제 카드와 모양이 다를 수 있다. html2canvas 같은 라이브러리 없이 MVP로 구현한 것으로 보인다.
- "링크 복사"는 텍스트 포맷(`[컬러 감각 테스트]\n나의 색 감각: S+등급...`)으로 클립보드에 복사된다. 카카오톡 등에 바로 붙여넣기 가능.
- **아쉬운 점**: 카카오톡 공유, 인스타 스토리 공유 같은 SNS 직접 공유 버튼이 없다. 23세 대학생으로서 이미지 저장 후 직접 업로드하는 건 번거롭다. 최소한 카카오톡 공유는 있어야 한다.
- **아쉬운 점**: 이미지 저장 결과물이 Canvas에 수동으로 그린 것이라 폰트, 레이아웃이 실제 카드 프리뷰와 다를 가능성이 높다. 공유했을 때 퀄리티가 떨어지면 안 올리게 된다.
- 닫기 애니메이션(modal-exit)이 있는 것은 좋다.

### 6. 기록 화면 (HistoryScreen)

- 최고 기록 카드, 통계(총 플레이/평균 점수/최고 라운드), 최근 기록 리스트가 잘 구성되어 있다.
- 각 기록에 등급 뱃지 색상이 표시되어 한눈에 성적 변화를 볼 수 있다.
- "BEST" 태그가 최고 기록에 붙는 것은 좋다.
- 빈 상태("아직 기록이 없어요")가 잘 처리되어 있다.
- **아쉬운 점**: 성장 그래프가 없다. 디자인 전공생으로서 내 색 감각이 시간에 따라 얼마나 좋아졌는지 시각적으로 보고 싶다. 점수 추이 차트가 있으면 다시 도전할 동기부여가 된다.

## 개선 제안

### P0 (필수)

1. **공유 기능 강화 — 카카오톡/SNS 직접 공유 추가**
   - 현재 "이미지 저장"과 "링크 복사"만 있다. 23세 대학생 타겟이면 카카오톡 공유, 인스타 스토리 공유는 필수다. Web Share API(`navigator.share`)를 활용하면 OS 기본 공유 시트를 띄울 수 있다.
   - 공유 카드 이미지 품질도 개선 필요. Canvas 수동 드로잉 대신 html2canvas 또는 서버 사이드 렌더링으로 실제 카드와 동일한 이미지를 생성해야 한다.

### P1 (중요)

2. **인트로 화면에 시각적 매력 요소 추가**
   - 컬러 감각 테스트인데 인트로에 색감이 전혀 없다. 타일 미리보기 애니메이션, 등급 분포 인포그래픽, 또는 컬러 그리드 배경 패턴 같은 시각 요소를 넣어서 "이 앱 예쁘다, 해보고 싶다"는 느낌을 줘야 한다.
   - 인스타 스토리에 앱 첫 화면 스크린샷을 올릴 때 매력적이어야 바이럴이 된다.

3. **정답/오답 피드백 강화**
   - 정답 시: 타일 반짝임 + 점수 팝업(+150 같은) + 가벼운 햅틱 피드백. 현재 100ms 딜레이는 너무 짧아 맞혔다는 쾌감이 없다.
   - 오답 시: 정답 타일 위치를 잠깐 보여주면 학습 효과가 생긴다.
   - 올클리어(20/20) 시: 특별한 축하 연출(컨페티, 특별 문구 등)이 있어야 성취감이 극대화된다.

### P2 (개선)

4. **기록 화면에 성장 그래프 추가**
   - 점수 추이 차트(최근 10회 등)를 넣으면 "나 점점 나아지고 있다"는 느낌을 받을 수 있다. 재방문 동기부여가 된다.

5. **F등급 칭호 개선**
   - "색맹은 아니죠...?"는 유머로 의도한 것 같지만, 실제 색각 이상이 있는 유저에게는 불쾌할 수 있다. "다시 도전해볼까요?" 또는 "워밍업 필요!" 같은 중립적 표현으로 바꾸는 것을 권장한다.

**종합**: 게임 코어 루프는 잘 만들어져 있고, 기술적 완성도가 높다. 그러나 디자인 전공 대학생으로서 "예쁜 앱을 친구들에게 공유하고 싶다"는 핵심 니즈가 충분히 충족되지 않는다. 인트로의 시각적 매력, 공유 경로 다양화, 게임 내 피드백 강화가 해결되면 80% 이상으로 올라갈 수 있다.
