# 페르소나 피드백: 박영미 (45세, 직장인)

> "카톡에서 공유된 거 보고 들어왔는데 재밌을까?"
> 앱에 익숙하지 않음, 큰 글씨/단순한 UI 선호, 게임을 잘 못함.

---

## 화면별 피드백

### 1. 홈 화면

- 제목 "컬러 감각 테스트"와 부제 "다른 색 하나를 찾아보세요"는 명확하고 이해하기 쉬움
- 눈 아이콘이 직관적으로 색 감각 테스트임을 전달함
- "시작하기" 버튼이 크고 보라색으로 눈에 잘 들어옴 -- 좋음
- **내 최고 기록** 영역이 있을 때와 없을 때가 있음 (새로고침 시 사라지는 현상 발견). 기록이 날아가면 다시 하고 싶은 마음이 사라짐
- "기록 보기" 링크가 작은 텍스트로 되어 있어서 놓치기 쉬움. 45세 사용자 입장에서 더 큰 버튼이 필요함
- 홈 화면 중앙부가 지나치게 비어 있음. 간단한 설명이나 예시 이미지가 있으면 뭘 하는 건지 미리 알 수 있어서 좋겠음

### 2. 게임 플레이 화면

- 어두운 남색 배경에 컬러 타일이 표시되는 구성은 색 구분에 집중하기 좋음
- 상단 정보: "Lv.1", 타이머 "6.8s", "1 / 20" -- 정보가 많지만 글씨가 작음
- **"다른 색 타일을 찾으세요"** 안내 문구가 회색 작은 글씨로 되어 있어 거의 보이지 않음
- 타일 크기 자체는 충분히 크고 터치하기 편함 (2x2 그리드 기준)
- 레벨이 올라가면 타일 수가 급격히 증가 (Lv.19에서 6x6 = 36개). 타일이 작아져서 색 차이를 구분하기 어려워질 수 있음
- **타이머가 10초로 시작하는데, 게임에 익숙하지 않은 사용자에게는 짧게 느껴짐**. 처음 하는 사람은 룰을 파악하는 데만 2-3초가 걸림
- 진행 바(보라색)가 상단에 있지만, 타이머 숫자와 중복 표시되어 어떤 것을 봐야 할지 헷갈림

### 3. 시간 초과 화면

- "시간 초과!" + "Round 1에서 탈락" 메시지가 단순명료함
- "결과 보기" 버튼 하나만 있어서 혼란 없음
- 다만 **첫 판에서 바로 탈락하면 허무함**. "연습 모드"나 시간 제한 없는 모드가 있으면 좋겠음

### 4. 결과 화면

- 점수, 등급(F~S+), 상위 퍼센트, 클리어 레벨, 평균 반응, 총 라운드 등 정보가 풍부함
- **0점일 때 "색맹은 아니죠...?" 라는 멘트가 공격적으로 느껴질 수 있음**. 타이머에 쫓겨서 못 한 건데 색맹 드립은 불쾌함
- S+ 등급일 때 "초인의 눈"은 재밌고 공유하고 싶은 느낌을 줌
- "다시 도전" 버튼이 크고 명확함
- "결과 공유" 버튼이 있어서 카톡 공유 시나리오에 적합
- "이 테스트는 의학적 진단이 아닙니다" 면책 문구는 적절함

### 5. 기록 보기 화면

- 최고 기록 카드, 통계(총 플레이/평균 점수/최고 라운드), 최근 기록 리스트 구성이 깔끔함
- 뒤로 가기("<") 버튼이 있음
- 최근 기록에 등급 뱃지(S+, F)가 색상으로 구분되어 직관적임
- 0점 F 기록이 여러 개 나열되면 보기에 우울함. 숨기거나 필터링 옵션이 있으면 좋겠음

### 6. 결과 공유 모달

- 공유 카드 디자인: 눈 아이콘 + 등급 + 점수 + "당신의 눈은 몇 등급?" 유도 문구 -- 카카오톡 공유에 적합한 구성
- "이미지 저장" / "링크 복사" / "닫기" 세 가지 옵션 제공
- 카카오톡 직접 공유 버튼이 없는 것은 아쉬움 (링크 복사 후 직접 붙여넣기 해야 함)

## 개선 제안 (5개)

### 1. 연습 모드 또는 시간 완화 옵션 추가
게임에 익숙하지 않은 사용자가 첫 판에 바로 시간 초과로 탈락하면 재미를 느끼기 전에 이탈함. "느긋하게 하기" 같은 이지 모드(시간 2배)나, 첫 3라운드는 타이머 없이 진행하는 튜토리얼 모드가 필요함.

### 2. 저점수 결과 멘트 순화
0점일 때 "색맹은 아니죠...?"는 실제 색각 이상이 있는 사용자에게 상처가 될 수 있고, 단순히 타이머에 쫓겨 못 맞춘 사용자에게도 불쾌함. "아쉽지만 다음엔 더 잘할 수 있어요!" 등 격려형 멘트로 변경 권장.

### 3. 글씨 크기 및 안내 문구 가독성 개선
게임 플레이 중 "다른 색 타일을 찾으세요" 안내가 너무 작고 연한 색임. 45세 이상 사용자는 작은 글씨를 읽기 어려움. 상단 Lv/타이머/라운드 정보도 조금 더 크게 표시하면 좋겠음. 홈 화면의 "기록 보기" 텍스트 링크도 버튼 형태로 키워야 함.

### 4. 홈 화면에 게임 방법 간단 안내 추가
카톡 링크로 처음 들어온 사용자는 "다른 색 하나를 찾아보세요"만으로 뭘 해야 하는지 바로 이해하기 어려울 수 있음. 2x2 타일 예시 이미지와 함께 "색이 다른 타일 하나를 터치하세요! 빠를수록 높은 점수!" 같은 1줄 설명을 추가하면 진입 장벽이 낮아짐.

### 5. 카카오톡 직접 공유 기능 추가
"카톡에서 공유된 거 보고 들어온" 사용자 시나리오를 고려하면, 결과 공유에서 "링크 복사" 외에 카카오톡 공유하기 버튼이 직접 있어야 바이럴 효과가 극대화됨. 현재는 링크 복사 후 직접 카톡에 붙여넣기해야 해서 단계가 하나 더 많음.
