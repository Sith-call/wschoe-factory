# 페르소나 피드백: 이준서 (31세, UI 디자이너)

> "직업병인데 동료들이랑 대결하고 싶어"
> 데이터와 시각적 디테일에 민감, 경쟁적, 기록 갱신에 집착.

---

## 화면별 피드백

### 1. 인트로 화면 (IntroScreen)

- 깔끔한 구성. 눈 아이콘 + "컬러 감각 테스트" 타이틀 + "다른 색 하나를 찾아보세요" 서브카피. 군더더기 없다.
- 최고 기록이 있을 때 "내 최고 기록 20,242점 S+등급 상위 1%" 표시되는 것은 좋다. 재방문 시 다시 도전하고 싶게 만든다.
- 다만 최고 기록 영역이 텍스트 한 줄로만 되어 있어 시각적 임팩트가 약하다. 디자이너 입장에서 배지(badge)나 등급 컬러가 여기서도 보이면 좋겠다.
- "기록 보기" 링크가 기록이 없으면 아예 숨겨지는 처리는 적절하다.

### 2. 게임 화면 (GameScreen)

- 상단 HUD(레벨, 타이머, 진행도 바, 라운드 표시)가 잘 정리되어 있다. 정보 밀도가 적절하다.
- 타이머 색상 변화(회색 -> 노란색 -> 빨간색)가 직관적이다. 하단 타이머 바도 같은 색 규칙을 따르는 점이 일관적.
- 타일의 터치 피드백(scale 0.95)이 미세하지만 존재. 다만 정답 시 flash 애니메이션(100ms)이 너무 짧아서 "맞았다"는 느낌이 거의 없다. 레벨 전환이 빠르다 보니 정답 확인감이 부족하다.
- 오답 시 전체 그리드가 흔들리고(wrong-shake) 빨간 테두리가 나타나는데, 타이머가 계속 흐른다. 긴장감이 유지되는 좋은 설계.
- 타임아웃 시 정답 타일에 빨간 ring이 표시되는데, 빨간색이 "틀림"의 의미와 겹쳐서 혼란스럽다. 정답 위치를 알려주는 것이라면 초록색 계열이 더 적절하다.
- 레벨 1에서만 가이드 텍스트("다른 색 타일을 찾으세요")가 나오는 것은 좋은 판단. 반복 플레이 시 방해가 안 된다.
- **다크 모드 배경(#1e1b2e)에서 게임 진행하다가 결과 화면은 라이트(#fafafa)로 바뀌는 전환**이 눈에 부담된다. 특히 어두운 방에서 플레이할 때 갑자기 하얀 화면이 뜨면 눈이 부시다.

### 3. 게임오버 화면 (GameOverScreen)

- "시간 초과!" 또는 "틀렸습니다!" + "Round N에서 탈락" 텍스트가 단순 명확하다.
- 2초 후 자동 전환은 좋지만, "결과 보기" 버튼도 있어서 바로 넘어갈 수 있다.
- 다만 이 화면이 너무 밋밋하다. 탈락 라운드만 보여주고 끝이라 감정적 여운이 없다. 최소한 해당 판의 점수라도 함께 보여주면 좋겠다.

### 4. 결과 화면 (ResultScreen)

- 점수 카운트업 애니메이션(ease-out 800ms)이 있어 기다리는 재미가 있다.
- 등급 배지가 팝 애니메이션으로 등장하고, 상위 퍼센트, 통계가 순차적으로 나타나는 연출이 잘 설계되어 있다.
- "NEW BEST! 이전보다 +N점" 배너는 기록 갱신의 성취감을 준다. 경쟁적인 성격에 딱 맞다.
- 통계 항목(클리어 레벨, 평균 반응, 총 라운드)이 깔끔하게 정리되어 있다.
- 그러나 **"상위 98%"(F등급, 0점)라는 표현이 부자연스럽다**. "하위 2%"가 아니라 "상위 98%"로 표기하면 마치 잘한 것 같은 오해를 줄 수 있다. 등급 테이블의 percentileLabel("하위 15%")과 실제 calculatePercentile 함수의 반환값(상위 X%) 사이에 표현이 혼재되어 있다.
- F등급 타이틀 "색맹은 아니죠...?"가 다소 공격적이다. 유머로 의도한 것 같지만, 실제로 색각 이상이 있는 사용자에게는 불쾌할 수 있다.

### 5. 기록 화면 (HistoryScreen)

- 최고 기록 카드 + 통계(총 플레이, 평균 점수, 최고 라운드) + 최근 기록 리스트 구성이 잘 되어 있다.
- 각 기록에 등급 배지, 점수, 라운드, 날짜가 한눈에 보인다. BEST 태그도 있어 최고 기록이 눈에 띈다.
- 통계가 "총 플레이 18회, 평균 점수 3,102, 최고 라운드 20"처럼 나오는데, **경쟁적인 사용자로서 부족하다**. 연속 S+ 횟수, 최고 연속 클리어, 반응 속도 추이 같은 심화 통계가 없다. 동료들과 비교할 만한 데이터가 점수 하나뿐이다.
- 기록 리스트가 maxHeight 320px로 스크롤 제한되어 있는데, 많이 플레이하면 과거 기록이 잘리는 느낌이 들 수 있다.

### 6. 공유 카드 (ShareCard)

- 다크 테마 카드 디자인이 깔끔하다. 등급 배지 + "상위 1%" + 점수 + "당신의 눈은 몇 등급?" CTA.
- 이미지 저장 기능은 Canvas API로 직접 그리는 방식인데, html2canvas 없이 수동으로 텍스트만 그리고 있어서 실제 카드 모양과 저장되는 이미지가 다를 수 있다. 폰트도 시스템 sans-serif로 fallback.
- "링크 복사" 기능은 텍스트 기반으로 동작. URL이 "https://color-sense.app"으로 하드코딩되어 있어 실제 배포 URL과 다를 수 있다.

## 개선 제안

### 1. 리더보드 / 대전 기능 추가
동료들과 대결하고 싶은 핵심 니즈를 충족시킬 수 없다. 최소한 링크를 공유해서 상대방의 점수와 비교하는 "1:1 대결 모드"나, 공유 링크에 내 점수가 포함되어 상대가 이겼는지/졌는지 확인하는 기능이 필요하다. 현재 공유 기능은 단순 텍스트/이미지 복사뿐이라 경쟁 요소가 전무하다.

### 2. 게임 화면 -> 결과 화면 전환 시 다크/라이트 모드 충돌 해결
게임 화면은 다크(#1e1b2e), 결과/인트로는 라이트(#fafafa)로 급격한 밝기 전환이 일어난다. 눈의 피로도가 높고 몰입감이 깨진다. 결과 화면도 다크 모드로 통일하거나, 전환 시 부드러운 밝기 그라데이션을 적용해야 한다.

### 3. 정답 피드백 강화
정답 시 flash 애니메이션이 100ms로 너무 짧아 "맞혔다"는 쾌감이 거의 없다. 최소 200-300ms의 피드백 + 점수 획득 팝업(+120점 같은)이 라운드마다 보이면 성취감과 긴장감이 공존할 수 있다. 현재는 맞든 틀리든 화면이 바로 넘어가서 감정의 고저가 부족하다.

### 4. 심화 통계 및 성장 그래프
기록 화면에 "반응 속도 추이 그래프", "레벨별 정답률", "연속 클리어 최고 기록" 등 심화 통계가 필요하다. 디자이너 직업병으로 데이터를 분석하고 자기 실력이 성장하는 걸 시각적으로 확인하고 싶다. 현재 통계 3개(총 플레이, 평균 점수, 최고 라운드)로는 깊이가 부족하다.

### 5. 타임아웃 시 정답 표시 색상 수정
타임아웃 시 정답 타일이 빨간 ring(ring-2 ring-[#ef4444])으로 표시되는데, 빨간색은 "오답/실패"의 시각 언어와 겹친다. 정답 위치를 알려주는 의도라면 초록색 또는 보라색(앱의 주 색상 #7c3aed)으로 바꿔야 의미가 명확해진다.

**총평:** 게임의 기본 구조(20라운드, 난이도 스케일링, 등급 체계, 기록 저장)는 탄탄하다. 하지만 UI 디자이너가 동료와 경쟁하려는 핵심 시나리오를 지원하지 못하고, 데이터 시각화가 얕아서 반복 플레이의 동기가 빠르게 소진된다. 혼자 플레이하는 퍼즐이 아니라 "대결 도구"가 되어야 한다.
