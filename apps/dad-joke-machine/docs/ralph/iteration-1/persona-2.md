# 페르소나 2 피드백: 박선생 (45세, 교사)

## 화면별 피드백

### 1. 홈 화면 (메인)
- **첫인상:** 깔끔하고 직관적. "눌러봐!" 버튼 하나만 있어서 뭘 해야 하는지 바로 알 수 있음. 아침 수업 전에 빠르게 개그 하나 뽑기에 딱 좋음.
- **아쉬운 점:** 화면 상단의 빈 공간이 너무 많음. 개그를 뽑기 전 상태에서 "눌러봐!" 버튼만 덩그러니 있으니까 앱이 좀 허전해 보임. 교무실에서 동료 교사가 보면 "이게 뭐야?" 할 수 있음.
- **"오늘 0개 봄" 카운터:** 하단에 작게 표시되는데, 난 이걸 처음에 못 봤음. 있으면 좋은 기능인데 위치가 눈에 안 띔.

### 2. 개그 질문 카드 (답 보기 전)
- **좋은 점:** "탭해서 답 보기"가 자연스러움. 학생들한테 "자, 이거 정답이 뭔지 맞춰봐~" 하면서 잠깐 뜸 들이기 좋음.
- **건너뛰기 버튼:** 질문이 마음에 안 들면 건너뛸 수 있어서 좋음. 수업 분위기에 안 맞는 개그는 패스해야 하니까.
- **아쉬운 점:** 카테고리(예: 음식, 동물, 말장난)가 없음. 내가 수학 시간에 쓸 건지, 점심시간에 쓸 건지에 따라 분위기가 다른데 골라서 볼 수가 없음.

### 3. 개그 답 공개 화면
- **좋은 점:** 답이 크고 주황색으로 강조되어서 눈에 잘 들어옴. "무료 (무'료)", "바나나우유 (banana milk)" 같은 말장난 설명이 괄호 안에 있어서 이해하기 쉬움.
- **즐겨찾기 별표:** 답 옆에 작은 별표가 있어서 마음에 드는 개그를 저장할 수 있음.
- **치명적 버그 발견:** 별표(즐겨찾기 추가)를 누르면 **즐겨찾기 목록 페이지로 이동**되면서 개그가 저장되지 않음! 3번 시도했는데 매번 빈 즐겨찾기 화면이 나옴. 이건 반드시 고쳐야 함. 좋은 개그를 발견했을 때 저장해두고 나중에 쓰려는 건데 핵심 기능이 안 됨.
- **"다음!" 버튼:** 잘 작동함. 다음 개그로 넘어감.

### 4. 즐겨찾기 화면
- **빈 상태:** "아직 저장한 개그가 없어요 — 마음에 드는 개그를 발견하면 별표를 눌러보세요"라는 안내 문구가 친절함.
- **"개그 뽑으러 가기" 버튼:** 홈으로 돌아가는 동선이 명확.
- **뒤로가기(← 화살표):** 잘 작동하긴 하는데, 즐겨찾기에서 뒤로 가면 이전에 보던 개그 카드로 돌아가면 좋겠는데 그냥 홈으로 감.
- **근본 문제:** 즐겨찾기 추가가 안 되니까 이 화면을 제대로 테스트할 수 없었음.

### 5. 반응형 (모바일/태블릿/데스크탑)
- 모바일: 완벽. 내가 주로 쓸 환경이 스마트폰이니까 이게 제일 중요한데, 잘 되어 있음.
- 태블릿: 콘텐츠가 중앙 정렬, 양옆에 여백. 깔끔함.
- 데스크탑: 교무실 PC에서도 볼 수 있을 정도. 다만 화면이 넓은데 콘텐츠가 좁아서 좀 심심.

## 개선 제안 (우선순위별)

### P0 (반드시 수정)
1. **즐겨찾기 추가 버그 수정** — 별표 버튼을 누르면 개그가 저장되어야 하는데, 즐겨찾기 목록 화면으로 이동만 됨. 저장 기능이 완전히 동작하지 않음. 나에게 이건 핵심 기능임 — 좋은 개그를 모아뒀다가 교실에서, 교무실에서, 가족 카톡방에서 써먹어야 하는데 저장이 안 되면 매번 새로 뽑아야 함.

### P1 (강력 추천)
2. **카테고리/태그 분류** — "음식", "동물", "말장난", "학교" 같은 카테고리로 개그를 분류해줬으면 좋겠음. 수학 시간에 쓸 개그와 가족 카톡방에 올릴 개그는 다름.
3. **개그 공유 기능** — 좋은 개그가 나오면 바로 카카오톡이나 문자로 보내고 싶음. "공유" 버튼이 있으면 가족 카톡방에 매일 저녁 하나씩 보내기 편함.

### P2 (있으면 좋겠음)
4. **홈 화면에 오늘의 한 줄 개그** — 앱 열자마자 오늘의 추천 개그 하나를 보여주면 좋겠음. 출근길에 앱 열고 바로 하나 챙겨가는 느낌.
5. **"이전에 본 개그" 기록** — 방금 봤던 개그가 뭐였는지 까먹을 때가 있음. 히스토리가 있으면 편함.
6. **데스크탑에서의 콘텐츠 활용** — 넓은 화면에서 여러 개그를 한눈에 볼 수 있는 모드가 있으면 교무실 PC에서 미리 개그를 골라둘 수 있음.

## 만족도 점수: 52/100

## 만족도 근거
- **긍정 요소 (+):**
  - 핵심 플로우(개그 뽑기 → 질문 보기 → 답 보기)가 직관적이고 빠름 (+15)
  - UI가 깔끔하고 군더더기 없음 (+10)
  - 건너뛰기 기능이 잘 작동함 (+5)
  - 반응형 대응이 잘 되어 있음 (모바일 우선) (+8)
  - 개그 퀄리티가 아재개그 본연의 맛이 있음 (+7)
  - "오늘 N개 봄" 카운터가 소소한 성취감 (+3)
  - JS 에러 없이 안정적 (+4)

- **부정 요소 (-):**
  - **즐겨찾기 기능 완전 고장** — 핵심 기능 미작동 (-25). 나 같은 교사는 좋은 개그를 모아서 반복 사용해야 하는데 저장이 안 되면 매일 새로 뽑아야 함. 학생들이 "또요?"하는 문제를 해결하려면 레퍼토리 관리가 필수인데, 그게 안 됨.
  - 카테고리/분류 없음 (-8). 수업/교무실/가족 상황별로 다른 개그가 필요한데 무작위만 됨.
  - 공유 기능 없음 (-5). 카톡방에 바로 보내고 싶은데 텍스트를 수동으로 복사해야 함.
  - 홈 화면이 허전함 (-3). 처음 열었을 때 개그 하나 바로 보여주면 좋겠음.
