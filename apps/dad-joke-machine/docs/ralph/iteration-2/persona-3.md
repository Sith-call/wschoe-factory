## 발견된 버그

### [BUG-1] 즐겨찾기 저장 불안정 (심각도: 높음)
- **증상**: 별표 눌러서 "저장됨!" 토스트가 떴는데, 즐겨찾기 페이지 갔더니 목록이 비어 있었음
- **재현 조건**: 개그에서 별표 클릭 → 즐겨찾기 페이지 이동 시 간헐적으로 발생
- **원인 추정**: localStorage에 `djm-favorites: "[]"`로 비어 있었다가 이후 재확인 시 `[19]`로 복구됨. 비동기 타이밍 이슈 가능성
- **영향**: 유저가 "저장됨!" 보고 즐겨찾기 갔는데 없으면 신뢰 깨짐. 과팅에서 "아까 그 개그 뭐였지" 하고 즐겨찾기 열었는데 비어있으면 당황스러움

### [BUG-2] 즐겨찾기 버튼이 페이지 이동 트리거 (심각도: 중간)
- **증상**: 개그 화면에서 "즐겨찾기 추가" 버튼(별표) 누르면 저장과 동시에 즐겨찾기 목록 페이지로 이동하는 경우가 있음
- **기대 동작**: 별표만 토글하고 현재 화면 유지
- **영향**: 개그 보다가 갑자기 화면 전환되면 흐름이 끊김

## 개선 제안 (우선순위별)

### 1순위 (필수)
1. **즐겨찾기 저장 안정성 수정** — 위 BUG-1 해결 필수. "저장됨!" 뜨면 반드시 저장되어 있어야 함
2. **개그 수 여전히 50개** — Iteration 1에서도 말했지만 100개는 있어야 반복 없이 쓸 수 있음. 과팅에서 빠르게 돌리면 50개 금방 소진됨
3. **카테고리/필터 기능** — "언어유희만", "상황극만" 골라 볼 수 있으면 활용도가 확 올라감

### 2순위 (있으면 좋음)
4. **공유 기능** — 카톡 공유는 과팅/동아리방에서 핵심 기능. 개그 카드 이미지 생성 + 공유 버튼
5. **개그 카드 영역 키우기** — 질문 텍스트가 화면 중앙에 좀 작게 몰려 있음. 카드 패딩이나 폰트 크기 키우면 시원시원해 보일 듯
6. **다크모드** — 여전히 필요. 시험 기간에 밤새 공부하다 쉬는 시간에 보는 앱인데 눈이 아픔

### 3순위 (재미 요소)
7. **"50개 전부 클리어!" 축하 화면** — 코드에 있는 것 같은데 아직 못 봤음. 이거 보면 성취감 들 것 같음
8. **개그 랭킹** — 다른 사람들이 재밌다고 한 개그 TOP 10
9. **연속 재생 모드** — 탭 없이 자동 넘김. 과팅에서 폰 세워놓고 쓰기 좋음
