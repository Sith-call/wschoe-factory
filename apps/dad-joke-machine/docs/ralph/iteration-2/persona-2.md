### 2. "건너뛰기" 버튼 Secondary 스타일 → PASS
- 주황색 아웃라인 스타일로 변경됨. "다음!" 버튼(CTA)과 명확히 구분됨

### 3. 즐겨찾기 별표 UX → 부분 수정 (버그 잔존)
- **첫 저장은 정상 동작!** 별표 클릭 시 화면 유지 + 별이 채워지고 "저장됨" 표시
- **하지만 "저장됨" 상태에서 재클릭하면 즐겨찾기 페이지로 이동하는 버그 발생!** 저장 해제(토글)를 시도하면 즐겨찾기 목록 화면으로 튕겨나감. Iter 1의 핵심 버그가 부분적으로만 수정됨.
- 즐겨찾기 페이지에서는 저장된 개그가 정상 표시됨 (이전엔 빈 화면이었으니 이건 개선)

### 4. 클립보드 복사 버튼 → 확인 불가
- 복사 버튼 자체는 존재하고 클릭 가능
- headless 환경에서 "복사 실패" 토스트가 나왔지만, 이는 브라우저 권한 제한일 가능성 높음
- 실제 모바일에서는 동작할 것으로 보이나, 실기기 테스트 필요

### 5. 즐겨찾기 삭제 터치 영역 확대 → PASS
- 삭제 버튼(휴지통 아이콘)이 잘 보이고 탭하기 쉬움
- 삭제 시 즉시 목록에서 제거됨. 깔끔.

### 6. 카운터 자정 리셋 → 확인 불가
- 현재 "오늘 8개 봄" 정상 카운팅 확인. 자정 리셋은 시간 제약으로 미검증.

## 화면별 피드백

### 1. 홈 화면
- **좋아진 점:** 서브 카피 추가로 훨씬 완성도 있어 보임. "눌러봐!" 버튼도 크고 눈에 잘 들어옴.
- **여전히 아쉬운 점:** 카드(질문) 위의 빈 공간이 여전히 넓음. 화면 중앙~하단에 카드가 몰려 있고 상단 절반이 비어 있어서 콘텐츠가 밀려 내려간 느낌.

### 2. 개그 질문 카드 (답 보기 전)
- "탭해서 답 보기" 힌트가 여전히 직관적
- 건너뛰기가 Secondary 스타일이라 "이건 보조 행동"이라는 게 바로 느껴짐. Iter 1에선 주버튼인지 부버튼인지 헷갈렸는데 해결됨

### 3. 개그 답 공개 화면
- 답이 크고 주황색 볼드로 강조 — 눈에 잘 들어옴
- 복사 + 저장 버튼이 나란히 배치되어 깔끔
- **"저장됨" 토글 버그**: 한 번 저장한 개그의 별을 다시 누르면 즐겨찾기 페이지로 이동됨. "실수로 저장했는데 취소하고 싶다"가 안 됨. 이건 꽤 불편 — 수업 중에 잘못 누르면 현재 보던 개그를 잃어버림.

### 4. 즐겨찾기 화면
- **큰 개선!** 저장된 개그가 실제로 목록에 표시됨 (Iter 1에서는 빈 화면이었음)
- 질문 + 답이 한 줄에 보여서 한눈에 확인 가능
- 삭제 버튼이 오른쪽에 있어서 자연스러움
- **아쉬운 점:** 즐겨찾기에서 개그를 탭하면 답이 펼쳐지거나 상세 보기가 되면 좋겠음. 지금은 그냥 리스트만 보임.
- 뒤로가기(←)가 정상 동작

## 개선 제안 (우선순위별)

### P0 (반드시 수정)
1. **"저장됨" 상태 별표 재클릭 시 페이지 이동 버그** — 저장 해제(토글 off) 동작이 즐겨찾기 페이지 이동으로 처리되고 있음. 현재 화면을 유지한 채 별이 빈 별로 돌아가고 저장이 해제되어야 함. Iter 1 핵심 버그의 잔존. 수업 중에 실수로 별표를 누르면 개그 카드를 잃어버리게 되어 당황할 수 있음.

### P1 (강력 추천)
2. **카테고리/태그 분류** — Iter 1에서도 요청했지만 여전히 없음. "음식", "동물", "말장난", "학교" 같은 태그가 있으면 수업 주제에 맞는 개그를 골라 쓸 수 있음. 교사에겐 핵심.
3. **공유 기능 (카카오톡/문자)** — 복사 버튼은 추가됐지만, "공유" 버튼으로 바로 카톡에 보내는 게 더 편함. 복사 후 카톡 열고 붙여넣기는 3단계인데, 공유는 1단계.

### P2 (있으면 좋겠음)
4. **상단 여백 줄이기** — 카드가 화면 중하단에 치우쳐 있어서, 위쪽 공간이 아까움. 카드를 조금 위로 올리면 전체적으로 더 안정감 있을 것.
5. **즐겨찾기에서 개그 탭 → 상세보기** — 리스트에서 탭하면 원래 카드 형태로 보여주면 더 재미있음. 학생들 앞에서 즐겨찾기를 열어 보여줄 때 카드 형태가 더 "쇼"하기 좋음.
6. **"이전에 본 개그" 히스토리** — Iter 1에서도 요청. 방금 봤던 개그가 뭐였는지 까먹을 때 있음.
