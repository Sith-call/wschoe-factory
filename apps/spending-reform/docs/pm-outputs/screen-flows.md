## 네비게이션 구조

```
IntroScreen → QuizScreen (12문항) → LoadingScreen → ResultScreen
                                                         ↓
                                              ChallengeIntroScreen
                                                         ↓
                                              ChallengeScreen (7일 루프)
                                                    ↕
                                              CheckInScreen (매일)
                                                         ↓
                                         ChallengeCompleteScreen → ShareCard (모달)
                                                    ↕
                                              HistoryScreen
                                                    ↓
                                         (재진단 → QuizScreen 또는 새 챌린지)
```

### 네비게이션 원칙
- **선형 플로우**: 탭바 없음 (R5 준수)
- **뒤로가기**: 각 화면 좌상단 뒤로가기 아이콘 (QuizScreen 제외 — 진단 중 이탈 방지)
- **전환 애니메이션**: 좌→우 슬라이드 (translateX, 300ms ease-out)
- **앱 재진입 분기**:
  - 챌린지 진행 중 → ChallengeScreen
  - 챌린지 완료 → ChallengeCompleteScreen
  - 데이터 없음 → IntroScreen

### 인터랙션
- 선택지 탭 → 테두리 색 변경(#0d9488) + 배경(#ccfbf1)
- 선택 후 300ms 딜레이 → 다음 문항 슬라이드 (좌로 밀림)
- 뒤로 스와이프 (제스처) 또는 좌상단 뒤로가기 → 이전 문항
- 격려 메시지 (조건부):
  - 6/12: "벌써 반이나 왔어요!"
  - 10/12: "거의 다 왔어요!"
  - 12/12: → LoadingScreen 전환

### 12문항 배치 순서

| 번호 | 카테고리 | 문항 |
|------|----------|------|
| 1 | 카페 | 일주일에 카페를 몇 번 가시나요? |
| 2 | 배달 | 일주일에 배달 음식을 몇 번 시키시나요? |
| 3 | 쇼핑 | 최근 한 달, 계획에 없던 물건을 산 적이 있나요? |
| 4 | 구독 | 현재 구독 중인 서비스가 몇 개인가요? |
| 5 | 교통 | 일주일에 택시를 몇 번 타시나요? |
| 6 | 유흥 | 한 달에 음주 모임이 몇 번인가요? |
| 7 | 카페 | 카페에서 한 번에 보통 얼마를 쓰시나요? |
| 8 | 배달 | 배달 주문 시 평균 금액은 얼마인가요? |
| 9 | 쇼핑 | 온라인 쇼핑몰 장바구니에 물건이 몇 개 있나요? |
| 10 | 구독 | 최근 3개월간 한 번도 안 쓴 구독이 있나요? |
| 11 | 교통 | 걸어서 15분 거리도 택시를 타시나요? |
| 12 | 유흥 | 한 번 음주 모임에 보통 얼마를 쓰시나요? |

각 문항 선택지 (4지선다, 1-4점):

**Q1 (카페 빈도):** (1) 거의 안 가요 / (2) 주 1-2회 / (3) 주 3-4회 / (4) 거의 매일
**Q2 (배달 빈도):** (1) 거의 안 시켜요 / (2) 주 1-2회 / (3) 주 3-4회 / (4) 거의 매일
**Q3 (충동 쇼핑):** (1) 없어요 / (2) 1-2번 / (3) 3-5번 / (4) 셀 수 없어요
**Q4 (구독 개수):** (1) 0-1개 / (2) 2-3개 / (3) 4-6개 / (4) 7개 이상
**Q5 (택시 빈도):** (1) 거의 안 타요 / (2) 주 1-2회 / (3) 주 3-4회 / (4) 거의 매일
**Q6 (음주 빈도):** (1) 월 0-1회 / (2) 월 2-3회 / (3) 월 4-6회 / (4) 월 7회 이상
**Q7 (카페 금액):** (1) 3천원 이하 / (2) 3-5천원 / (3) 5-7천원 / (4) 7천원 이상
**Q8 (배달 금액):** (1) 1만원 이하 / (2) 1-2만원 / (3) 2-3만원 / (4) 3만원 이상
**Q9 (장바구니):** (1) 0-2개 / (2) 3-5개 / (3) 6-10개 / (4) 10개 이상
**Q10 (미사용 구독):** (1) 없어요 / (2) 1개 / (3) 2-3개 / (4) 4개 이상
**Q11 (단거리 택시):** (1) 절대 안 타요 / (2) 가끔 / (3) 자주 / (4) 항상
**Q12 (음주 금액):** (1) 2만원 이하 / (2) 2-5만원 / (3) 5-10만원 / (4) 10만원 이상

### DESIGN_RULES.md 체크리스트

```
[✓] 기존 앱과 다른 색상 팔레트인가? → #0d9488 (기존 #10b981, #fbbc00 등과 다름)
[✓] 기존 앱과 다른 폰트 조합인가? → Wanted Sans + Sora (기존: Pretendard+Inter, Spoqa+Space Grotesk 등)
[✓] 그라디언트 텍스트를 안 쓰는가? → 미사용
[✓] 네비게이션 패턴이 앱 구조에 맞는가? → 선형 플로우, 탭바 없음
[✓] fadeInUp 외에 다른 (또는 없는) 애니메이션을 쓰는가? → 슬라이드, 스케일, 드로잉, 즉시 표시
[✓] 불필요한 장식 요소(파티클, 오브)가 없는가? → 미사용
[✓] backdrop-blur를 의미 있는 곳에만 쓰는가? → ShareCard 모달 1곳만
[✓] 다크/라이트 테마 비율이 균형 잡혀 있는가? → 라이트 모드
[✓] 이모지를 아이콘/장식으로 쓰지 않았는가? → UI는 SVG, 공유 텍스트에서만 이모지
```
