# PRD: 감정 약국 (Mood Pharmacy)

## 제품 개요
감정을 "처방전"으로 관리하는 무드 트래킹 앱. 사용자가 현재 감정 상태를 기록하면, 약국 컨셉으로 맞춤 처방전(활동, 확언, 호흡법, 음악 추천)을 제공하고, 감정 건강을 시각적으로 추적할 수 있다.

## 타겟 사용자
- 한국 20-30대 직장인
- 감정 관리에 관심이 있지만 전문적인 상담은 부담스러운 사람
- 가볍고 재미있는 방식으로 일상의 감정을 기록하고 싶은 사람

## 핵심 가치
1. **낮은 진입 장벽**: 1분 이내로 감정 기록 + 처방전 수령
2. **약국 컨셉**: 딱딱한 심리 앱이 아닌, 재미있는 처방전 메타포
3. **실용적 처방**: 실제로 실행 가능한 활동, 호흡법, 음악 추천
4. **시각적 추적**: 달력, 트렌드, 밸런스 차트로 감정 건강 모니터링

## 핵심 기능 (6 스크린)

### 1. IntroScreen (약국 홈)
- "감정 약국에 오신 것을 환영합니다" 메시지
- 연속 기록 streak 배지
- 오늘의 감정 처방받기 CTA 버튼
- 최근 기록 표시
- 8개 감정 퀵 선택 바
- 기록 보기 / 처방전 모음 네비게이션 카드

### 2. MoodCheckScreen (감정 진료)
- 8개 감정 선택: 행복/평온/불안/우울/분노/피로/설렘/공허
- 감정별 이모지, 설명, 색상 코드
- 강도 슬라이더 (1-5): 아주 조금 ~ 매우 강하게
- 시각적 강도 바
- 선택 메모 (200자 이내)

### 3. PrescriptionScreen (처방전)
- 의료 처방전 스타일 카드 디자인
- 4가지 처방 항목:
  - Rx.1 활동: 감정별 맞춤 활동 추천 (3-4개 중 랜덤)
  - Rx.2 확언: 감정별 긍정 확언 (4-5개 중 랜덤)
  - Rx.3 호흡: 감정별 호흡 운동 패턴
  - Rx.4 음악: 감정별 장르 + 예시곡 추천
- 즐겨찾기(별표) 기능
- 과다 복용 주의 문구 (유머)

### 4. MoodHistoryScreen (기록 보기)
- 달력 뷰: 월별 감정 이모지 표시
- 추이 뷰: 최근 7일 감정 강도 바 차트
- 총 기록 수, 연속 기록, 최근 감정 통계
- 기록 목록 (최근 10건)

### 5. PharmacyShelfScreen (처방전 모아보기)
- 전체/즐겨찾기/감정별 필터
- 접기/펼치기 카드 UI
- 처방전 상세 보기 (활동, 확언, 호흡, 음악)
- 즐겨찾기 토글

### 6. ProfileScreen (프로필)
- 감정 약국 프로필 카드
- 가장 자주 느끼는 감정
- 평균 감정 강도
- 감정 밸런스 바 (긍정/중립/부정 비율)
- 감정 분포 차트 (8개 감정별 %)
- 데이터 초기화 기능

## 감정 데이터

### 8개 감정 카테고리
| 감정 | 색상 | 분류 |
|------|------|------|
| 행복 | green | 긍정 |
| 평온 | purple | 긍정 |
| 불안 | amber | 부정 |
| 우울 | blue | 부정 |
| 분노 | red | 부정 |
| 피로 | gray | 중립 |
| 설렘 | amber | 긍정 |
| 공허 | gray | 중립 |

### 처방 데이터 규모
- 활동: 감정당 3-4개 (총 28개)
- 확언: 감정당 4-5개 (총 34개)
- 호흡법: 감정당 1-2개 (총 10개)
- 음악 추천: 감정당 2개 (총 16개, 예시곡 포함)

## 기술 스택
- Vite + React 19 + TypeScript
- Tailwind CSS CDN
- localStorage (데모 모드, DB 없음)
- 모바일 퍼스트 (430px viewport)

## 디자인 시스템
- **테마**: 다크 + 약국/한약방 분위기
- **베이스 색상**: Deep teal (#0a4a3a) -> surface (#061a14)
- **처방전 강조**: Warm amber (#d4a017)
- **평온 감정**: Soft purple (#9b72cf)
- **강렬한 감정**: Warm red (#e05555)
- **폰트**: Noto Sans KR (본문) + Noto Serif KR (강조)
- **애니메이션**: fadeIn, slideUp, float, breathe

## 데모 모드
- 최초 접속 시 7일간의 시뮬레이션 데이터 자동 생성
- 다양한 감정 (피로->불안->우울->분노->평온->행복->설렘) 순서
- 각 날짜별 메모 포함
- 즐겨찾기 처방전 2개 포함
- 초기화 시 데모 데이터 재생성

## 성공 지표
- 일일 활성 사용자 (DAU) 리텐션 > 30% (7일)
- 평균 세션 시간 > 2분
- 처방전 즐겨찾기 비율 > 20%
- 연속 기록 3일 이상 유지 비율 > 40%

## 향후 확장
- 감정 알림 (아침/저녁 기록 리마인더)
- AI 기반 처방전 개인화
- 감정 공유 (SNS, 친구)
- 주간/월간 감정 리포트
- 실제 음악 스트리밍 연동
