# PRD: 스피드 수학 (Speed Math)

## 개요
60초 타임어택 암산 챌린지 게임. 제한 시간 안에 최대한 많은 수학 문제를 풀고, 콤보로 보너스 시간을 획득한다.

## 문제 정의
- 일상에서 암산 능력을 훈련할 간편한 도구가 없다
- 기존 수학 앱은 학습 중심이라 재미 요소가 부족하다
- 짧은 시간에 집중력을 높이고 성취감을 느낄 수 있는 두뇌 트레이닝 수요가 있다

## 타겟 유저
학생, 직장인, 두뇌 트레이닝을 원하는 일반 사용자 (10대~40대)

## 유저 스토리

1. **난이도 선택** — 사용자는 초급/중급/고급 중 난이도를 선택할 수 있다. 내 수준에 맞는 문제를 풀고 싶다.
2. **60초 타임어택** — 사용자는 60초 제한 시간 안에 연속으로 출제되는 문제를 풀 수 있다. 시간 압박 속에서 집중력을 시험하고 싶다.
3. **콤보 보너스** — 사용자는 연속 정답 시 콤보가 쌓이고, 3콤보마다 +2초 보너스 시간을 받는다. 연속 정답의 보상이 있어야 몰입할 수 있다.
4. **즉각 피드백** — 사용자는 정답/오답 즉시 시각적 피드백을 받는다. 오답 시 콤보가 초기화된다.
5. **결과 확인** — 사용자는 라운드 종료 후 맞힌 수, 정확도, 최고 콤보, 등급(S/A/B/C/D)을 확인할 수 있다.
6. **최고 기록** — 사용자는 난이도별 최고 기록(점수, 콤보)이 자동 저장되어 언제든 확인할 수 있다.
7. **연습 모드** — 처음 접속한 사용자는 3문제 연습 모드를 통해 조작 방법을 익힐 수 있다.

## 핵심 기능 (MVP)

### 1. 난이도 시스템
| 난이도 | 연산 | 숫자 범위 | 예시 |
|--------|------|-----------|------|
| 초급 | +, - | 1~9 | 3 + 7 |
| 중급 | +, -, x | 10~99 | 45 + 27, 6 x 8 |
| 고급 | +, -, x, / | 100~999 (혼합) | 234 - 78, 12 x 15 |

### 2. 60초 타임어택
- 기본 제한 시간 60초
- 화면 상단에 타이머 바 표시
- 시간 종료 시 자동으로 결과 화면 전환

### 3. 콤보 시스템
- 연속 정답 시 콤보 카운트 +1
- 3콤보마다 보너스 시간 +2초
- 오답 시 콤보 초기화
- 현재 콤보 수 화면에 표시

### 4. 결과 화면
- 맞힌 문제 수 / 전체 시도 수
- 정확도 (%)
- 최고 콤보
- 등급: S(20+), A(15+), B(10+), C(5+), D(5 미만)
- "다시 도전" 버튼

### 5. 최고 기록
- localStorage에 난이도별 저장
- 홈 화면에서 난이도별 최고 점수 표시

### 6. 연습 모드 (데모)
- 최초 접속 시 자동 안내
- 타이머 없이 3문제 체험
- 조작법(숫자 입력 + Enter) 안내

## 화면 구성
1. **홈** — 난이도 선택 + 최고 기록 표시 + 연습 모드 버튼
2. **게임** — 문제 표시 + 입력창 + 타이머 + 콤보 카운트
3. **결과** — 점수/정확도/콤보/등급 + 다시 도전 + 홈으로

## 기술 스택
- React + TypeScript + Vite
- Tailwind CSS (CDN)
- localStorage (기록 저장)
- DB 없음, 서버 없음 — 완전 클라이언트 사이드

## 성공 지표
| 지표 | 목표 |
|------|------|
| 라운드당 평균 플레이 시간 | 65초 이상 (콤보 보너스 포함) |
| 재도전율 | 첫 플레이 후 70% 이상이 2회 이상 플레이 |
| 난이도 전환율 | 초급 플레이어 중 40%가 중급 시도 |
| 최고 기록 갱신 시도 | 유저당 평균 5회 이상 플레이 |

## Out of Scope (MVP 제외)

- 서버 기반 랭킹 시스템
- 커스텀 시간 설정
- 문제 유형 커스터마이징
- 소셜 공유 기능
- 사운드 효과 / BGM
- 모바일 네이티브 앱
