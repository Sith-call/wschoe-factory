## 네비게이션 구조

### 앱 흐름도

```
[첫 실행]
   │
   ▼
IntroScreen → GoalScreen → ScheduleScreen
   (온보딩 플로우: 선형, 뒤로가기 + 건너뛰기)
   │
   ▼ (온보딩 완료)
   │
HomeScreen ←──── 상단 세그먼트 컨트롤 ────→
   │              │              │              │
   홈         캘린더         인사이트         설정
   │              │              │              │
HomeScreen   CalendarScreen  InsightScreen  SettingsScreen
   │              │
   │              ▼
   │         DayDetailSheet (바텀시트)
   │
   ▼ (기록 CTA)
LogScreen (풀스크린 오버레이)
   │
   Step1 → Step2 → Step3 → Step4 → 완료
   │
   ▼ (완료 후)
HomeScreen
```

### 네비게이션 규칙
- **메인 네비**: 상단 세그먼트 컨트롤 (홈 / 캘린더 / 인사이트 / 설정)
- **바텀 탭바 사용 안 함** (DESIGN_RULES.md R5 준수)
- **온보딩**: 선형 플로우, 뒤로가기 + 건너뛰기
- **기록 플로우**: 풀스크린 오버레이, 닫기(X) + 뒤로가기
- **바텀시트**: 캘린더 날짜 상세, 드래그 닫기

## 화면 5: LogScreen (수면 기록 — 4단계)

### 목적

### 공통 레이아웃

```

### Dot Indicator 상세

```
현재 스텝:  ● (teal-600, 10px)

### Step 1: 취침 시간
