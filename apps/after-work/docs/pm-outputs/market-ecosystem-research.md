# 퇴근하면 뭐하지? — 시장 생태계 리서치

## 1. 직접 경쟁 앱 (마이크로 저널 / 한 줄 일기)

| 앱 | 다운로드 | 핵심 기능 | 차별점 |
|---|---|---|---|
| **Daylio** | 1,800만+ | 글 없이 무드+아이콘 탭으로 기록, 통계 차트 | 텍스트 입력 자체를 없앰. 감정 추적에 집중 |
| **DayGram** | 비공개 (App Store 4.7) | E-Ink 스타일 UI, 하루 한 줄 | 미니멀 디자인. 다만 큐레이션/영감 기능 없음 |
| **One Line a Day** | 비공개 | 하루 한 줄 + 연도별 비교 | 5년치 같은 날짜 비교가 핵심. 영감 제공 없음 |
| **Five Minute Journal** | 100만+ | 감사일기 3줄 + 목표 + 리뷰 | 긍정심리학 기반 구조화된 저널. 무겁고 진지함 |
| **Goodiary** | 50만+ | 심플 일기 + 감정 태그 | 광고/인앱결제 없음. 범용 일기 |
| **Day One** | 1,500만+ | 풀기능 저널 (사진, 위치, 날씨) | 프리미엄 저널. 한 줄 컨셉과 정반대의 풍부한 기록 |

**공통 약점**: 기존 앱들은 "기록"에만 집중. 뭘 쓸지 모르는 사용자에게 **영감/마중물을 주는 앱은 부재**.

## 2. 간접 경쟁 (퇴근 후 활동 / 습관 추적)

| 앱/서비스 | 성격 | 한계 |
|---|---|---|
| **프립(Frip)** | 액티비티 예약 플랫폼 | 예약/결제 중심. "오늘 뭐하지?" 고민 단계 해결 안 됨 |
| **주모(ZUMO)** | 핫플/맛집 큐레이션 | 장소 추천. 활동 자체의 영감이 아님 |
| **웬지** | 수공예 클래스 모음 | 클래스 상품 판매. 가벼운 영감과 거리 멀음 |
| **Habitica** | 게이미피케이션 습관 추적 | RPG 요소. "피곤한 직장인"과 톤 미스매치 |
| **Fabulous** | 루틴 빌더 | 운동/명상 루틴 설계. 의지력 필요 |
| **모카클래스** | B2B 복지 취미 추천 | 기업 복지용. 개인 일상과 괴리 |

**핵심 인사이트**: 활동 추천 앱은 "예약/구매"로 이어지는 커머스 모델. "한 줄로 생각만 적어보는" 저관여 저널은 존재하지 않음.

## 3. 블루오션 분석

기존 앱 생태계에서 "퇴근하면 뭐하지?"가 차지하는 고유 포지션:

| 축 | 기존 앱 | 퇴근하면 뭐하지? |
|---|---|---|
| **톤** | 응원/긍정/게이미피케이션 | 건조하고 말이 적음. 친한 척 안 함 |
| **입력 부담** | 3줄 이상 or 다중 선택 | 딱 한 줄 |
| **영감 제공** | 없음 (빈 입력창) or 과잉 (커머스 연결) | 마중물 카드 3~4장. 안 골라도 됨 |
| **목적** | 감정 추적 / 감사 / 습관 형성 | "하고 싶은 것" 인식. 그 자체가 끝 |
| **타겟 감정** | 성장 욕구, 자기계발 | 퇴근 후 피로, "아무것도 안 하고 싶지만 뭔가 하고 싶은" 모순 |

**차별화 요약**: "건조한 톤 + 한 줄 + 마중물 큐레이션"의 조합은 현재 시장에 없음. 감사일기의 진지함도, 습관앱의 압박도, 커머스 앱의 결제 유도도 없는 **"생각만 하게 해주는 앱"**.

## 4. 포화/기회 영역

### 포화 (레드오션)
- **풀기능 저널**: Day One, Journey 등 프리미엄 앱이 장악. CAGR 11-15%로 성장 중이나 진입 장벽 높음
- **감사 일기**: Five Minute Journal, Grateful 등 포지티브 심리학 기반 앱 과밀
- **습관 추적**: Habitica, Fabulous, Streaks 등 게이미피케이션 레드오션
- **액티비티 예약**: 프립, 클래스101, 탈잉 — 커머스 모델로 자본 싸움

### 기회 (블루오션)
- **"생각의 마중물" 저널**: 뭘 쓸지 모르는 사람에게 가벼운 영감을 주는 카테고리 자체가 공백
- **피곤한 직장인 타겟**: 대부분의 웰니스 앱이 "의지력 있는 사용자"를 전제. 에너지 제로 상태의 UX가 비어 있음
- **한 줄 + 축적**: One Line a Day류 존재하나 영감 제공이 없음. "적을 거리를 만들어주는" 결합이 미개척
- **건조한 톤의 웰니스**: 시장 전체가 따뜻한/긍정적 톤. "말 적고, 판단 없는" 톤은 차별화 자산
