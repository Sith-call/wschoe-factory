## 총평

와, 확실히 달라졌다. 앱을 열자마자 "클래스를 선택하세요"가 뜨는 순간 "오 이거 진짜 RPG 게임 시작이다" 하는 느낌이 바로 온다. 전사/마법사/힐러/레인저 4개 클래스 카드가 쫙 뜨고, 각각 SVG 캐릭터가 있어서 고르는 재미가 있다. 이름까지 입력하니까 "내 캐릭터"라는 소유감이 확 생겼다. 그리고 16개 루틴 중에서 내가 직접 4~12개를 고르는 화면은 진짜 필요했던 기능이다. "남의 습관 앱"에서 "내 습관 앱"이 된 느낌이다. 레벨업할 때 파티클이 쏟아지는 축하 모달도 기분이 좋다. 근데... 아직 부족한 것들이 남아있긴 하다.

## 1. 첫인상 (30%)

### 비주얼 퀄리티 (10%)
**점수: 7.5/10** (이전: 6/10, +1.5)

이모지 캐릭터에서 SVG 일러스트로 바뀐 게 확실히 체감된다. 전사의 검+방패, 마법사의 지팡이+별, 힐러의 날개+십자가, 레인저의 활+화살 — 각 클래스마다 고유한 실루엣이 있으니까 "내 캐릭터"라는 느낌이 확 살아난다. 힐러의 후광(halo)이 천천히 깜빡이는 애니메이션이나 마법사 지팡이 끝의 빛나는 구체도 세심하다. 홈 화면 캐릭터 카드에 glow 효과까지 있어서 존재감도 있다.

다만 "인스타에 올리고 싶을 정도"까지는 아직 아니다. SVG가 약간 단순한 도형 조합이라서, 쿠키런 캐릭터처럼 감정이입이 되거나 "이 아이 너무 귀여워" 하는 반응까지는 힘들다. 색상은 클래스별 그라데이션(전사=레드, 마법사=바이올렛, 힐러=에메랄드, 레인저=앰버)이 잘 나뉘어져서 미적으로 괜찮다. 이전 이모지 칼 하나보다는 100배 나아졌다.

### 온보딩 속도 (10%)
**점수: 9/10** (이전: 7/10, +2)

이번 온보딩은 정말 깔끔하다. 1단계: 클래스 선택(4개 카드 중 탭) → 2단계: 이름 입력 → 3단계: 루틴 선택(16개 중 4~12개). 3단계 다 합쳐도 1분 안에 끝난다. 온보딩이 3분 넘으면 "나중에 해야지"하고 영원히 안 하는 나한테 이건 완벽한 길이다.

클래스 카드마다 스탯 분배(체력 8/지력 3/정신 4/민첩 5)가 보여서 "나는 지력을 올리고 싶으니까 마법사!"처럼 전략적 선택이 가능하다. 각 클래스 설명도 한 줄로 핵심만 있어서 좋다. "이름을 입력하세요" 화면에서 선택한 클래스 SVG 아바타가 보이는 것도 일관성 있다. "클래스 다시 선택" 버튼으로 뒤로 가기도 가능하다.

루틴 선택 화면도 시간대별(아침/오후/저녁)로 분류되어 있고, 2열 그리드로 한눈에 보여서 고르기 편하다. 클래스 보너스 카테고리에 "+20%" 표시가 있어서 전략적으로 고를 수 있다. "4개 더 선택하세요" → "8개 루틴으로 시작!" 같은 동적 버튼 텍스트도 직관적이다.

### RPG 세계관 전달 (5%)
**점수: 8/10** (이전: 7/10, +1)

클래스 선택부터 시작하니까 "이건 게임이다"라는 첫인상이 확 강해졌다. 클래스별 스탯 차이, 카테고리별 XP 보너스(전사는 체력 루틴 +20%)까지 있어서 RPG 빌드 느낌이 난다. 쿠키런에서 쿠키 속성 고르는 것처럼, 클래스에 따라 전략이 달라지는 깊이가 생겼다. 다만 여전히 "스토리"는 없다 — 왜 모험을 떠나는지, 이 세계가 뭔지에 대한 서사는 빠져있다.

### 차별화 인지 (5%)
**점수: 7.5/10** (이전: 7/10, +0.5)

클래스 선택 + 루틴 커스터마이징 조합이 기존 습관 앱과의 차별점을 더 강하게 만들어준다. "전사 클래스로 시작해서 체력 루틴 위주로 선택"이라는 흐름은 Habitica에도 없는, 이 앱만의 경험이다. 클래스 보너스가 실제 XP 계산에 반영되는 것도 확인했다(코드에서 classBonus 로직). 단순 스킨이 아니라 시스템적 차이가 있다는 게 좋다.

**첫인상 소계: 8.0/10 (가중치 적용: 24.0/30)** (이전: 20.1/30, +3.9)

## 2. 사용성 (25%)

### 루틴 체크 속도 (8%)
**점수: 8/10** (이전: 8/10, 변동 없음)

이 부분은 이전에도 잘 되어 있었고 지금도 잘 된다. 퀘스트 화면에서 한 번 탭으로 완료, XP 획득 애니메이션 건재. 추가된 것은 클래스 보너스 힌트가 퀘스트 화면 하단에 표시되는 것 — "전사 클래스 보너스: 체력 루틴 XP +20%"가 보여서 어떤 루틴이 보너스인지 바로 알 수 있다.

### 루틴 커스터마이징 (7%)
**점수: 7/10** (이전: 3/10, +4)

드디어! 16개 루틴 라이브러리에서 4~12개를 내가 직접 고를 수 있다. 시간대별로 정리되어 있고, 카테고리 태그(체력/지력/정신력/사교)도 보이고, 클래스 보너스(+20%)까지 표시돼서 전략적으로 고를 수 있다. "아침 6:30 기상"이나 "자정 전 취침" 같은 내가 원하는 루틴이 라이브러리에 포함되어 있진 않지만, 비슷한 항목("기상 스트레칭", "디지털 디톡스")이 있어서 대체 가능하다.

아쉬운 점은 — 자유 입력이 안 된다. 16개 중에서만 고를 수 있고, 내 루틴을 직접 타이핑해서 추가할 수는 없다. 그리고 한 번 선택하면 온보딩 후에 변경할 수 없다. 루틴 추가/삭제/수정은 앱 내에서 불가능하다. 그래도 "아무것도 못 고른다"에서 "16개 중 고른다"로 바뀐 건 큰 진전이다.

### 진행 상황 한눈에 (5%)
**점수: 8/10** (이전: 8/10, 변동 없음)

여전히 잘 된다. 홈 대시보드에 캐릭터 카드(이제 SVG 아바타 포함), 스트릭, 달성률, 시간대별 퀘스트 현황 다 보인다. 프로필 화면도 건재.

### 알림 설정 (5%)
**점수: 1/10** (이전: 1/10, 변동 없음)

여전히 알림 없음. 이건 다음 이터레이션에서 꼭 다뤄야 한다.

**사용성 소계: 6.12/10 (가중치 적용: 15.3/25)** (이전: 12.7/25, +2.6)

## QA 이슈

### 통과 항목
- TypeScript 컴파일 에러 없음
- 온보딩 → 루틴선택 → 홈 네비게이션 플로우 정상
- localStorage 저장/불러오기 정상 (onboardingDone 플래그 체크)
- 클래스 보너스 XP 계산 로직 정상 (warrior-body, mage-mind, healer-soul, ranger-social 매칭)
- 레벨업 감지 로직 정상 (prevLevelRef 비교)
- 루틴 선택 화면 최소 4개/최대 12개 제한 정상

### 발견된 이슈
1. **LevelUpModal 파티클 애니메이션**: `animate-particle` keyframe이 정의되어 있고 동작하지만, 20개 파티클이 랜덤 위치에 생성되어 모두 위로 올라가기만 한다. 좌우 퍼지는 효과가 없어서 약간 단조롭다.
2. **스킬트리 해금 버튼 부재**: "해금 가능!" 텍스트는 뜨지만 클릭해서 해금하는 기능이 없다. SkillCard에 onClick 핸들러 없음.
3. **온보딩 후 루틴 변경 불가**: 한 번 선택하면 앱 내에서 루틴 구성을 바꿀 수 없다. localStorage를 직접 삭제하지 않는 한.
4. **길드 멤버 아바타**: DEMO_GUILD의 avatar가 여전히 이모지 문자열인 반면, 플레이어는 ClassAvatar SVG를 사용. 일관성 부재.
5. **친구 초대 버튼**: onClick 핸들러 없이 빈 button 태그.

## 개선된 것 vs 남은 것

### 개선됨 (Iteration 1 → 2)
1. **온보딩 플로우**: 없음 → 클래스 선택 + 이름 입력 (RPG 캐릭터 생성 경험)
2. **루틴 커스터마이징**: 고정 8개 → 16개 중 4~12개 선택 가능
3. **캐릭터 비주얼**: 이모지 → SVG 일러스트 (클래스별 고유 디자인)
4. **레벨업 보상감**: 없음 → 파티클 + 축하 모달 + 새 칭호

### 아직 부족한 것
1. **알림 시스템** — 여전히 없음 (가장 시급)
2. **루틴 자유 입력** — 라이브러리에서 선택만 가능, 직접 추가 불가
3. **스킬트리 실제 해금** — 보기만 가능
4. **스토리/세계관** — 서사 부재
5. **공유 기능** — 캐릭터 카드 이미지 내보내기/SNS 공유 없음
6. **레벨별 캐릭터 외형 변화** — Lv.1과 Lv.10이 같은 SVG

## 개선 가설 Top 3 (다음 이터레이션)

### 1. 루틴 자유 입력 + 온보딩 후 루틴 편집 (예상 임팩트: +5~7점)
- 16개 라이브러리 외에 직접 루틴을 추가하는 "커스텀 루틴 만들기" 기능
- 설정 또는 프로필에서 루틴 추가/삭제/재정렬 가능
- 이렇게 되면 사용성 "루틴 커스터마이징" 항목이 9~10점까지 올라갈 수 있음

### 2. 알림 시스템 (예상 임팩트: +6~8점)
- 시간대별 기본 알림 (아침 7시, 점심 12시, 저녁 8시)
- Web Notification API 또는 Service Worker 기반
- "내일 보스전 D-1!" 같은 게임 이벤트 알림
- 사용성 "알림 설정" 항목 + 재방문 의도에 직접적 영향

### 3. 스킬트리 해금 + 패시브 적용 (예상 임팩트: +4~6점)
- 조건 충족 시 스킬 카드 탭하면 실제 해금
- 해금된 패시브가 XP 계산, 스탯, 스트릭 보호 등에 반영
- 몰입감 "스킬트리 재미" 항목이 크게 오를 것

## 친구 추천 여부

**N (아직 추천하지 않음)**

이전보다 확실히 나아졌다. 클래스 선택하고 루틴 고르는 온보딩은 진짜 재미있었고, SVG 캐릭터도 이모지보다 100배 낫다. 레벨업 축하 모달도 "오!" 하는 순간이 있었다. 근데 친구한테 추천하려면... 알림이 없는 습관 앱을 추천하긴 어렵다. "이거 좋은데 알림이 없어서 직접 기억하고 열어야 해"라고 말하면 "그럼 뭐하러 깔아?"가 된다. 루틴 직접 추가도 안 되고. 점수가 60점대로 올라와서 "보류"로 바뀐 건 맞는데, "추천"까지는 아직 갈 길이 남았다.
