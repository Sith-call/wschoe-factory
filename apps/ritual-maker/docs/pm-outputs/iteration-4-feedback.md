- **알림 끄기가 실제로 동작하는지 의문** — NotificationToggle에서 enabled=false로 세팅하지만, App.tsx의 sendNotification은 이 토글 상태를 참조하지 않고 그냥 Notification.permission만 체크한다. 토글을 끄더라도 브라우저 권한이 granted면 계속 알림이 온다

그래도 없던 것에서 있는 것으로의 차이는 엄청나다. 6점.

## 3. 몰입감 (25%)

### 스킬트리 재미 (8%)
**점수: 9/10** (이전: 8/10, +1)

이전에 "패시브 효과 적용됨"이라고 표시만 되고 실제로는 적용 안 된다고 했었는데, 이번에 진짜로 적용된다! calculatePassiveBonus()가 모든 해금된 스킬을 순회하면서 실제 보너스를 계산하고, handleCompleteRitual에서 이 값을 XP 계산에 반영한다.

코드를 보면:
- `morningBonus = ritual.xpReward * passives.morningXpBonus` (아침 XP +10%)
- `xpGained = Math.floor(baseXp * passives.xpMultiplier)` (전체 XP 곱연산)

"새벽 전사" 스킬을 해금하면 아침 루틴에서 실제로 XP를 더 받는 거다. 이건 스킬 해금의 의미를 "비주얼 뱃지 수집"에서 "전략적 게임 메카닉"으로 완전히 바꿔놓는다. 쿠키런에서 토핑 세팅하는 느낌이랄까. "어떤 스킬을 먼저 찍어야 XP 효율이 좋을까?" 하는 고민이 생긴다.

프로필에 "적용 중인 패시브 효과" 섹션이 추가된 것도 좋다. "전체 XP x1.15", "아침 XP +10%", "체력 +2" 같은 걸 한눈에 볼 수 있어서, "내가 뭘 갖고 있는지" 파악이 된다.

다만 statBonuses(체력+2, 지력+1 등)는 calculatePassiveBonus에서 계산되지만, 실제로 character.stats에 반영되는 로직은 안 보인다. XP 보너스만 적용되고 스탯 보너스는 표시만 되는 것 같다.

### 보스배틀 긴장감 (7%)
**점수: 8/10** (이전: 7.5/10, +0.5)

보스 인트로에 스토리 텍스트가 추가된 게 큰 차이다. 슬라임 보스전에 들어가면 "매일 아침, 5분만 더...라는 속삭임이 당신의 의지를 녹이고 있습니다"라는 이야기가 나온다. 이건 단순히 "슬라임 HP 100, 약점: 체력"이었던 이전과 완전히 다른 경험이다. 내가 왜 이 보스를 물리쳐야 하는지 이해되니까 동기부여가 확 올라간다.

각 보스가 하나의 "나쁜 습관"을 상징하는 설정이 특히 좋다:
- 슬라임 = 나태 (아침 알람 미루기, 내 최대 적)
- 고블린 = 혼란 (할 일 산더미, 뭐부터 해야 할지 모르겠음)
- 드래곤 = 번아웃 (과도한 열정이 만든 그림자)
- 피닉스 = 완벽주의 (시작하지 못하는 두려움)

승리하면 "끈적한 나태가 사라지고, 몸이 가벼워집니다"라는 승리 서사 + "다음 장: 혼란의 고블린이 당신의 집중력을 노리고 있습니다"라는 예고편이 나온다. 다음 보스가 궁금해지는 장치가 확실히 있다.

### 캐릭터 성장 실감 (5%)
**점수: 7.5/10** (이전: 7/10, +0.5)

패시브 효과가 실제로 XP 계산에 반영되면서, "스킬을 찍으면 성장이 빨라진다"는 느낌이 생겼다. 프로필에서 패시브 목록을 볼 때 "전체 XP x1.15"를 보면 "내가 15% 더 빠르게 성장하고 있구나"라는 실감이 있다.

### 스토리/세계관 (5%)
**점수: 7/10** (이전: 3.5/10, +3.5)

이건 이번 이터레이션에서 가장 큰 점수 상승이다. 월드 인트로 + 보스 스토리 조합으로 완전한 서사 구조가 만들어졌다:

1. **세계관 설정** — 리추얼의 힘이 사라지고 네 그림자가 세상을 뒤덮음
2. **주인공의 역할** — 마지막 수호자로서 루틴의 힘을 되찾는 여정
3. **보스별 서사** — 각 보스가 구체적인 나쁜 습관을 상징, 왜 물리쳐야 하는지 명확
4. **챕터 연결** — 보스 처치 후 다음 챕터 예고

이전에는 "왜 보스를 때려야 하지?"라는 의문이 있었는데, 이제는 "나태의 슬라임이 내 게으름을 먹고 자란다니, 내가 루틴하면 슬라임이 약해지는 거구나"라는 연결이 생겼다. 이게 습관 앱에 서사가 주는 힘이다.

다만 아직 7점인 이유: 월드 인트로가 2페이즈(세계관 + 미션)뿐이라 좀 짧다. 각 보스 챕터 사이에 인터미션 스토리나, 보스 처치 후 세계가 변하는 비주얼 같은 게 있으면 10점에 더 가까워질 것 같다.

**몰입감 소계: 7.98/10 (가중치 적용: 19.95/25)** (이전: 16.8/25, +3.15)

- 기존 저장 데이터 있으면 worldIntro 건너뛰고 home 직행 정상

### 발견된 이슈
1. **알림 토글 상태와 실제 발송 불일치** — NotificationToggle에서 enabled=false로 세팅해도, App.tsx의 sendNotification은 Notification.permission만 체크하므로 토글 off 상태에서도 알림이 간다. 토글 상태를 state에 저장하고 sendNotification 호출 시 참조해야 한다.
2. **알림이 앱 열려있을 때만 동작** — setInterval 기반이라 브라우저/탭을 닫으면 알림 안 옴. Service Worker 기반으로 전환해야 진정한 푸시 알림이 됨.
3. **스탯 보너스 미적용** — calculatePassiveBonus의 statBonuses가 계산은 되지만, character.stats에 실제로 더해지는 로직 없음. XP 보너스만 적용됨.
4. **streakProtection 미적용** — calculatePassiveBonus에서 streakProtection=true 계산은 되지만, handleCompleteRitual의 streak 계산에서 이 값을 참조하지 않음.
5. **bossWeaknessBonus 미적용** — calculatePassiveBonus에서 bossWeaknessBonus 계산은 되지만, BossBattleScreen의 데미지 계산에서 사용하지 않음.
6. **길드 멤버 아바타 불일치** — DEMO_GUILD의 avatar가 여전히 이모지 문자열.
7. **친구 초대 버튼** — 여전히 onClick 핸들러 없음.

74점이면 솔직히 꽤 좋은 앱이다. 세계관 스토리가 생기면서 "이건 그냥 습관 앱이 아니라 RPG다"라는 확신이 들었고, 알림도 드디어 들어왔다. 패시브 효과가 실제로 XP에 반영되면서 스킬트리의 전략적 깊이도 올라갔다.

근데 80점까지 6점이 남았다. 공유 기능이 아직 없어서 "이거 봐봐!" 하고 인스타에 올릴 수 없고, 카카오톡으로 친구 초대도 안 되니까 "같이 하자"라고 말하기가 어렵다. 내가 추천을 하려면 최소한 보스전 승리 스크린샷을 예쁘게 공유할 수 있어야 한다. "나태의 슬라임을 물리쳤습니다!" 같은 이미지가 인스타 스토리에 올라간다고 생각해봐 — 그거 보면 "뭐야 이게 뭔 앱이야" 하고 다운받을 사람이 분명 있다.

다음 이터레이션에서 공유 기능이 들어오면 80점 넘길 수 있을 것 같다. 거의 다 왔어!
