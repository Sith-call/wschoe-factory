## 4. 공유성 (10%)

### SNS 공유 비주얼 (5%)
**점수: 8.5/10** (이전: 7/10, +1.5)

이것이 이번 이터레이션의 핵심이다. 저번에 "이미지가 아니라 텍스트로 나간다"가 공유성의 가장 큰 병목이었는데, 완전히 해결됐다.

잘 된 점:
1. **실제 이미지 내보내기** — cardToImage()가 canvas+SVG foreignObject로 카드를 2x 해상도 PNG로 캡처한다. Retina 대응까지 했다
2. **Web Share API with files** — `navigator.share({ files: [File] })`로 모바일에서 인스타/카톡에 이미지를 직접 공유할 수 있다. `canShare()` 체크도 있어서 지원 안 되면 다운로드로 fallback
3. **다운로드 fallback** — share API가 없거나 파일 공유 미지원 시 자동으로 PNG 다운로드. 데스크탑에서도 쓸 수 있다
4. **워터마크** — 카드 하단에 "#RitualMaker #루틴메이커"가 8px 크기로 들어간다. 누가 스토리에 올리면 앱 이름이 자연스럽게 노출된다
5. **두 개의 공유 버튼** — "📸 이미지로 저장/공유" (primary, amber) + "📋 텍스트로 공유" (secondary). 용도에 따라 선택 가능
6. **inline style** — background를 inline으로 잡아서 foreignObject 캡처 시 Tailwind class가 안 먹히는 문제를 우회

솔직히 말하면, 이제 인스타 스토리에 올릴 수 있다. "📸 이미지로 저장/공유" 누르면 → PNG가 생성되고 → 인스타 공유 시트가 뜨고 → 스토리에 바로 올린다. 스크린샷 찍을 필요 없이 깔끔한 9:16 카드가 나간다. 이건 진짜 된다.

다만 foreignObject 방식의 한계가 있다:
- **브라우저 호환성** — foreignObject는 일부 브라우저에서 보안 정책으로 막힐 수 있다. 특히 외부 폰트나 이미지 참조 시 tainted canvas 문제가 생길 수 있는데, ClassAvatar가 inline SVG라서 그 부분은 괜찮을 것 같다
- **스타일 누락 가능성** — Tailwind의 CSS class는 foreignObject 안에서 적용 안 될 수 있어서 inline style로 우회한 건 좋은 판단이지만, `rounded-2xl`, `text-white` 같은 class는 여전히 외부 CSS 의존이라 캡처 시 일부 스타일이 빠질 가능성이 있다
- html2canvas 라이브러리를 쓰면 더 안정적이겠지만, 외부 의존성 없이 구현한 건 이해한다

그래도 핵심 기능이 동작한다는 것 자체가 크다. 7점에서 8.5점으로 올린다. 완벽한 9점이 아닌 이유는 foreignObject 호환성 불안감 + 카드 비주얼 자체가 아직 "반드시 올리고 싶은" 수준은 아니라서. 일러스트 캐릭터가 있으면 9점 가능.

## QA 이슈

### 통과 항목
- cardToImage() 함수 — canvas+SVG foreignObject로 카드 캡처, 2x Retina 해상도 PNG 생성
- handleShareImage() — navigator.share({ files: [File] })로 이미지 파일 공유 + canShare() 체크 + 다운로드 fallback
- handleShareText() — navigator.share({ text }) + clipboard fallback (기존 텍스트 공유 유지)
- 두 개의 공유 버튼 — "📸 이미지로 저장/공유" (primary) + "📋 텍스트로 공유" (secondary)
- sharing 상태 — 이미지 생성 중 "준비 중..." 표시 + disabled 처리
- inline style background — foreignObject 캡처 호환성을 위해 background를 inline으로 적용
- 프로필 onShareProfile → setShareInfo({ type: 'profile' }) → ShareCardModal 열기 (App.tsx line 352)
- ShareCardModal의 세 가지 type (profile/levelUp/bossDefeat) 모두 이미지 내보내기 가능
- 카드 하단 "#RitualMaker #루틴메이커" 워터마크 표시

### 발견된 이슈
1. **foreignObject 호환성** — SVG foreignObject 기반 캡처는 일부 환경에서 보안 정책(tainted canvas)에 걸릴 수 있다. ClassAvatar가 inline SVG라 외부 이미지 참조는 없지만, Tailwind CSS class(`rounded-2xl`, `text-white`, `font-bold` 등)는 foreignObject 안에서 적용 안 될 수 있어서 캡처된 이미지가 원본과 약간 다를 수 있다.
2. **친구 초대 버튼** — 여전히 onClick 핸들러 없음.
3. **알림 토글 상태와 실제 발송 불일치** — 이전 이터레이션에서 발견된 이슈 미해결.
4. **길드 멤버 아바타 불일치** — 여전히 미해결.
5. **스트릭 보호 1일 제한** — 2일 이상 연속으로 빠지면 보호 안 됨. 사용자에게 안내 없음.
6. **카드 QR코드/앱 링크** — 공유 카드에 앱 다운로드 유도 장치가 없어서 바이럴 루프가 약함.
