## Iteration 3에서 구현한 사항

### 1. "한마디로" 한줄 요약
- 모든 8개 개념 상세 페이지 상단에 금색 강조 박스로 한줄 요약 표시
- 예: 수요/공급 → "가격이 오르면 사려는 사람은 줄고, 팔려는 사람은 늘어난다"
- 예: GDP → "나라 경제의 크기 = 소비 + 투자 + 정부지출 + 수출 - 수입"
- 예: 인플레이션 → "돈이 많아지면 물건값이 오른다"

### 2. 난이도 태그 (입문/중급/심화)
- 모든 개념 카드와 상세 페이지에 컬러 뱃지 표시
  - 입문 (초록): 수요/공급, GDP, 인플레이션
  - 중급 (금색): 탄력성, PPF, 비교우위, 승수효과
  - 심화 (빨강): IS-LM
- 용어 사전(ConceptListPage)에서도 제목 옆에 뱃지 표시

### 3. "뉴스에서 보면" 섹션
- 각 개념 상세 페이지에 실제 뉴스 헤드라인 기반 설명 추가
- 출처(한국경제, 연합뉴스, 한국은행 등)와 날짜 표시
- 뉴스 → 경제 원리 연결 설명 포함
- 예: "삼겹살 가격 또 올랐다" → 돼지 공급 감소 → 공급곡선 좌측 이동
- 예: "소비자물가 3.5% 상승" → 인플레이션 3.5%

### 4. 홈 화면 "오늘의 경제 뉴스 한줄" 카드
- 다크 배경 + 금색 악센트의 뉴스 카드
- "한국은행, 기준금리 3.5% 동결" 헤드라인
- "이게 무슨 뜻일까?" → 인플레이션 개념 페이지로 직접 연결
- 클릭 시 인플레이션 상세 페이지로 이동
