## 2. 화면별 상세

### 2.1 Home (홈)

**목적:** 앱의 중심. 무엇을 할 수 있는지 한눈에 보여주고, 빠른 진입을 제공.

**구성 요소:**
- **헤더:** 앱 로고 + 타이틀 "경제 실험실"
- **이어서 학습 섹션:** 마지막 학습 항목 카드 1개 (첫 방문 시 숨김)
- **추천 모델 섹션:** P0 모델 중 아직 안 해본 것 1-2개 카드
- **카테고리 그리드:** 미시경제, 거시경제, 국제경제, 재정/통화정책 (4칸 그리드)
  - 각 카테고리에 아이콘 + 이름 + 진도율(%)
- **하단 탭 바:** 홈, 용어, 실험실, 진도

**진입/이탈:**
- 진입: 앱 시작, 하단 탭 "홈"
- 이탈: 카테고리 탭 → ConceptList, 추천 모델 탭 → InteractiveLab, 이어서 학습 탭 → ConceptDetail 또는 InteractiveLab

---

### 2.2 ConceptList (용어 목록)

**목적:** 경제 용어를 카테고리별로 탐색. 원하는 용어를 찾아 카드를 열기.

**구성 요소:**
- **상단 카테고리 필터:** 가로 스크롤 칩 (전체, 미시, 거시, 국제, 정책)
- **검색 바:** 용어명 검색 (P1)
- **용어 카드 리스트:** 각 카드에
  - 용어명 (한글 + 영어)
  - 한 줄 정의
  - 카테고리 태그
  - 학습 완료 체크 표시
- **빈 상태:** 해당 카테고리에 용어가 없을 때 안내 메시지

**진입/이탈:**
- 진입: 홈 카테고리 탭, 하단 탭 "용어"
- 이탈: 용어 카드 탭 → ConceptDetail

---

### 2.3 ConceptDetail (용어 상세)

**목적:** 개별 경제 용어의 깊은 이해. 읽기 → 실험 전환 유도.

**구성 요소:**
- **헤더:** 뒤로가기 + 용어명
- **용어 정보 영역:**
  - 용어명 (한글, 큰 글자) + 영어명
  - 카테고리 태그
  - 한 줄 정의 (하이라이트)
- **쉬운 설명:** 일상 비유 중심 3-5문장
- **시각 자료:** 정적 그래프 또는 다이어그램 (SVG)
- **"실험실에서 직접 해보기" 버튼:** 해당 모델의 InteractiveLab으로 연결 (모델이 있는 경우에만 표시)
- **관련 용어:** 2-4개 카드 (가로 스크롤)
- **하단:** 학습 완료 표시 (자동 - 카드 열면 "열람" 처리)

**진입/이탈:**
- 진입: ConceptList에서 카드 탭, 관련 용어 탭, 홈 이어서 학습
- 이탈: "실험실로" 버튼 → InteractiveLab, 관련 용어 탭 → 다른 ConceptDetail, 뒤로가기 → ConceptList

---

### 2.4 InteractiveLab (인터랙티브 실험실)

**목적:** 경제 모델을 직접 조작하며 변수 간 관계를 체감.

**구성 요소:**
- **헤더:** 뒤로가기 + 모델명 + 리셋 버튼
- **그래프 영역 (화면 상단 60%):**
  - SVG/Canvas 기반 인터랙티브 그래프
  - 축 라벨 (X축, Y축)
  - 곡선/막대/파이 등 모델에 맞는 차트 타입
  - 균형점/교차점 하이라이트
  - 이동 전 곡선 점선 잔상 (비교용)
- **슬라이더 패널 (화면 하단 25%):**
  - 변수별 슬라이더 (1-4개)
  - 각 슬라이더에 변수명 + 현재 값 + 단위
  - 슬라이더 드래그 시 그래프 실시간 업데이트
- **설명 패널 (접이식):**
  - 현재 상태에 대한 동적 텍스트 설명
  - 변수값에 따라 내용 변경
  - 위로 스와이프하여 펼치기/접기

**모델별 화면 구성:**

| 모델 | 그래프 타입 | 슬라이더 |
|------|-----------|---------|
| 수요와 공급 | XY 곡선 (수요/공급) | 가격, [소득, 기술] |
| 탄력성 | XY 곡선 + 바 차트 | 가격 변화율, 재화 유형 선택 |
| GDP 구성요소 | 스택 바 차트 | C, I, G, X-M |
| 인플레이션-통화량 | 시계열 라인 차트 | 통화량(M) |

**진입/이탈:**
- 진입: ConceptDetail "실험실로" 버튼, 홈 추천 모델 탭, 하단 탭 "실험실" → 모델 선택
- 이탈: 뒤로가기 → 이전 화면, "용어 카드 보기" → ConceptDetail, 관련 모델 추천 탭 → 다른 InteractiveLab

---

### 2.5 ModelSimulator (모델 시뮬레이터) — P1

**목적:** 복합 모델 탐색 + 시나리오 기반 학습.

**구성 요소:**
- **헤더:** 뒤로가기 + 모델명
- **모델 선택 탭:** IS-LM, 승수 효과, PPF, 비교우위
- **그래프 영역:** InteractiveLab과 동일 구조이나 복합 변수
- **시나리오 모드 토글:**
  - ON: 시나리오 목록에서 선택 → 변수 자동 설정 → 결과 비교
  - OFF: 자유 조작 모드
- **시나리오 카드:**
  - 시나리오 제목 ("정부가 세금을 10% 올리면?")
  - 변화 전/후 비교 뷰
- **인과관계 체인:** A → B → C 시각화 (간단한 플로우)

**진입/이탈:**
- 진입: 하단 탭 "실험실" → 모델 선택, InteractiveLab 관련 모델 추천
- 이탈: 뒤로가기, 다른 모델 선택

---

### 2.6 Progress (학습 진도)

**목적:** 학습 현황 파악 + 동기 부여.

**구성 요소:**
- **전체 진도 요약:**
  - 원형 프로그레스: 전체 완료율(%)
  - "열어본 용어: 12/20" | "실험한 모델: 3/4"
- **카테고리별 진도:**
  - 가로 막대 차트 (미시: 80%, 거시: 40%, 국제: 0%, 정책: 20%)
  - 각 막대 탭 시 해당 카테고리의 ConceptList로 이동
- **최근 학습 이력:**
  - 날짜별 그룹핑
  - 각 항목: 시간 + 용어/모델명 + 유형(카드 열람/실험)
- **연속 학습 표시:** "3일 연속 학습 중" (선택적)

**진입/이탈:**
- 진입: 하단 탭 "진도", 홈에서 진도 영역 탭
- 이탈: 카테고리 막대 탭 → ConceptList, 이력 항목 탭 → ConceptDetail/InteractiveLab

---

### 2.7 Settings (설정) — P2

**목적:** 앱 관리 기능.

**구성 요소:**
- **학습 데이터:**
  - 저장된 데이터 용량 표시
  - "학습 진도 초기화" 버튼 → 확인 다이얼로그 → 초기화
- **앱 정보:**
  - 버전
  - 경제 모델 출처 (맨큐 경제학 등)
- **문의/피드백:** 이메일 링크 (선택적)

**진입/이탈:**
- 진입: Progress 화면의 설정 아이콘, 또는 홈 헤더 설정 아이콘
- 이탈: 뒤로가기 → 이전 화면

## 3. 화면 전환 플로우

### 3.1 전체 네비게이션 구조

```
┌─────────────────────────────────────────────────┐
│                하단 탭 바 (전역)                   │
│   [홈]     [용어]     [실험실]     [진도]          │
└──┬──────────┬──────────┬──────────┬──────────────┘
   │          │          │          │
   ▼          ▼          ▼          ▼
  Home    ConceptList  LabIndex   Progress
   │          │          │          │
   │          ▼          ▼          │
   │     ConceptDetail  InteractiveLab ──→ Settings
   │          │    ▲     │
   │          ▼    │     ▼
   │     InteractiveLab  ModelSimulator (P1)
   │          │
   └──────────┘ (이어서 학습, 추천 모델)
```

### 3.2 주요 유저 플로우

#### 플로우 A: 용어 학습 → 실험 (가장 일반적)
```
Home → ConceptList → ConceptDetail → InteractiveLab
 (1)       (2)            (3)              (4)

(1) 홈에서 카테고리 선택 또는 하단 탭 "용어"
(2) 용어 목록에서 관심 있는 용어 탭
(3) 용어 카드 읽기 → "실험실에서 직접 해보기" 버튼 탭
(4) 슬라이더 조작 → 그래프 변화 관찰
```

#### 플로우 B: 실험 직행
```
Home → InteractiveLab
 (1)        (2)

(1) 홈에서 추천 모델 카드 탭 또는 하단 탭 "실험실"
(2) 바로 슬라이더 조작 시작
```

#### 플로우 C: 재방문 이어서 학습
```
Home → ConceptDetail 또는 InteractiveLab
 (1)              (2)

(1) 홈 "이어서 학습하기" 섹션의 카드 탭
(2) 마지막 학습 위치로 바로 이동
```

#### 플로우 D: 연관 탐색 (위키 스타일)
```
ConceptDetail → ConceptDetail → ConceptDetail → InteractiveLab
    (수요)    →    (탄력성)   →   (총수입)    →   (탄력성 실험)

관련 용어를 따라가며 지식 확장 → 실험으로 체감
```

#### 플로우 E: 진도 기반 학습
```
Progress → ConceptList → ConceptDetail
  (1)          (2)            (3)

(1) 진도 화면에서 완료율 낮은 카테고리의 막대 탭
(2) 해당 카테고리의 미완료 용어 확인
(3) 미완료 용어 카드 학습
```

#### 플로우 F: 시나리오 학습 (P1)
```
Home → ModelSimulator → 시나리오 선택 → 결과 비교
 (1)        (2)              (3)            (4)

(1) 하단 탭 "실험실" → 모델 선택
(2) 시나리오 모드 토글 ON
(3) "정부가 세금을 올리면?" 선택
(4) 변화 전/후 그래프 비교
```

## 4. 하단 탭 바 구성

| 탭 | 아이콘 | 라벨 | 연결 화면 |
|----|-------|------|----------|
| 1 | 집 모양 | 홈 | Home |

- 현재 활성 탭은 색상 변경으로 표시
- 탭 전환 시 이전 탭의 스크롤 위치 유지

## 5. 실험실 모델 목록 (실험실 탭 진입 시)

하단 탭 "실험실" 탭 시 보이는 모델 선택 화면:

**구성:**
- **P0 모델 섹션:** "기본 모델"
  - 수요와 공급
  - 탄력성
  - GDP 구성요소
  - 인플레이션과 통화량
- **P1 모델 섹션:** "심화 모델" (P1 구현 후)
  - 기회비용과 PPF
  - 비교우위와 무역
  - 승수 효과
  - IS-LM 모델
- 각 모델 카드에 완료 여부 표시
- 미완료 모델에 강조 표시

탭 시 → InteractiveLab (P0) 또는 ModelSimulator (P1)

## 6. 상태별 화면 변화

### 6.1 첫 방문 vs 재방문

| 요소 | 첫 방문 | 재방문 |
|------|---------|-------|
| 홈 "이어서 학습" | 숨김 | 마지막 항목 표시 |
| 홈 추천 모델 | P0 첫 번째 모델 | 아직 안 해본 모델 |
| 카테고리 진도율 | 모두 0% | 실제 진도 반영 |
| 진도 화면 | "아직 학습 기록이 없어요" | 실제 이력 표시 |

### 6.2 모든 P0 모델 완료 시

- 홈 추천 모델 → P1 모델 또는 "모든 기본 모델 완료!" 축하 카드
- 진도 화면 → P0 카테고리 100% 표시
- 실험실 탭 → P0 모델 완료 체크, P1 모델 강조

## 7. 반응형 레이아웃 가이드

### 모바일 (< 768px)
- 하단 탭 바 고정
- 그래프 영역: 화면 상단 55-60%
- 슬라이더: 하단 세로 배치
- 설명 패널: 접이식 (기본 접힘)
- 용어 목록: 1열 리스트

### 태블릿/데스크톱 (>= 768px)
- 좌측 사이드 네비게이션 (또는 상단 탭)
- 그래프 영역: 좌측 60%
- 슬라이더 + 설명: 우측 40% 사이드 패널
- 용어 목록: 2-3열 그리드
