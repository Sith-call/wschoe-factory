# Domain Expert Review — 경제 실험실

## 검수 소스
- 경제금융용어 700선 (한국은행)
- Principles of Economics 9th Edition (Mankiw)

| 모델 | 정확도 | 핵심 이슈 |
|------|--------|----------|
| 수요/공급 | ⚠️ | 시나리오1 범주 오류 — 기준금리(거시)를 미시 모델에 대입 |
| 탄력성 | ✅ | 시간 경과 요인 추가 권장 |

## 2. 추가 권장 모델 (우선순위순)

### 1순위: 총수요-총공급(AD-AS) 모형
- **근거**: 기준금리 시나리오의 범주 오류 해결. 미시→거시 연결 고리
- **시뮬레이션**: 통화정책/재정정책 → AD 이동, 유가 충격 → SRAS 이동 → 스태그플레이션
- **연결**: 수요/공급(미시 확장), 인플레이션(물가 결정), IS-LM(정책 효과)

### 2순위: 실질 GDP와 잠재 GDP (GDP 갭)
- **근거**: GDP 모델의 심화. 경제 안정화 정책의 필요성 이해
- **시뮬레이션**: 명목/실질 전환 스위치, 잠재 GDP 대비 현재 수준 계기판
- **연결**: GDP 구성요소, 승수효과, IS-LM

### 3순위: 구축효과 (Crowding-out Effect)
- **근거**: 승수효과의 현실적 한계. 정부 지출 → 이자율↑ → 민간 투자↓
- **시뮬레이션**: G 슬라이더↑ → GDP 1차 팽창 → 이자율↑ → I 자동 감소
- **연결**: GDP(I 감소), 승수효과(약화), IS-LM(이자율)

### 4순위: 유동성 함정 (Liquidity Trap)
- **근거**: IS-LM의 극단적 위기 시나리오. 통화정책의 한계
- **시뮬레이션**: 이자율 0% → LM 수평 → 통화정책 무력화 → 재정정책만 작동
- **연결**: IS-LM(예외 상황), 인플레이션(디플레이션)

### 5순위: 매몰비용 (Sunk Cost)
- **근거**: 일상생활 의사결정에 가장 실용적인 미시 개념
- **시뮬레이션**: 영화표 예매/프로젝트 중단 의사결정 미니게임
- **연결**: 기회비용(확장), 한계적 사고

## 3. 즉시 수정 필요 항목

1. 수요/공급 시나리오1: "기준금리 인상" → 미시적 요인으로 교체 또는 AD-AS 모델로 이동
2. GDP 설명: 명목/실질 GDP 구분 추가
3. 인플레이션: 단기/장기 효과 구분 주의사항 추가
4. 승수효과: 구축효과 언급 추가
