## Iteration 2에서 구현한 사항

### 1. 경제 모델 4개 추가 (총 8개)
- **기회비용과 생산가능곡선 (PPF)**: 오목한 PPF 곡선, 기술 수준/자원 배분 슬라이더
- **비교우위와 무역**: 두 국가 PPF 비교, 무역 후 소비가능 영역 확대 시각화
- **승수 효과**: 연쇄 소비 라운드 막대 차트, MPC/정부지출 슬라이더
- **IS-LM 모델**: IS곡선(우하향)/LM곡선(우상향) 교차, 재정/통화정책 시뮬레이션

### 2. 개념 설명 보강
- 모든 8개 concept의 `coreExplanation`을 200자 이상으로 확장
- 핵심 포인트, 실생활 연결, 학술적 배경 포함

### 3. 시나리오 기반 학습 (16개 시나리오)
- 각 모델에 2개씩 실생활 시나리오 프리셋 추가
- 시나리오 선택 시 슬라이더 자동 설정 + 상세 해설 표시
- 예: "기준금리 0.5%p 인상", "코로나 봉쇄로 소비 30% 감소", "양적완화로 통화량 2배" 등

### 4. 퀴즈 모드 (24문제)
- 각 개념에 3문제 객관식 퀴즈
- 정답/오답 시 시각적 피드백 (초록/빨강)
- 정답 해설 표시
- 전체 정답 수 카운터

### 5. 그래프 엔진 확장
- MultiplierBarChart: 승수 효과 전용 수직 막대 차트
- 모델별 축 라벨 커스터마이징 (IS-LM: 이자율/국민소득, PPF: 재화A/재화B 등)
