# PRD: 인맥 텃밭 (Relation Garden)

> 소중한 사람들과의 관계를 식물로 키우는 관계 관리 앱

**버전:** 1.0
**작성일:** 2026-03-20
**상태:** Draft

---

## 1. 문제 정의 (Problem Statement)

### 배경

한국 20-30대 직장인은 바쁜 일상 속에서 소중한 인간관계를 방치하기 쉽다. "연락해야지" 하면서도 미루다가 어느새 서먹해진 친구, 고마운 마음을 전하지 못한 동료, 멀어진 가족. 관계는 눈에 보이지 않기에 악화되고 있어도 깨닫기 어렵다.

### 핵심 문제

1. **관계 방치 무감각:** 연락이 뜸해지고 있다는 걸 인지하지 못한다. 갑자기 "왜 연락 안 했지?" 하고 깨닫는다.
2. **연락 부채감:** 오래 연락 못 한 사람에게 먼저 연락하기가 심리적으로 어렵다. "뭐라고 연락하지?"
3. **관계 투자 편향:** 자주 만나는 사람에게만 에너지를 쏟고, 정말 소중한 사람은 오히려 소홀해진다.
4. **감사 표현 부재:** 고마운 마음, 그리운 마음이 있어도 표현할 계기가 없다.
5. **관계 건강 비가시성:** 내 인맥의 전체적인 건강 상태를 한눈에 볼 수 없다.

### 가설

> 인간관계를 "식물 키우기"로 메타포화하면, 관계 관리의 심리적 장벽을 낮추고 지속적인 케어 동기를 부여할 수 있다. 연락 = 물주기, 감사 메모 = 비료, 방치 = 시들기라는 직관적 피드백으로 **한 달 내 소홀한 관계 3개 이상 재활성화**할 수 있다.

## 2. 타겟 유저 (Target Users)

### 주요 타겟

| 항목 | 내용 |
|------|------|
| **연령** | 25-35세 |
| **직업** | 직장인, 프리랜서 |
| **특성** | 인간관계를 소중히 여기지만 관리에 서툰 사람 |
| **디지털 습성** | 카카오톡/인스타 등 SNS 활발 사용 |
| **핵심 니즈** | 소중한 관계를 유지하고 싶지만 방법을 모르겠는 사람 |

### 3 페르소나 요약

#### 페르소나 1: 김도연 (29세, 마케팅 대리)

- 대학 친구 10명 중 정기적으로 연락하는 사람이 3명으로 줄어든 상태
- "바빠서" 가 핑계인 걸 알지만 먼저 연락하기 민망해서 계속 미루는 중
- SNS로 근황은 보지만 DM 한 줄 보내기가 어렵다
- **니즈:** 연락 타이밍 넛지 + 뭐라고 말할지 아이디어

#### 페르소나 2: 이준서 (32세, 개발자)

- 직장 동료 외에는 인간관계가 거의 없다고 느낌
- 가족(부모님, 형)에게 감사하지만 표현을 잘 못 함
- 관계의 "현황판"이 있으면 좋겠다고 생각
- **니즈:** 관계 시각화 + 감사 기록 아카이빙

#### 페르소나 3: 박서현 (27세, 디자이너)

- 사회생활 초기라 다양한 관계가 형성되는 중
- 새로운 사람을 만나면 신나지만, 관계 유지 에너지가 부족
- 친구들과 "관계 점수"를 공유하며 재미있게 관리하고 싶음
- **니즈:** 관계 정원 공유 + 게이미피케이션 요소

## 3. 핵심 기능 (Core Features) — 8 스크린

### Screen 1: GardenHomeScreen (내 텃밭)
- 내 관계 텃밭을 한눈에 보는 메인 화면
- 식물들이 심어진 정원 뷰 (그리드/리스트 토글)
- 각 식물은 관계 상태에 따라 시각적으로 변화 (싹 → 새싹 → 꽃 → 나무 / 시들기)
- 상단: 정원 이름, 식물 수, 전체 건강도 게이지
- 하단 네비게이션: 텃밭 | 기록 | 분석 | 설정
- "시들어가는 관계" 알림 배지

### Screen 2: PlantDetailScreen (관계 상세)
- 특정 관계(식물)의 상세 정보
- 식물 성장 단계 일러스트 (5단계: 씨앗 → 새싹 → 줄기 → 꽃 → 나무)
- 관계 건강 점수 (0-100)
- 마지막 연락 일자 + "D+N일째 물 안 줌" 표시
- 관계 태그: 가족/친구/동료/멘토/연인
- 감사 메모 타임라인 (과거 기록 스크롤)
- 액션 버튼: 물주기(연락 기록) | 비료주기(감사 메모) | 햇빛(추억 기록)

### Screen 3: WateringScreen (물주기 — 연락 기록)
- 연락 유형 선택: 전화 / 문자 / 만남 / SNS / 선물
- 연락 품질 평가: 가벼운 안부 → 깊은 대화 (1-5)
- 기분 태그: 즐거웠다 / 고마웠다 / 아쉬웠다 / 그리웠다
- 한줄 메모 (선택)
- 기록 완료 시 식물 성장 애니메이션
- 연락 유형별 물의 양이 달라짐 (만남 > 전화 > 문자 > SNS)

### Screen 4: GratitudeScreen (비료주기 — 감사 메모)
- "OO에게 감사한 점" 작성 폼
- 감사 카테고리: 위로 / 도움 / 응원 / 함께 / 배움 / 웃음
- 추억 날짜 선택 (그때 그 일)
- 감사 편지 스타일로 꾸미기 (배경색 선택)
- 작성 완료 시 비료 효과 + 식물 건강 UP
- 공유 가능: 카카오톡으로 감사 카드 보내기

### Screen 5: AddPlantScreen (새 식물 심기)
- 새로운 관계를 정원에 추가
- 이름, 관계 유형(가족/친구/동료/멘토/기타), 별명 입력
- 식물 종류 선택 (장미/해바라기/선인장/대나무/벚꽃/허브 — 6종)
- 목표 연락 주기 설정: 매일/매주/2주/매월/분기
- 첫 번째 감사 메모 작성 (온보딩)
- "이 사람과의 첫 번째 기억은?" 프롬프트

### Screen 6: JournalScreen (관계 일기)
- 날짜별 관계 활동 타임라인
- 캘린더 뷰 + 리스트 뷰 토글
- 각 날짜에 물주기/비료/추억 기록 아이콘 표시
- 월별 관계 활동 히트맵 (잔디밭 스타일)
- 연속 관계 케어 스트릭 표시
- "이번 주 가장 많이 돌본 관계" 하이라이트

### Screen 7: InsightScreen (정원 분석)
- 관계 건강 대시보드
- 관계 유형별 분포 도넛 차트 (가족/친구/동료/멘토)
- 가장 건강한 관계 TOP 3 / 시들어가는 관계 TOP 3
- 월간 연락 트렌드 그래프
- "관계 성격 진단" — 내가 어떤 유형의 관계 관리자인지 (정원사 타입)
  - 꼼꼼한 정원사 / 자유방임 정원사 / 폭풍 관리형 / 선택과 집중형
- 이번 달 관계 리포트 카드 (공유 가능)

### Screen 8: ShareCardScreen (공유 카드)
- 내 관계 정원 스냅샷 공유용 카드 생성
- 정원 전체 일러스트 축소판
- 핵심 통계: 관계 수, 평균 건강도, 가장 오래 가꾼 관계
- 정원사 타입 뱃지
- 이미지/링크 공유 (카카오톡, 인스타 스토리)

## 4. 핵심 메카닉

### 식물 성장 시스템
| 단계 | 이름 | 조건 | 시각적 |
|------|------|------|--------|
| 1 | 씨앗 | 새로 등록 | 작은 흙더미 |
| 2 | 새싹 | 첫 물주기 완료 | 초록 새싹 |
| 3 | 줄기 | 연락 3회 이상 | 잎이 달린 줄기 |
| 4 | 꽃 | 연락 7회 + 감사메모 2회 | 꽃이 핀 상태 |
| 5 | 나무 | 연락 15회 + 감사메모 5회 | 풍성한 나무 |

### 시들기 시스템
| 상태 | 조건 | 시각적 변화 |
|------|------|-------------|
| 건강 | 목표 주기 내 연락 | 생기 있는 색상 |
| 주의 | 목표 주기 1.5배 초과 | 약간 누렇게 |
| 위험 | 목표 주기 2배 초과 | 많이 시들음 |
| 위독 | 목표 주기 3배 초과 | 거의 마름 |

> 시들어도 물을 주면 다시 살아남 (관계 회복 메타포)

### 정원사 타입 진단
연락 패턴 데이터 기반 자동 진단:
- **꼼꼼한 정원사:** 고르게 모든 관계를 관리
- **자유방임 정원사:** 연락이 불규칙하지만 깊은 대화
- **폭풍 관리형:** 한꺼번에 몰아서 연락
- **선택과 집중형:** 소수에게 집중적으로 투자

## 5. Success Metrics

### Primary Metrics
| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 등록 관계 수 | 평균 7개 이상 | localStorage 데이터 |
| 주간 활성 사용률 | 60% (4주 기준) | 주 1회 이상 물주기 |
| 공유 전환율 | 20% | 공유 카드 생성 비율 |
| 시들어가는 관계 재활성화 | 월 3건 이상 | 위험→건강 전환 |

### Secondary Metrics
| 지표 | 목표 |
|------|------|
| 감사 메모 작성률 | 등록 관계의 50% |
| 정원사 타입 공유율 | 30% |
| 세션 평균 시간 | 3분 이상 |

- [ ] 관계별 기념일 등록 + 리마인더

## 7. 차별화 포인트

| 기존 앱 | 인맥 텃밭 차별점 |
|---------|------------------|
| dream-factory (꿈 해몽) | 관계 도메인, 외부 지향적 (타인과의 관계) |
| ritual-maker (습관 RPG) | 게이미피케이션이지만 "관계 케어"라는 새로운 축 |
| mood-pharmacy (감정 처방) | 내면 감정이 아닌 "관계"라는 사회적 영역 |
| 일반 연락처 앱 | 단순 저장이 아닌 관계의 건강/성장 시각화 |
| CRM 앱 | 비즈니스가 아닌 개인 관계, 감성적 접근 |

### 바이럴 요소
1. **정원사 타입 공유:** "나는 자유방임 정원사 유형!" — MBTI처럼 공유 욕구
2. **관계 정원 스냅샷:** 내 정원 모습 자체가 콘텐츠 (꽃 가득 vs 시든 정원)
3. **감사 카드 전송:** 감사 메모를 카카오톡으로 보내면 받는 사람도 유입
4. **관계 건강 리포트:** 월간 리포트 공유 — "이달의 MVP 관계"
