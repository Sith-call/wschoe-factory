# 유저 스토리: 인맥 텃밭 (Relation Garden)

---

## US-01: 정원 첫 방문 (온보딩) [P0]

**As a** 처음 방문한 사용자
**I want to** 앱의 컨셉과 사용법을 빠르게 이해하고 첫 번째 식물을 심고 싶다
**So that** 앱 사용을 바로 시작할 수 있다

### 수락 기준
- [ ] 앱 진입 시 3단계 온보딩 카드 표시 (정원 소개 → 식물 = 관계 → 물주기 = 연락)
- [ ] 온보딩 완료 후 "첫 번째 식물 심기" 화면으로 자동 이동
- [ ] 온보딩은 최초 1회만 표시, 이후 바로 GardenHome 진입
- [ ] "건너뛰기" 옵션 제공

---

## US-02: 새 식물 심기 (관계 등록) [P0]

**As a** 사용자
**I want to** 소중한 사람을 정원에 식물로 등록하고 싶다
**So that** 그 관계를 시각적으로 관리할 수 있다

### 수락 기준
- [ ] 이름(필수), 관계 유형(필수), 별명(선택) 입력
- [ ] 관계 유형: 가족/친구/동료/멘토/기타 중 택 1
- [ ] 식물 종류 6종 중 선택 (장미/해바라기/선인장/대나무/벚꽃/허브)
- [ ] 목표 연락 주기 설정: 매일/매주/2주/매월/분기
- [ ] "이 사람과의 첫 번째 기억은?" 프롬프트 표시 (선택 입력)
- [ ] 등록 완료 시 씨앗 심기 애니메이션 재생
- [ ] localStorage에 관계 데이터 저장

---

## US-03: 정원 전체 보기 [P0]

**As a** 사용자
**I want to** 내 모든 관계를 정원 형태로 한눈에 보고 싶다
**So that** 어떤 관계가 건강하고 어떤 관계가 시들고 있는지 빠르게 파악할 수 있다

### 수락 기준
- [ ] 그리드 뷰: 식물 아이콘 + 이름 + 건강 상태 색상 표시
- [ ] 리스트 뷰: 이름, 관계 유형, 마지막 연락일, 건강도 바
- [ ] 뷰 전환 토글 버튼 (그리드/리스트)
- [ ] 상단에 정원 요약: 총 식물 수, 평균 건강도, 시들어가는 관계 수
- [ ] "시들어가는 관계" 필터 버튼 (위험/위독 상태만 표시)
- [ ] 식물 탭 시 PlantDetailScreen으로 이동
- [ ] 우측 하단 FAB: "새 식물 심기" 버튼

---

## US-04: 물주기 (연락 기록) [P0]

**As a** 사용자
**I want to** 누군가에게 연락한 것을 "물주기"로 기록하고 싶다
**So that** 식물이 성장하고 관계 건강이 유지된다

### 수락 기준
- [ ] 연락 유형 선택: 전화(물 많이) / 만남(물 가장 많이) / 문자(물 보통) / SNS(물 조금) / 선물(물+비료)
- [ ] 연락 품질 1-5 슬라이더: "가벼운 안부" ~ "깊은 대화"
- [ ] 기분 태그 1개 이상 선택: 즐거웠다/고마웠다/아쉬웠다/그리웠다
- [ ] 한줄 메모 입력 (선택, 100자 이내)
- [ ] 물주기 완료 시 식물 반응 애니메이션 (물방울 + 성장)
- [ ] 관계 건강 점수 업데이트
- [ ] 성장 조건 충족 시 성장 단계 업 + 축하 메시지

---

## US-05: 비료주기 (감사 메모) [P0]

**As a** 사용자
**I want to** 누군가에 대한 감사한 마음을 "비료"로 기록하고 싶다
**So that** 감사 기억이 쌓이고 식물이 더 건강해진다

### 수락 기준
- [ ] 감사 카테고리 선택: 위로/도움/응원/함께/배움/웃음
- [ ] 감사 내용 자유 입력 (200자 이내, 필수)
- [ ] 추억 날짜 선택 (기본값: 오늘)
- [ ] 배경색 4종 중 선택 (감사 카드 스타일링)
- [ ] 비료주기 완료 시 반짝이 애니메이션
- [ ] 감사 메모는 PlantDetail의 타임라인에 누적 표시
- [ ] "카카오톡으로 보내기" 공유 버튼 (텍스트 + 이미지)

---

## US-06: 관계 상세 보기 [P0]

**As a** 사용자
**I want to** 특정 관계의 히스토리와 현재 상태를 상세히 보고 싶다
**So that** 그 관계에 대한 전체 그림을 파악할 수 있다

### 수락 기준
- [ ] 식물 성장 단계 일러스트 (현재 단계 강조, 전체 5단계 표시)
- [ ] 관계 건강 점수 (0-100) + 상태 라벨 (건강/주의/위험/위독)
- [ ] 마지막 연락일 + "D+N일" 표시
- [ ] 다음 연락 권장일 표시 (목표 주기 기반)
- [ ] 감사 메모 + 연락 기록 통합 타임라인 (최신순)
- [ ] 하단 액션 바: 물주기 | 비료주기 | 수정 | 삭제
- [ ] 삭제 시 확인 모달 + "정말 이 관계를 정원에서 뽑으시겠어요?" 메시지

---

## US-07: 관계 일기 (캘린더 뷰) [P1]

**As a** 사용자
**I want to** 날짜별로 내 관계 활동을 돌아보고 싶다
**So that** 관계 케어 패턴을 인지하고 개선할 수 있다

### 수락 기준
- [ ] 월간 캘린더 뷰: 각 날짜에 활동 도트 표시 (물주기=파랑, 비료=초록, 추억=노랑)
- [ ] 날짜 탭 시 해당 날의 활동 리스트 표시
- [ ] 리스트 뷰 토글: 전체 활동을 시간순 스크롤
- [ ] 활동 히트맵: 잔디밭 스타일 (활동 많은 날 = 진한 초록)
- [ ] 연속 케어 스트릭 표시 ("N일 연속 관계 가꾸는 중")
- [ ] 월 전환 가능 (이전/다음 월)

---

## US-08: 정원 분석 대시보드 [P1]

**As a** 사용자
**I want to** 내 관계 관리 패턴을 데이터로 분석해서 보고 싶다
**So that** 관계 관리 습관을 개선할 인사이트를 얻을 수 있다

### 수락 기준
- [ ] 관계 유형별 분포 도넛 차트 (가족/친구/동료/멘토/기타)
- [ ] 가장 건강한 관계 TOP 3 (식물 아이콘 + 점수)
- [ ] 시들어가는 관계 TOP 3 (경고 아이콘 + D+ 표시)
- [ ] 월간 연락 트렌드 바 차트 (주별 연락 횟수)
- [ ] 정원사 타입 진단 결과 카드 (데이터 기반 자동 판정)
- [ ] "이번 달 리포트 공유하기" 버튼

---

## US-09: 정원사 타입 진단 [P1]

**As a** 사용자
**I want to** 나의 관계 관리 스타일을 재미있게 진단받고 싶다
**So that** 자기 이해도를 높이고 결과를 친구에게 공유할 수 있다

### 수락 기준
- [ ] 물주기 패턴 데이터 기반 자동 진단 (최소 물주기 5회 이상 시 활성화)
- [ ] 4가지 타입: 꼼꼼한 정원사 / 자유방임 정원사 / 폭풍 관리형 / 선택과 집중형
- [ ] 타입별 설명 카드 (성격, 장점, 개선 포인트)
- [ ] 타입 결과 공유 카드 생성 (이미지 형태)
- [ ] 데이터 부족 시 "아직 더 가꿔야 해요" 안내 표시

---

## US-10: 공유 카드 생성 [P1]

**As a** 사용자
**I want to** 내 관계 정원 상태를 예쁜 카드로 만들어 공유하고 싶다
**So that** 친구들에게 자랑하고 앱을 소개할 수 있다

### 수락 기준
- [ ] 정원 스냅샷 카드: 식물 미니어처 배치 + 통계
- [ ] 포함 정보: 관계 수, 평균 건강도, 가장 오래 가꾼 관계, 정원사 타입
- [ ] 카드 배경 테마 3종 선택 (봄/여름/가을)
- [ ] "이미지 저장" + "카카오톡 공유" + "링크 복사" 옵션
- [ ] 공유 카드에 앱 브랜딩 워터마크 포함

---

## US-11: 시들어가는 관계 알림 [P1]

**As a** 사용자
**I want to** 소홀해지고 있는 관계를 미리 알려줘서 대응하고 싶다
**So that** 관계가 완전히 식어버리기 전에 연락할 수 있다

### 수락 기준
- [ ] GardenHome 진입 시 시들어가는 관계 배너 표시 (주의/위험/위독)
- [ ] 배너 탭 시 해당 식물 상세로 이동
- [ ] 시들기 단계별 다른 메시지 ("슬슬 목마른 것 같아요" → "많이 시들었어요" → "곧 마를 것 같아요")
- [ ] "지금 물주러 가기" CTA 버튼

---

## US-12: 식물 성장 축하 [P2]

**As a** 사용자
**I want to** 식물이 새로운 단계로 성장하면 축하받고 싶다
**So that** 관계 관리에 대한 보람과 동기를 느낀다

### 수락 기준
- [ ] 성장 단계 업 시 풀스크린 축하 모달 표시
- [ ] 성장 전후 비교 일러스트 (before → after)
- [ ] 다음 단계까지의 조건 안내
- [ ] "성장 기념 공유하기" 버튼
- [ ] 최고 단계(나무) 달성 시 특별 축하 + 뱃지

---

## US-13: 관계 데이터 관리 [P2]

**As a** 사용자
**I want to** 등록한 관계 정보를 수정하거나 삭제하고 싶다
**So that** 정원을 깔끔하게 관리할 수 있다

### 수락 기준
- [ ] PlantDetail에서 "수정" 버튼으로 이름/관계유형/별명/식물종류/연락주기 편집
- [ ] "삭제" 버튼 + 확인 모달
- [ ] 삭제 시 관련 물주기/감사메모 기록도 함께 삭제
- [ ] 삭제 후 GardenHome으로 자동 이동

---

## US-14: 데모 데이터 체험 [P0]

**As a** 처음 방문한 사용자
**I want to** 데모 데이터로 앱 기능을 미리 체험하고 싶다
**So that** 직접 데이터를 입력하기 전에 앱의 가치를 느낄 수 있다

### 수락 기준
- [ ] 온보딩 완료 후 "데모 정원 구경하기" 옵션 제공
- [ ] 데모 데이터: 5명의 관계 (다양한 성장 단계 + 시들기 상태 포함)
- [ ] 데모 모드에서 모든 화면 탐색 가능
- [ ] 데모 데이터 기반 분석/정원사 타입 결과 표시
- [ ] "내 정원 시작하기" 버튼으로 데모→실제 전환 (데모 데이터 삭제)

---

## US-15: 설정 및 초기화 [P2]

**As a** 사용자
**I want to** 앱 설정을 관리하고 데이터를 초기화할 수 있다
**So that** 필요할 때 새로 시작할 수 있다

### 수락 기준
- [ ] 정원 이름 변경
- [ ] 데이터 전체 초기화 + 확인 모달 ("모든 식물이 사라집니다")
- [ ] 온보딩 다시 보기 옵션
- [ ] 앱 버전 정보 표시
