## 1. P0 User Story 검증

| US | 제목 | 상태 | 비고 |
|----|------|------|------|
| US-01 | 앱 컨셉 이해 | ✅ | "피부는 어제의 결과예요" 한 문장 전달, 3스텝 온보딩, "시작하기" 버튼 명확 |
| US-02 | 피부 타입 설정 | ✅ | 건성/복합성/지성/민감성 4가지, 복수 선택 가능, 건너뛰기 옵션 있음 |
| US-03 | 초기 제품 등록 | ✅ | 9개 카테고리 칩, 제품명 자유 입력, "나중에 추가할게요" = 시작하기 버튼으로 대체, 데모 모드 별도 버튼 |
| US-04 | 첫 밤 기록 | ⚠️ | 제품 선택 on/off 동작, 생활 변수 칩 동작, 저장 후 홈 복귀 정상. **"생리전"이 "생리"로 변경됨.** "+ 변수 추가" 버튼 없음 (US-16 연동) |
| US-05 | 첫 아침 기록 | ⚠️ | 점수 1~5 탭 동작, 라벨 표시(최악/별로/보통/좋아/꿀피부), 메모 optional. **키워드 불일치: 뾰루지, 번들, 매끈 누락. 유분기 추가됨, 붉음→붉은기로 변경** |
| US-06 | 어젯밤과 동일 | ✅ | 밤 기록 상단에 "어젯밤과 동일" 버튼 존재 |
| US-07 | 주간 피부 추이 | ✅ | 홈에 7일 라인 차트, 이번 주 평균 표시, 주요 키워드 표시 |
| US-08 | 기록 스트릭 | ✅ | 홈 하단에 "N일 연속 기록 중" 표시. 첫 로딩 시 보이지 않다가 기록 후 노출됨 (정상) |
| US-09 | 캘린더 월간 패턴 | ✅ | 월간 캘린더 뷰, 점수별 색상 표시, 이전/다음 달 화살표, 월 평균 + 기록률 표시 |
| US-10 | 제품별 분석 | ⚠️ | 제품 분석 탭 존재, 사용 횟수/사용 시 평균/미사용 시 평균/차이 표시. **"주의가 필요한 제품" 섹션과 "아직 판단하기 이른 제품" 섹션 미구현 — 전부 "피부에 좋은 제품"으로 표시** |
| US-11 | 변수별 분석 | ✅ | "피부에 안 좋은 변수" + "피부에 좋은 변수" 섹션 구분, 문장형 인사이트 포함 ("음주 다음날, 피부 점수가 평균 1.5점 낮아져요") |
| US-12 | 키워드 빈도 변화 | ❌ | 인사이트 화면에서 제품별 키워드 상세 (예: "세럼A 사용 시 '촉촉' 등장 70%") 기능 미구현 |
| US-13 | 밤 기록 알림 | ❌ | 설정에서 알림 설정 섹션 자체가 없음 (웹앱 한계일 수 있으나 UI 자체 미구현) |
| US-14 | 아침 기록 알림 | ❌ | US-13과 동일 |
| US-15 | 제품 추가/아카이브 | ⚠️ | 제품 관리 화면에서 추가 + 보관(아카이브) 가능. **제품별 평균 점수 미표시. 아카이브된 제품 섹션 펼침/접기 미확인** |
| US-16 | 커스텀 변수 추가 | ❌ | 밤 기록 화면과 설정 모두에서 "+ 변수 추가" 기능 없음 |
| US-17 | 데이터 내보내기 | ✅ | 설정에서 "데이터 내보내기 (JSON)" 버튼 존재 |
| US-18 | 인사이트 데이터 부족 안내 | ⚠️ | 데모 모드에서는 확인 불가. 코드상 구현 여부 미확인 |
| US-19 | 날짜별 상세 확인 | ⚠️ | 캘린더 날짜 탭 시 바텀시트 표시. 아침 피부(점수+키워드) + 전날 밤 기록(루틴) 표시. **"생활 변수" 미표시. "수정하기" 버튼 없음** |
| US-20 | 설정 관리 | ⚠️ | 이름/피부 타입/데모 모드/데이터 관리/기록 통계/앱 버전 표시. **알림 설정 없음, 생활 변수 관리 없음** |

## 2. 화면 존재 여부 (8개)

| # | 화면 | 상태 | 비고 |
|---|------|------|------|
| 1 | IntroScreen (온보딩) | ✅ | 3스텝 프로그레스, 컨셉→피부타입→제품등록. 이름 입력 필드 추가됨 (스펙에 없지만 개인화에 기여) |
| 2 | HomeScreen (홈) | ✅ | 인사말, 아침/밤 기록 카드, 주간 차트, 스트릭 |
| 3 | NightLogScreen (밤 기록) | ✅ | 풀스크린 모달, 제품 선택, 생활 변수, 저장 |
| 4 | MorningLogScreen (아침 기록) | ✅ | 풀스크린 모달, 점수 선택, 키워드, 메모 |
| 5 | CalendarScreen (캘린더) | ✅ | 월간 뷰, 색상 히트맵, 날짜 탭 바텀시트 |
| 6 | InsightScreen (인사이트) | ✅ | 제품/변수 분석 세그먼트, 비교 바 차트 |
| 7 | ProductListScreen (제품 관리) | ✅ | 카테고리별 목록, 추가/보관 |
| 8 | SettingsScreen (설정) | ✅ | 내 정보, 데모, 데이터, 통계, 앱 정보 |

**전체 8개 화면 존재 확인 완료.**

- **US-10**: 제품 분류 섹션 미구현 (긍정/부정/판단 이른 제품 구분)
- **US-15**: 제품별 평균 점수 미표시
- **US-19**: 바텀시트에서 생활 변수 미표시, 수정하기 버튼 없음
- **US-20**: 알림 설정/변수 관리 섹션 없음

- **US-14**: 아침 기록 알림 미구현
