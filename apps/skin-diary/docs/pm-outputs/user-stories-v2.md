# 피부 일지 V2 — User Stories

> V1 유저 스토리 US-01~US-20은 유지. V2 신규 스토리는 US-V2-01부터.
> 각 스토리에 Pillar 태그 (A: 깊이 있는 분석 / B: 습관 형성 / C: 소셜/공유)

---

## Pillar A: 깊이 있는 분석

### US-V2-01. 부위별 트러블 기록 [Pillar A] [P0]

**As a** 특정 부위에 반복적으로 트러블이 나는 유저
**I want to** 아침 기록에서 트러블이 난 부위를 선택하고 싶다
**So that** "턱선에만 계속 나는 건지, 전체적으로 안 좋은 건지" 구분할 수 있다

**수용 기준**:
- 아침 기록 화면에서 키워드 선택 아래에 "부위" 섹션 추가
- 부위 선택지: 턱선 / 이마 / 볼 / 코 / 전체 (칩 형태)
- 복수 선택 가능 (턱선 + 볼)
- 선택은 optional — 건너뛰어도 저장 가능
- 부위를 선택하면 해당 날짜 기록에 troubleAreas 필드 저장
- 기존 아침 기록 15초 목표 유지 (부위 선택은 추가 5초 이내)

**안티패턴**:
- 부위 선택을 필수로 강제
- 부위가 10개 이상 (너무 세밀)
- 얼굴 그림에 직접 탭하는 복잡한 UI

**페르소나 반응**:
- 서유진: "턱선! 맨날 여기만 올라와. 이걸 추적할 수 있다니 좋다"
- 김수빈: "전체적으로 건조하니까 '전체' 선택. 가끔 볼만 각질 날 때는 '볼' 선택"
- 이채은: "코 + 이마. T존 문제가 많아서 따로 볼 수 있으면 좋겠어"

### US-V2-02. 부위별 트러블 분석 확인 [Pillar A] [P0]

**As a** 2주 이상 부위를 기록한 유저
**I want to** 부위별 트러블 빈도 추이를 보고 싶다
**So that** 특정 부위의 문제가 개선되고 있는지 확인한다

**수용 기준**:
- 인사이트 화면에 "부위별 트러블" 섹션 추가
- 부위별 트러블 등장 빈도: "턱선: 최근 2주 7회 → 이전 2주 3회"
- 부위별 관련 제품/변수 연결: "턱선 트러블 날에 '밀가루' 변수 동시 등장 80%"
- 최소 데이터: 해당 부위 기록 5회 이상일 때만 분석 표시

**안티패턴**:
- 데이터 2-3건으로 결론 도출
- 의학적 조언 ("턱선 트러블은 호르몬 문제일 수 있습니다")

**페르소나 반응**:
- 서유진: "턱선 트러블이 생리전에 80% 동시 발생... 역시 호르몬이구나"
- 김수빈: "볼 각질이 에어컨 변수랑 연관 있네. 가습기 놓자"
- 이채은: "T존 번들이 줄어들고 있어! 나이아신아마이드가 효과 있나봐"

### US-V2-03. 3일 미니인사이트 확인 [Pillar A] [P0]

**As a** 기록을 막 시작한 유저 (3-6일차)
**I want to** 7일 기다리지 않고 초기 인사이트를 보고 싶다
**So that** 기록에 대한 보상감을 빨리 느끼고 계속 기록한다

**수용 기준**:
- 3일 기록 시 인사이트 탭에 "미니인사이트" 활성화
- 3일 미니인사이트 내용:
  - 3일간 피부 점수 추이 (상승/하락/유지)
  - 가장 많이 등장한 키워드 Top 3
  - 가장 많이 사용한 제품 Top 3
- 5일 기록 시 추가:
  - 제품 1-2개에 대한 초기 영향 분석 (⚠ "데이터가 아직 적어요" 경고 포함)
- 7일 이상: 기존 V1 인사이트 전체 활성화
- "미니인사이트"라는 라벨로 기존 인사이트와 구분

**안티패턴**:
- 3일 데이터로 확정적 결론 ("세럼A가 효과 있어요!")
- 7일 인사이트와 같은 수준의 분석 제공
- 미니인사이트 없이 "4일 더 기록하세요"만 표시

**페르소나 반응**:
- 서유진: "3일 만에 뭔가 나오니까 기록할 맛이 난다! '촉촉' 키워드가 가장 많대"
- 김수빈: "5일차인데 에센스 초기 분석이 보인다. 아직 데이터 부족하다고 하는 것도 솔직해서 좋아"
- 이채은: "미니인사이트 봤는데 벌써 재미있어. 7일 채우면 뭐가 더 나올지 기대돼"

### US-V2-04. 키워드 2주 트렌드 확인 [Pillar A] [P1]

**As a** 2주 이상 기록한 유저
**I want to** 피부 키워드의 변화 추이를 시각적으로 보고 싶다
**So that** "건조가 줄어들고 있는지, 트러블이 늘고 있는지" 파악한다

**수용 기준**:
- 인사이트 화면에 "키워드 트렌드" 섹션 추가
- 최근 14일 vs 이전 14일 키워드 빈도 비교
- 상위 5개 키워드만 표시, 나머지는 "더보기"
- 변화를 문장으로 해석: "건조 키워드가 57% 감소했어요"
- 막대 차트 또는 숫자 비교 형태

**안티패턴**:
- 11개 키워드 모두 나열
- 빈도 0-1회 키워드도 표시
- 차트만 보여주고 문장형 해석 없음

**페르소나 반응**:
- 서유진: "트러블이 지난 2주 대비 반으로 줄었어! 세럼 바꾼 게 맞았나봐"
- 김수빈: "건조 키워드가 줄고 촉촉이 늘었어. 에센스 효과인 듯"
- 이채은: "번들이 아직 많네... 클렌징을 바꿔볼까"

### US-V2-05. 제품 콤보 분석 확인 [Pillar A] [P1]

**As a** 여러 제품을 조합해서 사용하는 유저
**I want to** 제품 조합별 피부 점수를 비교하고 싶다
**So that** 나에게 맞는 최고의 조합을 찾는다

**수용 기준**:
- 인사이트 > 제품 분석 하단에 "콤보 분석" 섹션
- 자주 함께 사용한 제품 2개 조합의 평균 점수 표시
- 조건: 두 제품 함께 사용 3회 이상일 때만
- 상위 3개 콤보만 표시 (좋은 조합 + 안 좋은 조합)
- 예: "세럼A + 크림B: 함께 4.3점, 세럼A만 3.7점 (+0.6)"
- 3개 이상 동시 조합은 V2에서 제외 (YAGNI)

**안티패턴**:
- 모든 가능한 조합을 나열 (조합 폭발)
- 데이터 1-2건으로 콤보 분석 표시
- 3개 이상 제품 동시 조합 분석

**페르소나 반응**:
- 서유진: "세럼A + 수분크림이 최고 조합이래! 4.5점! 이 조합 유지해야지"
- 김수빈: "에센스 + 아이크림 조합이 생각보다 점수 높네. 둘 다 계속 쓰자"
- 이채은: "C세럼 + D크림이 최악 조합이야 -0.8... 같이 쓰면 안 되는 거였어"

### US-V2-06. 주간 리포트 확인 [Pillar A] [P1]

**As a** 꾸준히 기록하는 유저
**I want to** 매주 자동으로 한 주를 요약해주는 리포트를 받고 싶다
**So that** 매번 인사이트를 뒤지지 않아도 핵심을 파악한다

**수용 기준**:
- 매주 일요일 밤 자동 생성 (앱 진입 시 계산)
- 리포트 내용:
  - 이번 주 평균 점수 + 추이 (↑↓→)
  - 가장 영향력 있었던 제품 1개 (긍정)
  - 가장 영향력 있었던 변수 1개 (부정)
  - 키워드 변화 핵심 1-2개
  - 기록 일수 / 7일
- 홈 화면 상단 배너로 표시 ("이번 주 리포트가 도착했어요")
- 인사이트 탭에서 과거 리포트 목록 확인 가능
- 리포트 공유 버튼 (C1과 연결)

**안티패턴**:
- 매일 리포트 (과잉 정보)
- 리포트 내용이 인사이트 화면과 완전히 동일 (요약이 아님)
- 기록 3일 미만인 주에도 리포트 생성

**페르소나 반응**:
- 서유진: "일요일에 리포트 보면서 한 주 되돌아보는 거 좋다. 다음 주는 라면 줄여봐야지"
- 김수빈: "이번 주 평균 3.9, 지난주 3.4. 에센스 효과가 서서히 나오는 듯"
- 이채은: "리포트 공유해서 인스타에 올려야지. 나의 피부 기록 일지~"

### US-V2-07. 주간 차트에서 일별 상세 보기 [Pillar A] [P0]

**As a** 홈 화면에서 주간 차트를 보는 유저
**I want to** 특정 날짜 점을 탭해서 그날 기록을 바로 보고 싶다
**So that** "왜 그날 점수가 낮았지?" 바로 확인한다

**수용 기준**:
- 주간 차트의 날짜 점(dot) 탭 시 바텀시트 열림
- 바텀시트 내용: 캘린더 상세 보기와 동일 (밤 기록 + 아침 기록)
- 바텀시트 컴포넌트는 캘린더 화면과 공유 (재사용)
- 기록 없는 날은 "기록 없음" 표시

**안티패턴**:
- 탭해도 아무 반응 없음
- 별도 화면으로 이동 (맥락 유지 실패)
- 밤 기록만 보여주고 아침 기록은 생략

**페르소나 반응**:
- 서유진: "목요일이 2점이었는데, 탭하니까 바로 보이네. 밀가루 + 수면부족이었구나"
- 김수빈: "금요일 4점, 운동했던 날이네. 역시 운동이 최고야"
- 이채은: "차트에서 바로 확인할 수 있어서 캘린더로 안 가도 돼. 편하다"

## Pillar B: 습관 형성

### US-V2-08. 변수 프리셋 설정 [Pillar B] [P0]

**As a** 매일 반복되는 생활 변수가 있는 유저
**I want to** 특정 변수를 "항상 ON"으로 고정하고 싶다
**So that** 매번 같은 걸 탭하지 않아도 된다

**수용 기준**:
- 설정 > "기본 변수" 메뉴
- 변수 목록에서 핀 아이콘 토글로 프리셋 설정
- 프리셋 설정된 변수는 밤 기록 진입 시 자동으로 ON
- 밤 기록 화면에서 프리셋 변수 해제 가능 ("오늘은 아니에요")
- 프리셋 변수 칩에 작은 핀 아이콘으로 시각적 구분
- 프리셋 해제가 쉬워야 함 (탭 한 번으로 OFF)

**안티패턴**:
- 프리셋 해제가 어려움 (설정으로 돌아가야 해제)
- 프리셋 변수가 기록 화면에서 다른 위치에 표시
- "오늘은 아니에요" 확인 다이얼로그

**페르소나 반응**:
- 서유진: "앞머리 프리셋 설정! 맨날 있으니까. 잘라야 해제하면 되지"
- 김수빈: "야근은 자주 바뀌니까 프리셋 안 하고, 에어컨(커스텀)은 프리셋"
- 이채은: "마스크 프리셋. 학교 다닐 때는 매일 쓰니까"

### US-V2-09. 밤 기록 후 아침 유도 메시지 [Pillar B] [P0]

**As a** 밤 기록을 막 완료한 유저
**I want to** 내일 아침 기록을 자연스럽게 유도받고 싶다
**So that** 밤-아침 쌍 데이터를 빠뜨리지 않는다

**수용 기준**:
- 밤 기록 저장 직후: "내일 아침에 피부 상태도 기록해봐요" 토스트 메시지 (3초 후 자동 사라짐)
- 다음날 홈 화면 진입 시 (밤 기록 O + 아침 기록 X):
  - 상단에 "어젯밤 기록 완료! 오늘 아침 피부는 어때?" 배너
  - 배너 탭 시 아침 기록 화면으로 이동
- 3일 연속 아침 미기록 시: 인사이트 연결 안내 "아침 기록이 있어야 인사이트가 정확해져요"
- 메시지 톤: 부드러운 유도, 압박 없음

**안티패턴**:
- 밤 기록 후 팝업으로 "아침 기록도 하실 건가요?" 강제
- 매일 반복되는 동일한 메시지 (3일 후부터는 빈도 줄이기)
- "아침 기록 안 하면 스트릭이 끊겨요!" 압박

**페르소나 반응**:
- 서유진: "밤에 기록하고 보니까 '내일 아침에도' 알림이. 까먹을 뻔 했는데 다행"
- 김수빈: "아침에 홈 열자마자 배너 보이니까 자연스럽게 기록하게 돼"
- 이채은: "매일 같은 메시지면 지겨울 수 있는데, 자연스러워서 괜찮아"

### US-V2-10. 커스텀 변수 추가 [Pillar B] [P0]

**As a** 프리셋 10개 외에 다른 변수를 추적하고 싶은 유저
**I want to** 나만의 변수를 자유롭게 만들고 싶다
**So that** 내 생활에 맞는 정밀한 추적이 가능하다

**수용 기준**:
- 밤 기록 화면의 "변수 추가" 버튼 → 이름 입력 → 즉시 칩으로 생성
- 설정 > "변수 관리"에서도 추가/삭제/아카이브 가능
- 커스텀 변수도 프리셋 설정 가능 (US-V2-08과 연결)
- 인사이트에서 프리셋 변수와 동일하게 분석됨
- 삭제 시 과거 기록은 유지 (인사이트에서 분석 가능)
- 아카이브: 안 쓰는 변수는 목록에서 숨기되 기록 유지
- 커스텀 변수 개수 제한 없음

**안티패턴**:
- 커스텀 변수 3개 제한
- 추가 과정이 3단계 이상 (모달 → 카테고리 선택 → 이름 → 확인)
- 커스텀 변수가 프리셋과 다른 UI/분석 제공

**페르소나 반응**:
- 서유진: "카페인 추가! 커피 3잔 넘게 마신 날 피부가 별로인 것 같아서"
- 김수빈: "에어컨, 가습기 추가. 환경 변수를 따로 추적하고 싶었어"
- 이채은: "핫요가, 사우나, 수영 각각 추가. 운동 종류별로 비교하고 싶어"

### US-V2-11. 기록 마일스톤 축하 [Pillar B] [P1]

**As a** 꾸준히 기록해온 유저
**I want to** 연속 기록 마일스톤에서 성취감을 느끼고 싶다
**So that** 기록 습관을 계속 유지하는 동기가 된다

**수용 기준**:
- 마일스톤: 7일 / 14일 / 30일 / 60일 / 100일 연속 기록
- 달성 시 모달 카드로 축하 메시지 표시
- 메시지 예:
  - 7일: "일주일 연속 기록! 이제 본격적인 인사이트를 볼 수 있어요"
  - 14일: "2주 연속! 패턴이 보이기 시작해요"
  - 30일: "한 달 달성! 데이터가 충분해서 더 정확한 분석이 가능해요"
- 달성한 마일스톤은 설정 > "내 기록"에서 확인 가능
- 과도한 게이미피케이션 없음 — 배지 수집이 아니라 인사이트 연결

**안티패턴**:
- 매일 "1일 기록 달성!" 알림
- 배지/뱃지 수집 시스템 (앱 목적에서 벗어남)
- 마일스톤 못 채웠을 때 실패감 유발

**페르소나 반응**:
- 서유진: "14일 달성! 패턴이 보인다니까 더 하고 싶어진다"
- 김수빈: "30일 달성하면 더 정확한 분석이 나온다고? 좋아"
- 이채은: "100일 달성하면 캡쳐해서 자랑해야지 ㅋㅋ"

### US-V2-12. 기록률 시각화 [Pillar B] [P1]

**As a** 내가 얼마나 꾸준히 기록하는지 알고 싶은 유저
**I want to** 월별 기록률을 확인하고 싶다
**So that** 기록 습관의 일관성을 점검한다

**수용 기준**:
- 홈 화면 스트릭 영역 확장: "이번 달 기록률 87% (26/30일)"
- 설정 > "내 기록"에서 월별 기록률 히스토리
- 기록률 = 밤+아침 모두 기록한 날 / 해당 월 일수
- 시각적으로 % 바 또는 숫자로 표시 (과도한 그래프 없음)

**안티패턴**:
- 기록률 낮은 달에 빨간색 강조
- "지난 달보다 기록률이 떨어졌어요" 압박
- 시간별 기록 패턴 분석 (과도한 복잡도)

**페르소나 반응**:
- 서유진: "87%? 꽤 잘 하고 있네. 빠뜨린 날은 주말이구나"
- 김수빈: "93%. 빠뜨린 건 출장 갔을 때뿐. 뿌듯하다"
- 이채은: "이번 달 100% 도전해봐야지"

### US-V2-13. 미기록 넛지 [Pillar B] [P2]

**As a** 기록을 까먹기 쉬운 유저
**I want to** 기록을 빠뜨렸을 때 가볍게 알림을 받고 싶다
**So that** 스트릭이 끊어지기 전에 기록한다

**수용 기준**:
- 홈 화면 상단 배너: "어제 기록을 빠뜨렸어요. 오늘은 기록해볼까요?"
- 배너는 닫기(X) 가능
- 2일 연속 미기록 시: "다시 시작해볼까요?" (비압박)
- 3일 이상 미기록 시: 추가 넛지 없음 (관두기도 존중)
- 넛지 끄기 옵션: 설정에서 가능

**안티패턴**:
- 매시간 재알림
- "스트릭이 끊어졌어요!" 죄책감 유발
- 3일 이상 넛지 반복

**페르소나 반응**:
- 서유진: "어제 빠뜨렸구나. 오늘은 해야지. 부드럽게 알려줘서 괜찮다"
- 김수빈: "출장이었어서 빠뜨렸는데, 압박은 없어서 좋아"
- 이채은: "시험 기간에 3일 빠뜨렸는데 넛지가 그만 오더라. 센스 있네"

## Pillar C: 소셜/공유

### US-V2-14. 인사이트 카드 이미지 공유 [Pillar C] [P1]

**As a** 내 피부 분석 결과를 공유하고 싶은 유저
**I want to** 인사이트 카드를 예쁜 이미지로 만들어서 공유하고 싶다
**So that** 인스타 스토리나 카톡으로 보여줄 수 있다

**수용 기준**:
- 인사이트 카드(제품 분석, 변수 분석, 주간 리포트)에 "공유" 아이콘 버튼
- 탭 시 이미지 미리보기 화면 (ShareCardPreview)
- 이미지 스타일: dusty rose 배경 + Gowun Batang 제목 + 데이터 요약
- 이미지 비율 선택: 1:1 (인스타 피드) / 9:16 (인스타 스토리)
- "피부 일지" 워터마크 + 날짜 하단에 표시
- Web Share API로 공유 → 미지원 시 이미지 다운로드 폴백
- 개인 식별 정보(이름, 제품 브랜드) 포함 — 유저가 직접 공유하는 거라 OK

**안티패턴**:
- 공유 시 앱 다운로드 링크 강제 포함
- 워터마크가 데이터를 가리는 것
- 이미지 생성에 5초 이상 걸리는 것

**페르소나 반응**:
- 서유진: "세럼A 효과 +1.1점 카드 만들어서 친구한테 추천해야지"
- 김수빈: "주간 리포트 캡쳐 대신 깔끔한 이미지로 공유할 수 있어서 좋다"
- 이채은: "인스타 스토리에 딱이야! 9:16 비율 있어서 좋고, 디자인도 예쁘다"

### US-V2-15. 내 루틴 프로필 페이지 [Pillar C] [P2]

**As a** 내 루틴을 정리해서 보여주고 싶은 유저
**I want to** 내 스킨케어 루틴을 한 페이지로 요약한 프로필을 갖고 싶다
**So that** "내 루틴 뭐야?" 물어보는 사람에게 보여줄 수 있다

**수용 기준**:
- 홈 상단 프로필 아이콘에서 접근
- 프로필 내용:
  - 피부 타입
  - 현재 사용 제품 목록 (카테고리별)
  - 총 기록 일수
  - 최고 조합: 가장 점수 높았던 제품 조합 (14일+ 데이터 필요)
  - 주의 변수: 피부에 안 좋았던 Top 3 변수
- "프로필 공유" 버튼 → 이미지로 공유 (C1과 동일 방식)
- URL 공유 불가 (서버 없음)

**안티패턴**:
- SNS 프로필 연동
- 다른 유저의 프로필 검색/탐색
- 프로필에 사진 업로드

**페르소나 반응**:
- 서유진: "친구가 루틴 물어볼 때 이 페이지 보여주면 돼. 깔끔하다"
- 김수빈: "5스텝 루틴이 정리돼 있으니까 보기 좋아. 새 제품 추가할 때 참고하기도 좋고"
- 이채은: "최고 조합이랑 주의 변수 같이 보이니까 전문적인 느낌. 공유하고 싶다"

### US-V2-16. 인기 루틴 탐색 [Pillar C] [P2 — MVP 이후 재검토]

**As a** 다른 사람의 루틴이 궁금한 유저
**I want to** 비슷한 피부 타입의 인기 루틴을 보고 싶다
**So that** 루틴 구성에 참고한다

**수용 기준**:
- 인사이트 탭 하단 또는 별도 "탐색" 섹션
- **정적 데이터**: 피부 타입별 3-5개 샘플 루틴 (앱에 하드코딩)
- 각 루틴: 피부 타입 + 제품 목록(카테고리별) + 한 줄 설명
- "이 루틴 따라하기" → 해당 제품들을 내 제품 목록에 일괄 추가
- 실제 유저 데이터 수집/분석 없음 (서버 없으므로)

**안티패턴**:
- 실시간 랭킹 시스템
- 유저 리뷰/평점
- 광고성 제품 추천

**페르소나 반응**:
- 서유진: "복합성 피부 루틴이 3개 있네. 참고해볼까"
- 김수빈: "건성 루틴 중에 에센스 추천 있어서 제품 추가해봤어"
- 이채은: "지성 피부 루틴이 내 것과 비슷하네. 검증된 느낌"
