# 피부 일지 (Skin Diary) — Screen Flows & Wireframes

> 8개 화면 상세 명세 + ASCII 와이어프레임 + 컴포넌트 스펙

---

## 네비게이션 구조

```
┌─────────────────────────────────────────────┐
│                                             │
│  [제품관리]              [설정]              │  ← 홈 상단 아이콘
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│              메인 콘텐츠 영역                 │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│    [홈]        [캘린더]       [인사이트]       │  ← 하단 3탭
│                                             │
└─────────────────────────────────────────────┘

기록 화면 (NightLog, MorningLog):
  → 풀스크린 모달로 진입 (하단탭 숨김)
  → 기록 완료 시 홈으로 복귀

IntroScreen:
  → 첫 실행 시에만 표시
  → 완료 후 HomeScreen으로 이동
```

## Screen 1: IntroScreen (온보딩)

### 목적
첫 방문 유저에게 앱 컨셉을 전달하고, 최소 설정(피부 타입 + 제품 등록)을 완료시킨다.

### 와이어프레임

```
┌─────────────────────────────┐
│                             │
│     Step 1 / 3              │
│     ───────── (프로그레스)    │
│                             │
│                             │
│   피부는 어제의 결과예요       │
│                             │
│   어젯밤 바른 것 + 오늘 한 것 │
│   → 내일 아침 피부 상태       │
│                             │
│   매일 30초씩 기록하면         │
│   나에게 맞는 루틴을           │
│   데이터로 찾을 수 있어요      │
│                             │
│                             │
│   ┌─────────────────────┐   │
│   │      다음            │   │
│   └─────────────────────┘   │
│                             │
└─────────────────────────────┘

┌─────────────────────────────┐
│                             │
│     Step 2 / 3              │
│     ───────── (프로그레스)    │
│                             │
│                             │
│   피부 타입을 알려주세요       │
│                             │
│   ┌────────┐  ┌────────┐    │
│   │  건성   │  │ 복합성  │    │
│   └────────┘  └────────┘    │
│   ┌────────┐  ┌────────┐    │
│   │  지성   │  │ 민감성  │    │
│   └────────┘  └────────┘    │
│                             │
│   복수 선택 가능              │
│                             │
│   ┌─────────────────────┐   │
│   │      다음            │   │
│   └─────────────────────┘   │
│   건너뛰기                   │
│                             │
└─────────────────────────────┘

┌─────────────────────────────┐
│                             │
│     Step 3 / 3              │
│     ───────── (프로그레스)    │
│                             │
│                             │
│   지금 쓰고 있는 제품을        │
│   등록해볼까요?               │
│                             │
│   카테고리:                   │
│   [토너] [세럼] [크림] ...    │  ← 가로 스크롤 칩

│   ┌─────────────────────┐   │
│   │  제품명 입력          │   │
│   └─────────────────────┘   │
│   ┌───────────┐             │
│   │  + 추가    │             │
│   └───────────┘             │
│                             │
│   등록된 제품:                │
│   · 토너 — 라운드랩 독도토너   │
│   · 세럼 — 코스알엑스 세럼     │
│   · 크림 — 아이소이 수분크림   │
│                             │
│   ┌─────────────────────┐   │
│   │    시작하기            │   │
│   └─────────────────────┘   │
│   나중에 추가할게요            │
│                             │
└─────────────────────────────┘
```

### 컴포넌트 스펙

| 컴포넌트 | 타입 | 스펙 |
|---------|------|------|
| 프로그레스 바 | ProgressBar | 3스텝, 현재 스텝 하이라이트 |
| 피부 타입 칩 | ToggleChip | 복수 선택 가능, 선택 시 배경색 채움 |
| 카테고리 칩 | SelectChip | 단일 선택, 가로 스크롤 |
| 제품명 입력 | TextInput | placeholder "제품명을 입력하세요", 한 줄 |
| 추가 버튼 | SecondaryButton | 아웃라인 스타일 |
| 등록된 제품 리스트 | List | 카테고리 라벨 + 제품명, 스와이프로 삭제 |
| 다음/시작하기 | PrimaryButton | 솔리드 배경, 하단 고정 |
| 건너뛰기/나중에 | TextButton | 텍스트만, 하단 |

## Screen 2: HomeScreen (홈)

### 목적
오늘의 기록 상태를 한눈에 보여주고, 기록 CTA를 제공하며, 주간 추이를 확인한다.

### 와이어프레임

```
┌─────────────────────────────┐
│ [제품관리]         [설정]    │  ← 상단 아이콘
│                             │
│ 안녕, 오늘 피부는 어때?       │  ← 인사말 (기록 전)
│                             │
│ ┌─────────────────────────┐ │
│ │                         │ │
│ │  오늘 아침               │ │  ← 기록 상태 카드
│ │  아직 기록 전이에요        │ │
│ │                         │ │
│ │  ┌───────────────────┐  │ │
│ │  │  아침 피부 기록하기  │  │ │  ← Primary CTA
│ │  └───────────────────┘  │ │
│ │                         │ │
│ └─────────────────────────┘ │
│                             │
│ ┌─────────────────────────┐ │
│ │  오늘 밤 루틴            │ │  ← 밤 기록 CTA (아직이면)
│ │  ┌───────────────────┐  │ │
│ │  │  밤 루틴 기록하기   │  │ │  ← Secondary CTA
│ │  └───────────────────┘  │ │
│ └─────────────────────────┘ │
│                             │
│ 이번 주 피부                 │
│ ┌─────────────────────────┐ │
│ │  5 ·                    │ │
│ │  4 ·    ·         ·     │ │  ← 주간 라인 차트
│ │  3 ·  ·   ·   ·         │ │
│ │  2 ·         ·          │ │
│ │  1 ·                    │ │
│ │    월  화  수  목  금  토  │ │
│ │                         │ │
│ │  이번 주 평균 3.4          │ │
│ └─────────────────────────┘ │
│                             │
│ 14일 연속 기록 중             │  ← 스트릭
│                             │
├─────────────────────────────┤
│  [홈]     [캘린더]  [인사이트] │  ← 하단탭
└─────────────────────────────┘
```

**기록 완료 상태 (아침 기록 후):**

```
│ ┌─────────────────────────┐ │
│ │                         │ │
│ │  오늘 아침 피부           │ │
│ │                         │ │
│ │  4점 · 촉촉, 화사        │ │  ← 점수 + 키워드
│ │                         │ │
│ └─────────────────────────┘ │
```

### 컴포넌트 스펙

| 컴포넌트 | 타입 | 스펙 |
|---------|------|------|
| 상단 아이콘 | IconButton | 좌: 제품관리(리스트 아이콘), 우: 설정(기어 아이콘) |
| 인사말 | Heading | H2, semibold, 좌측 정렬 |
| 기록 상태 카드 | Card | 배경색 구분, 내부에 CTA 버튼 |
| 아침 기록 CTA | PrimaryButton | 기록 전에만 표시 |
| 밤 기록 CTA | SecondaryButton | 아웃라인 스타일 |
| 주간 차트 | LineChart | 7일, Y축 1-5, 점 + 선, 탭 가능 |
| 주간 평균 | Caption | 차트 하단, medium weight |
| 스트릭 | Badge | 아이콘 + 텍스트, 가벼운 스타일 |
| 하단탭 | BottomNav | 3탭, 현재 탭 하이라이트 |

### 상태 분기

| 상태 | 화면 변화 |
|------|----------|
| 아침 미기록 + 밤 미기록 | 아침 CTA(Primary) + 밤 CTA(Secondary) |
| 아침 기록 완료 + 밤 미기록 | 아침 결과 표시 + 밤 CTA(Primary으로 승격) |
| 아침 + 밤 모두 기록 | 양쪽 결과 표시, CTA 없음 |
| 첫 사용 (기록 0일) | 차트 대신 "기록을 시작하면 여기에 추이가 나와요" |

## Screen 3: NightLogScreen (밤 기록)

### 목적
오늘 밤 스킨케어 루틴과 오늘 하루 생활 변수를 기록한다.

### 와이어프레임

```
┌─────────────────────────────┐
│ [X 닫기]       밤 기록       │  ← 상단바
│                             │
│ ┌─────────────────────────┐ │
│ │  어젯밤과 동일            │ │  ← 빠른 입력 버튼
│ └─────────────────────────┘ │
│                             │
│ 오늘 밤 뭘 발랐어?           │  ← 섹션 제목
│                             │
│ 클렌징                       │  ← 카테고리 라벨
│ ┌──────┐                    │
│ │ 클렌폼 │                   │  ← 제품 칩 (off)
│ └──────┘                    │
│                             │
│ 토너                        │
│ ┌────────┐                  │
│ │ 독도토너 │  ← 선택됨 (on)   │
│ └────────┘                  │
│                             │
│ 세럼                        │
│ ┌────────┐  ┌────────┐     │
│ │ 세럼A   │  │ 세럼B   │     │  ← 복수 가능
│ └────────┘  └────────┘     │
│                             │
│ 크림                        │
│ ┌──────────┐                │
│ │ 수분크림   │                │
│ └──────────┘                │
│                             │
│ + 제품 추가                  │  ← 텍스트 버튼
│                             │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │  ← 구분선
│                             │
│ 오늘 하루 어땠어?             │  ← 섹션 제목
│                             │
│ ┌──────┐ ┌──────┐ ┌─────┐  │
│ │밀가루 │ │매운거 │ │ 음주 │  │  ← 생활 변수 칩
│ └──────┘ └──────┘ └─────┘  │
│ ┌──────┐ ┌──────┐ ┌──────┐ │
│ │ 운동  │ │수면부족│ │앞머리│ │
│ └──────┘ └──────┘ └──────┘ │
│ ┌───────┐ ┌─────┐ ┌──────┐ │
│ │스트레스│ │ 야근 │ │마스크 │ │
│ └───────┘ └─────┘ └──────┘ │
│ ┌──────┐                    │
│ │생리전 │  + 변수 추가        │
│ └──────┘                    │
│                             │
│ ┌─────────────────────────┐ │
│ │        저장하기           │ │  ← Primary, 하단 고정
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

### 컴포넌트 스펙

| 컴포넌트 | 타입 | 스펙 |
|---------|------|------|
| 상단바 | TopBar | 좌: X(닫기), 중: "밤 기록" |
| 어젯밤과 동일 | QuickButton | 아웃라인 스타일, 이전 기록 없으면 비활성 |
| 섹션 제목 | Heading | H3, medium weight, 좌측 정렬 |
| 카테고리 라벨 | Caption | 작은 텍스트, 회색, 좌측 정렬 |
| 제품 칩 | ToggleChip | off: 아웃라인, on: 배경 채움. 탭으로 토글 |
| 제품 추가 | TextButton | "+ 제품 추가", 탭 시 인라인 입력 또는 ProductAdd 모달 |
| 구분선 | Divider | 가벼운 점선 또는 여백 |
| 변수 칩 | ToggleChip | 제품 칩과 동일 스타일, 다른 색상 계열 가능 |
| 변수 추가 | TextButton | "+ 변수 추가", 탭 시 이름 입력 |
| 저장하기 | PrimaryButton | 하단 고정, 솔리드 배경 |

### 인터랙션

| 액션 | 결과 |
|------|------|
| 제품 칩 탭 | on/off 토글 (시각적 피드백: 배경색 변경) |
| "어젯밤과 동일" 탭 | 어젯밤 선택 상태 그대로 복원, 이후 개별 수정 가능 |
| "+ 제품 추가" 탭 | 카테고리 선택 + 이름 입력 인라인 폼 표시 |
| "저장하기" 탭 | 저장 → 홈 복귀 → "밤 기록 완료" 표시 |
| "X 닫기" 탭 | 변경사항 있으면 "저장하지 않고 나갈까요?" 확인 |

## Screen 4: MorningLogScreen (아침 기록)

### 목적
오늘 아침 피부 상태를 점수 + 키워드로 기록한다.

### 와이어프레임

```
┌─────────────────────────────┐
│ [X 닫기]      아침 기록      │  ← 상단바
│                             │
│                             │
│ 오늘 아침 피부는 어때?        │  ← 질문
│                             │
│                             │
│ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐
│ │ 1 │ │ 2 │ │ 3 │ │ 4 │ │ 5 │  ← 점수 선택
│ │   │ │   │ │   │ │   │ │   │
│ │최악│ │별로│ │보통│ │좋음│ │꿀 │
│ └───┘ └───┘ └───┘ └───┘ └───┘
│                             │
│                             │
│ 어떤 느낌이야?               │  ← 섹션 제목
│                             │
│ ┌──────┐ ┌──────┐ ┌──────┐ │
│ │ 촉촉  │ │ 건조  │ │트러블 │ │  ← 키워드 칩
│ └──────┘ └──────┘ └──────┘ │
│ ┌──────┐ ┌──────┐ ┌──────┐ │
│ │뾰루지 │ │ 각질  │ │ 번들  │ │
│ └──────┘ └──────┘ └──────┘ │
│ ┌──────┐ ┌──────┐ ┌──────┐ │
│ │ 칙칙  │ │ 화사  │ │ 붉음  │ │
│ └──────┘ └──────┘ └──────┘ │
│ ┌──────┐ ┌──────┐          │
│ │ 당김  │ │ 매끈  │          │
│ └──────┘ └──────┘          │
│                             │
│                             │
│ 메모 (선택)                  │
│ ┌─────────────────────────┐ │
│ │ 코 옆에 뾰루지 하나       │ │  ← 한 줄 입력
│ └─────────────────────────┘ │
│                             │
│                             │
│ ┌─────────────────────────┐ │
│ │        저장하기           │ │  ← Primary, 하단 고정
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

### 컴포넌트 스펙

| 컴포넌트 | 타입 | 스펙 |
|---------|------|------|
| 질문 텍스트 | Heading | H2, semibold |
| 점수 버튼 | ScoreButton | 5개 가로 배열. 미선택: 아웃라인. 선택: 채움 + 스케일 0.95→1 피드백. 라벨 아래 표시 |
| 섹션 제목 | Heading | H3, medium |
| 키워드 칩 | ToggleChip | 복수 선택. 긍정 키워드(촉촉/화사/매끈)와 부정 키워드 구분 가능 (색상 힌트) |
| 메모 입력 | TextInput | 한 줄, placeholder "특이사항이 있다면 적어봐", optional |
| 저장하기 | PrimaryButton | 점수 선택 전 비활성, 선택 후 활성 |

### 인터랙션

| 액션 | 결과 |
|------|------|
| 점수 탭 | 해당 점수 선택 (이전 선택 해제), 가벼운 스케일 애니메이션 |
| 키워드 칩 탭 | on/off 토글, 복수 선택 가능 |
| 저장 | 점수 필수, 키워드/메모 optional. 저장 → 홈 복귀 |
| 점수 미선택 시 저장 | 저장 버튼 비활성 상태 유지 |
