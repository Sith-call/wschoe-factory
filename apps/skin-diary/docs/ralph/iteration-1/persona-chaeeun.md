## 화면별 피드백

### 1. 온보딩 (OnboardingPage)

**좋은 점:**
- "피부는 어제의 결과예요" 카피가 직관적이고 공감됨. 매일 루틴 실험하는 나한테 딱 맞는 톤.
- 3단계 프로그레스바가 부담 없고, 피부 타입 복수 선택 가능한 점이 좋음 (나는 지성이지만 T존/U존 다르니까).
- 데모 데이터로 체험하기 버튼이 있어서 "일단 써보고 판단하자" 할 수 있음.

**아쉬운 점:**
- 피부 타입에 "지성"은 있는데 "모공" 같은 피부 고민을 선택하는 단계가 없음. 나는 피부 타입보다 모공, 블랙헤드, 피지가 핵심 고민인데 그걸 입력할 곳이 없다.
- Step 2에서 피부 타입을 선택 안 해도 "다음" 버튼이 비활성화되지 않음. 건너뛰기 버튼도 있는데 이러면 의도가 모호함.
- 제품 등록 시 카테고리 칩이 한 줄에 다 안 들어갈 수 있는데, 375px 뷰포트에서 "아이크림", "선크림", "마스크팩", "기타"가 잘리거나 줄바꿈될 때 가독성이 떨어질 수 있음.

### 2. 홈 (HomePage)

**좋은 점:**
- "안녕, 채은님 오늘 피부는 어때?" 인사가 친근함. 반말 톤이 또래 느낌.
- 아침/밤 기록 카드가 명확하게 분리되어 있고, CTA 버튼이 잘 보임.
- 주간 요약 차트 (WeeklySummary)가 한눈에 피부 추이를 보여주는 게 좋음. 이게 진짜 핵심 가치.
- 연속 기록 일수 (streak) 표시가 동기부여됨.

**아쉬운 점:**
- 상단 아이콘 두 개 (LeafIcon, SettingsIcon)가 뭘 의미하는지 처음엔 모르겠음. 잎사귀가 "제품 관리"라는 게 직관적이지 않음.
- 밤 루틴 기록 버튼이 아침을 안 했을 때 outline 스타일인데, 아침/밤 순서가 강제되는 느낌. 나는 밤에 먼저 기록하고 다음날 아침에 피부 상태를 기록하고 싶을 수도 있는데, 그 순서가 UX적으로 명확하지 않음.
- 오늘 기록을 수정하는 방법이 없어 보임. 기록 완료 후 카드에 "수정" 버튼이 없음.

### 3. 아침 기록 (MorningLogPage)

**좋은 점:**
- 점수 선택 UI가 원형으로 깔끔하고, 색상 그라데이션(빨강~초록)이 직관적.
- "어떤 느낌이야?" 키워드 칩이 실제 피부 상태를 잘 반영함 (촉촉, 번들, 트러블, 뾰루지 등).
- 메모가 선택 사항인 점이 기록 부담을 줄여줌.
- 저장하지 않고 나가려 할 때 confirm 대화상자가 뜨는 건 좋은 안전장치.

**아쉬운 점:**
- 키워드에 "모공" 관련 항목이 없음. "모공 벌어짐", "피지" 같은 키워드가 있으면 지성 피부인 나한테 더 유용할 텐데.
- 점수 5단계가 너무 단순할 수 있음. 부위별 점수 (이마, 볼, 턱)를 기록하고 싶은 니즈가 있음. 지성 피부는 T존과 U존이 완전히 다르니까.
- 메모 입력이 단일 `input`인데, 여러 줄 쓰고 싶을 때 `textarea`가 더 적합할 것 같음.

### 4. 밤 기록 (NightLogPage)

**좋은 점:**
- "어젯밤과 동일" 버튼이 진짜 편리함. 나는 루틴을 자주 바꾸지만, 가끔 같은 루틴을 며칠 쓸 때 이게 꿀기능.
- 제품 선택이 카테고리별로 그룹핑되어 있어서 찾기 쉬움.
- 생활 변수 (밀가루, 매운 음식, 음주, 운동 등)가 실제로 피부에 영향 주는 요인들이라 공감됨.
- 기록 중에 바로 "제품 추가" 할 수 있는 점이 좋음. 신제품 샀을 때 바로 등록 가능.

**아쉬운 점:**
- 생활 변수에 "카페인", "물 많이 마심", "새 제품 사용" 같은 항목이 없음. 특히 신제품 마니아인 나한테 "새로운 제품 처음 사용" 변수가 있으면 트러블 원인 파악에 도움될 텐데.
- 제품을 바른 순서를 기록할 수 없음. 토너 > 세럼 > 크림 순서와 세럼 > 토너 > 크림 순서는 결과가 다를 수 있는데, 현재는 선택만 할 뿐 순서 정보가 없음.
- 제품별 사용량(1펌프, 2펌프 등)이나 사용 부위 기록이 없음.

### 5. 캘린더 (CalendarPage)

**좋은 점:**
- 점수에 따른 배경색 + 하단 도트가 한눈에 피부 상태 추이를 보여줌. 한 달 전체 흐름이 보이는 게 유용함.
- 월별 평균 점수와 기록률 표시가 동기부여됨.
- 날짜 탭하면 바텀시트로 상세 정보가 나오는 UX가 자연스러움.

**아쉬운 점:**
- DayDetail 바텀시트에서 "전날 밤 기록"을 함께 보여주는 건 좋지만, "수정하기" 버튼이 CalendarPage에서 `onEdit`를 `setSelectedDate(null)`로 처리해서 실제로 수정 기능이 동작하지 않는 것 같음. 수정하면 그냥 바텀시트가 닫힘.
- 기록이 없는 날짜를 탭해도 아무 반응이 없는데, "이 날 기록하기" 같은 안내가 있으면 좋겠음.
- 기록률이 "N일"로만 표시되고 비율(%)이 없음. "21/31일 (68%)" 같은 형태가 더 동기부여됨.

### 6. 인사이트 (InsightPage)

**좋은 점:**
- 제품 분석 / 변수 분석 세그먼트 컨트롤이 깔끔함.
- "피부에 좋은 제품" / "주의가 필요한 제품" 분류가 직관적. 이게 내가 이 앱을 쓰는 핵심 이유.
- 바 차트로 사용 시 / 미사용 시 평균 점수 비교가 명확함.
- 변수 분석에서 "밀가루 다음날, 피부 점수가 평균 0.8점 낮아져요" 같은 문장형 인사이트가 이해하기 쉬움.
- 7일 이상 데이터가 쌓여야 분석을 보여주고, 그전에는 프로그레스바로 안내하는 게 적절함.

**아쉬운 점:**
- 기간 필터가 없음. 최근 2주 vs 최근 한 달 데이터로 비교해보고 싶을 때가 있을 텐데.
- 특정 제품과 특정 변수의 조합 분석이 없음. 예를 들어 "아이소이 세럼 + 운동한 날"의 평균 점수 같은 교차 분석.
- 제품 인사이트에서 해당 제품의 사용 이력 (날짜별)을 볼 수 없음. "왜 이 점수가 나왔지?" 궁금할 때 세부 데이터에 접근하기 어려움.

### 7. 제품 관리 (ProductListPage)

**좋은 점:**
- 카테고리별 그룹핑, 사용 횟수 표시, 보관/복원/삭제 관리가 잘 되어 있음.
- "보관" 개념이 있어서 안 쓰는 제품을 삭제하지 않고 보관할 수 있는 점이 좋음. 나중에 다시 꺼내 쓸 수도 있으니까.

**아쉬운 점:**
- 제품 사진을 등록할 수 없음. 제품이 많아지면 이름만으로는 뭐가 뭔지 헷갈릴 수 있음.
- 제품별 메모(성분, 구매처, 가격 등)를 남길 수 없음.
- 제품 정렬이 등록 순서 고정인데, 사용 빈도순이나 최근 사용순으로 정렬할 수 있으면 좋겠음.

### 8. 설정 (SettingsPage)

**좋은 점:**
- 데이터 내보내기(JSON) 기능이 있음. 백업이 가능하다는 안심감.
- 기록 통계 (총 기록 일수, 최장 스트릭, 사용 제품 수) 한눈에 보기 좋음.
- 데모 모드 토글이 편리함.
- 초기화 시 이중 confirm이 안전장치로 작동.

**아쉬운 점:**
- 이름이나 피부 타입을 수정하는 기능이 없음. 설정에서 "내 정보"를 보여주기만 하고 편집 불가.
- 데이터 가져오기(import) 기능이 없음. 내보내기만 있고 복원이 안 되면 반쪽짜리.
- 알림/리마인더 설정이 없음. 매일 기록하려면 밤 10시에 "오늘 뭘 발랐어?" 푸시가 오면 좋겠는데.

### 9. 전체 디자인 & UX

**좋은 점:**
- 전체적으로 따뜻한 톤 (sd-bg: #fdf8f4, sd-primary: #c2847a)이 스킨케어 앱 분위기에 잘 맞음.
- Gowun Batang 세리프 헤딩 + SUIT 본문 조합이 품격 있으면서 읽기 편함.
- 반말 톤 ("어때?", "뭘 발랐어?")이 또래 친구 같아서 부담 없음.
- 터치 영역 최소 44px 확보한 점, aria-label 적용한 점 등 접근성 고려가 보임.
- 탭바가 3개 (홈/캘린더/인사이트)로 단순해서 길을 잃지 않음.

**아쉬운 점:**
- 전체적으로 모노톤이라 화면 간 구분감이 약함. 모든 페이지가 비슷한 sd-primary-light 카드 스타일.
- 로딩 상태나 스켈레톤 UI가 없음. 데이터가 많아지면 체감될 수 있음.
- 다크 모드 미지원. 밤에 기록할 때 눈이 부실 수 있음.

## 개선 제안 (Top 5)

### 1. 피부 고민 & 커스텀 키워드 추가
온보딩에 "피부 고민" 선택 단계 추가 (모공, 블랙헤드, 피지, 주름, 색소침착 등). 아침 기록 키워드에도 "모공 벌어짐", "피지 과다" 같은 지성 피부 맞춤 키워드를 추가하거나, 사용자가 커스텀 키워드를 만들 수 있게 해야 함. 현재 11개 키워드로는 지성/건성/민감성 등 다양한 피부 타입의 세부 고민을 담기 어려움.

### 2. 부위별 피부 기록
전체 점수 하나로는 T존은 번들거리고 볼은 건조한 상황을 표현할 수 없음. 최소한 "이마/코", "볼", "턱" 3개 부위별 상태 키워드를 선택할 수 있으면 훨씬 유용한 데이터가 됨. 점수는 전체 하나로 유지하되, 키워드를 부위별로 분리하는 것만으로도 큰 차이.

### 3. 인사이트에 기간 필터 + 교차 분석
최근 1주 / 2주 / 1개월 / 전체 기간 필터를 추가해서, 시간에 따른 변화를 추적할 수 있게 해야 함. 또한 "제품 A + 변수 B" 조합의 교차 분석이 가능하면, 신제품을 실험하는 사용자에게 훨씬 깊이 있는 인사이트를 제공할 수 있음.

### 4. 과거 기록 수정 기능
캘린더에서 날짜를 탭했을 때 DayDetail 바텀시트의 "수정하기" 버튼이 실제로 동작하지 않음. 과거 기록을 수정하거나, 기록 못 한 날을 소급 기록할 수 있어야 함. 특히 아침에 까먹고 점심에 기록하거나, 어제 밤 루틴을 오늘 아침에 기록하는 상황이 빈번할 것.

### 5. 밤 기록 리마인더 & 다크 모드
밤에 기록하는 게 핵심인데, 알림이 없으면 까먹기 쉬움. 최소한 "매일 밤 N시에 알림" 설정이 필요함. 또한 밤에 기록할 때 밝은 화면은 눈이 부시므로 다크 모드 지원이 필요함. 이 두 가지가 합쳐져야 매일 기록하는 습관이 만들어짐.

## 만족도 점수

**62 / 100**

### 근거

**잘 되어 있는 것 (+):**
- 핵심 루프 (밤 루틴 기록 > 아침 피부 상태 > 인사이트)가 명확하고 30초 안에 기록 가능 (+15)
- 제품별/변수별 인사이트 분석이 "뭘 바르면 좋아지고 뭘 바르면 올라오는지"라는 내 핵심 니즈에 부합 (+15)
- 디자인 톤과 카피가 타겟 사용자(20대 여성)에게 잘 맞음 (+10)
- 데모 모드로 바로 체험 가능 (+5)
- 주간 차트, 캘린더 히트맵 등 시각화가 유용 (+7)

**부족한 것 (-):**
- "모공", "피지" 같은 지성 피부 핵심 키워드 부재. 내 피부 고민을 정확히 기록할 수 없음 (-10)
- 부위별 기록 불가. T존/U존이 완전히 다른 피부 상태를 하나의 점수로 퉁치는 건 데이터 정확도를 떨어뜨림 (-8)
- 과거 기록 수정이 안 됨 (수정 버튼이 실질적으로 미동작) (-5)
- 인사이트에 기간 필터/교차 분석 없음. 신제품 실험할 때 "이 제품 쓴 최근 2주만 보고 싶다"가 안 됨 (-5)
- 알림 없음. 매일 기록 습관 형성의 가장 큰 장벽 (-5)

**결론:**
핵심 컨셉과 기본 기능은 잘 갖춰져 있지만, 지성 피부 사용자로서 "내 고민에 맞는 세밀한 기록"이 어렵고, 신제품 실험 결과를 깊이 분석하기엔 인사이트 기능이 아직 얕음. 매일 기록하게 만드는 리마인더도 없어서 습관 형성이 어려울 것 같음. 기본기는 좋으니 위 5가지만 개선되면 80% 이상 만족할 것 같음.
