# Success Metrics — Argue Gym

## 북극성 지표
**세션 완주율** — 3개 반론 카드 모두에 재반박을 작성하고 리포트에 도달한 비율.

## MVP 성공 기준 (앱이 "만족스럽게 작동한다"고 판정하는 선)

### 기능 지표
- 첫 진입 → 첫 반론 카드 도달: 3초 이내
- 샘플 모드 체험 완주율: 60% 이상
- 평균 재반박 글자수: 80자 이상
- 같은 세션에서 "다시 도전" 탭률: 35% 이상

### 페르소나 만족도 (Ralph Loop 통과 기준)

| 페르소나 | 80% 기준이 구체적으로 의미하는 것 |
|---------|----------------------------------|
| P1 송지우 (실용) | 10분 안에 세션 완주 가능, 반론 카드 3장 중 2장 이상이 업무 맥락에 설득력 있음, 리포트 코멘트가 구체적 행동(예: "데이터 논거를 추가해보세요")을 제안 |
| P2 이태훈 (구조) | 근거 유형 라벨이 모든 카드에 명확, 재도전 시 겹치지 않는 카드, 로그에서 과거 세션을 다시 열어 재반박 복기 가능 |
| P3 박수연 (감성) | 라이트 테마의 타이포·여백이 "글 쓸 맛 나는" 수준, 저장/완료 액션에 명확한 시각 피드백, 스킵·건너뛰기에 압박 문구 없음 |

### 디자인 지표 (Design Visionary 7/10+ 기준)
- DESIGN_RULES.md 체크리스트 18항목 통과
- 기존 앱과 겹치지 않는 고유 팔레트 (라이트 테마, 따뜻한 중성색 + 1 액센트)
- 좌측 정렬 기본, 중앙 정렬은 리포트 숫자만
- 카드 네스팅 없음, 반론 카드는 border-left 액센트로 구분
- fadeInUp 없음, entrance는 즉시 표시 또는 스태거 슬라이드

### PRD 반영도 (PRD Verifier 90%+ 기준)
- P0 유저 스토리 US-1 ~ US-5 전부 구현
- 데모 모드 동작 확인 (localStorage, 가입 없음, 샘플 주장 프리필)
- 5개 카테고리 × 최소 12장 반론 덱 번들링 확인

## 비-지표 (의도적으로 추적하지 않음)
- 하루 방문 횟수, 연속 사용일수 — MVP는 훈련 도구이지 중독성 앱이 아님
- 공유/초대 수 — 혼자 훈련하는 앱의 정체성
