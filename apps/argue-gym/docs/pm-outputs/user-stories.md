# User Stories — Argue Gym

## P0 (MVP 필수)

### US-1. 첫 진입 즉시 체험
- **As a** 처음 들어온 유저
- **I want** 가입 없이 샘플 주장으로 바로 세션을 체험
- **So that** 앱의 가치를 3초 안에 이해할 수 있다
- 수락 조건: 랜딩에 "샘플로 체험하기" CTA, 누르면 "재택근무가 생산성을 올린다" 주장이 프리필되고 반론 카드 3장이 표시됨

### US-2. 내 주장 입력
- **As a** 훈련하고 싶은 유저
- **I want** 한 문장 주장 + 카테고리를 골라 세션을 시작
- **So that** 맞춤형 반론을 받을 수 있다
- 수락 조건: 주장 입력 필드(최대 120자) + 5개 카테고리 칩, "반론 받기" 버튼 활성화 조건 명확

### US-3. 반론 카드 3장을 본다
- **As a** 세션 중인 유저
- **I want** 내 주장에 대한 서로 다른 근거 유형의 반론 3개를 본다
- **So that** 다각도로 내 논리를 점검할 수 있다
- 수락 조건: 카드에 반론 문장 + 근거 유형 라벨(데이터/감정/원칙/사례) 표시, 3장 모두 다른 유형

### US-4. 재반박 작성
- **As a** 세션 중인 유저
- **I want** 각 카드 아래에 재반박을 직접 텍스트로 쓴다
- **So that** 내 논리를 실제로 단련한다
- 수락 조건: 카드별 textarea, 자동 저장(입력 300ms debounce), 글자 수 카운터

### US-5. 세션 리포트
- **As a** 세션을 끝낸 유저
- **I want** 내 재반박의 특징(논거 유형 분포, 회피한 카드 수, 총 글자)을 시각적으로 본다
- **So that** 내 논쟁 패턴을 인지할 수 있다
- 수락 조건: 바 형태 유형 분포 + 숫자 지표 3개 + 한 줄 코멘트("감정 논거에 치우쳐 있어요" 등 규칙 기반)

## P1

### US-6. 로그 아카이브
- **As a** 반복 유저
- **I want** 지난 세션 리스트를 보고 특정 주장을 다시 열어본다
- **So that** 같은 주장을 재도전하거나 성장 궤적을 본다
- 수락 조건: 날짜별 정렬, 주장 프리뷰, 탭 시 리포트 재표시

### US-7. 같은 주장 재도전
- **As a** 한 번 훈련한 유저
- **I want** 같은 주장으로 다른 반론 카드 3장을 받는다
- **So that** 다른 각도로 방어력을 키운다
- 수락 조건: 리포트 화면 "다시 도전" 버튼, 이전과 겹치지 않는 카드 3장 제공 (덱 내 제외)

### US-8. 카드 건너뛰기
- **As a** 애매한 반론을 받은 유저
- **I want** 특정 카드를 "스킵"할 수 있다
- **So that** 리듬을 깨지 않고 훈련을 이어간다
- 수락 조건: 카드당 "스킵" 버튼, 스킵 시 리포트에서 회피 지표에 카운트됨
