# Argue Gym — PRD

## 한 줄 정의
내 주장을 적으면, 앱이 반대 입장 3가지를 던지고, 유저가 재반박을 적으며 논리 근육을 단련하는 "혼자 하는 토론 연습장".

## 문제
- 회의/발표/면접/글쓰기에서 "반대 의견이 나오면 머리가 하얘진다"는 경험은 직장인·학생의 공통 고민이다.
- 기존 토론 연습은 파트너가 필요하거나(리얼 토론), 정답이 정해진 퀴즈(밸런스게임)에 머무른다.
- "내 주장"을 입력 → "예상되는 반론"을 구조화해서 보여주고 → "내 재반박"을 글로 써보는 경량 훈련 도구가 없다.

## 해결
1. 주장 입력 (한 문장)
2. 카테고리 선택 (일·관계·돈·사회·라이프스타일)
3. 앱이 카테고리별 반박 템플릿 덱에서 3가지 반대 입장 카드를 제시 (근거 유형: 데이터·감정·원칙·사례)
4. 유저가 각 카드에 재반박을 텍스트로 입력
5. 세션 종료 시 "내 논리의 강점/약점" 리포트 (텍스트 길이, 사용한 논거 유형 다양성, 회피한 반론 개수)
6. 주장·재반박 세션을 기록으로 쌓아 "내 논쟁 근육 로그"로 복기

## 타겟
- 주 타겟: 회의에서 자기 의견 방어가 어려운 2~5년차 직장인
- 보조 타겟: 논술/면접 대비 학생, 에세이 쓰는 사람
- 비타겟: 실시간 팀 토론, 경쟁적 디베이트

## 핵심 기능 (MVP)
1. **세션 시작** — 주장 + 카테고리 입력
2. **반론 카드 3장** — 덱에서 휴리스틱 선택 (카테고리 + 근거 유형 다양성 보장)
3. **재반박 에디터** — 카드별 텍스트 입력, 자동저장
4. **세션 리포트** — 논거 유형 분포, 회피 지표, 짧은 피드백 문장
5. **로그 아카이브** — 과거 세션 리스트, 주장으로 검색, 재도전 버튼
6. **데모 모드** — 앱 첫 진입 시 샘플 주장 "재택근무가 생산성을 올린다"로 즉시 체험 가능 (DB 없이 localStorage만 사용)

## 비기능 요구사항
- 데모 모드 필수: 가입/로그인 없음, localStorage 기반, 첫 진입 3초 안에 첫 반론 카드 도달
- 반론 덱: JSON 정적 번들, 카테고리당 최소 12장 × 5카테고리 = 60장
- 오프라인 동작 가능
- 모바일 뷰포트 우선 (375x812)

## 성공 지표
- 첫 세션 완주율(3개 재반박 모두 작성) ≥ 60%
- 평균 재반박 글자 수 ≥ 80자
- 두 번째 세션 진입률(같은 세션에서 "다시" 버튼 탭) ≥ 35%
- 페르소나 만족도 80% 이상 (Ralph 루프 기준)

## 범위 밖 (Non-goals)
- 실시간 멀티플레이어
- AI/LLM 기반 반론 생성 (MVP는 정적 카드 덱)
- 음성 입력
- 소셜 공유 기능
- 사용자 계정/동기화
