# PRD: 하루 회고 (Daily Reflect)

## 제품 비전
"오늘 하루, 나는 어땠을까?" — 퇴근 후 3분 회고로 자기 이해를 깊게 하는 앱.

## 타겟 세그먼트
- 한국 20-35세 직장인
- 자기 성장에 관심 있지만, 일기 쓰기는 부담스러운 사람
- 하루를 "그냥 흘려보내는" 느낌이 싫은 사람

## 핵심 가치
1. **3분 회고**: 길게 쓰지 않아도 됨. 선택+한줄이면 충분
2. **패턴 발견**: 2주 이상 쌓이면 나만의 패턴이 보인다
3. **자기 이해**: 감정, 에너지, 성취의 상관관계를 시각화

## 핵심 기능

### F1: 오늘의 회고 (3분 플로우)
1. 감정 선택 (6개 중 택 1 + 강도 슬라이더)
2. 에너지 레벨 (1-5 배터리)
3. 오늘의 하이라이트 (한줄 + 카테고리)
4. 감사한 것 하나 (선택)

### F2: 회고 갤러리
- 캘린더 뷰로 과거 회고 확인
- 감정 색상으로 한눈에 패턴 확인
- 월별 감정 분포 차트

### F3: 인사이트
- 주간/월간 패턴 분석
- "당신은 월요일에 에너지가 가장 낮아요" 같은 인사이트
- 감정-에너지 상관관계 그래프

### F4: 프로필/설정
- 알림 시간 설정 (퇴근 시간 기반)
- 회고 연속 기록 스트릭
- 총 회고 수, 가장 많은 감정

## 디자인 방향
- 따뜻하고 차분한 톤 (퇴근 후 편안함)
- 밤/저녁 느낌 — 어두운 배경에 따뜻한 포인트
- 미니멀하지만 풍성한 비주얼 (일러스트, 그라데이션)
- 430px 모바일 우선

## 기술 스택
- React + Vite + TypeScript
- Tailwind CSS (CDN)
- localStorage (DB 없음, 데모 모드 포함)
- 차트: SVG 직접 구현 (외부 라이브러리 없음)

## 성공 지표
- Time to First Value < 30초
- 3분 이내 회고 완료
- 유저 만족도 80%+ (Ralph Loop)
