# User Stories: 하루 회고

## US-01: 첫 방문 온보딩
**As a** 처음 앱을 연 유저
**I want to** 앱이 뭘 하는지 3초 안에 이해하고
**So that** 바로 첫 회고를 시작할 수 있다

### Acceptance Criteria
- AC-1: 앱 열면 한눈에 "하루 회고" 앱임을 알 수 있는 인트로 화면
- AC-2: CTA 버튼 "오늘의 회고 시작" 클릭 → 회고 플로우 시작
- AC-3: 첫 방문 시에만 인트로 표시, 이후에는 바로 홈으로

## US-02: 감정 선택
**As a** 회고하는 유저
**I want to** 오늘의 감정을 빠르게 선택하고
**So that** 길게 고민하지 않고 기록할 수 있다

### Acceptance Criteria
- AC-1: 6가지 감정 (행복, 평온, 감사, 피곤, 불안, 우울) 카드로 선택
- AC-2: 감정 선택 시 카드에 시각적 피드백 (색상 변화, 체크 표시)
- AC-3: 강도 슬라이더 (1-5) 표시
- AC-4: 감정 선택 없이는 다음 단계 진행 불가

## US-03: 에너지 레벨
**As a** 회고하는 유저
**I want to** 오늘의 에너지 레벨을 기록하고
**So that** 에너지 패턴을 추적할 수 있다

### Acceptance Criteria
- AC-1: 1-5 배터리 아이콘으로 에너지 선택
- AC-2: 선택 시 배터리 채워지는 시각적 피드백
- AC-3: 기본값 없음 — 반드시 선택해야 진행

## US-04: 하이라이트 기록
**As a** 회고하는 유저
**I want to** 오늘의 하이라이트를 한줄로 적고
**So that** 하루의 핵심을 기억할 수 있다

### Acceptance Criteria
- AC-1: 카테고리 선택 (업무, 사람, 자기계발, 취미, 건강)
- AC-2: 한줄 텍스트 입력 (최대 50자)
- AC-3: 감사한 것 한가지 (선택, 건너뛰기 가능)
- AC-4: 완료 버튼 → 회고 저장 + 완료 화면

## US-05: 회고 완료
**As a** 회고를 마친 유저
**I want to** 오늘의 회고 요약을 보고
**So that** 기록이 잘 됐다는 만족감을 느낀다

### Acceptance Criteria
- AC-1: 오늘 기록한 감정, 에너지, 하이라이트 요약 카드
- AC-2: 연속 기록 스트릭 표시
- AC-3: "홈으로" 버튼

## US-06: 홈 화면
**As a** 재방문 유저
**I want to** 최근 회고 요약과 스트릭을 한눈에 보고
**So that** 기록 동기가 유지된다

### Acceptance Criteria
- AC-1: 오늘 회고 여부 표시 (완료/미완료)
- AC-2: 최근 7일 감정 미니 캘린더 (감정 색상으로)
- AC-3: 연속 기록 스트릭
- AC-4: "오늘의 회고" CTA 버튼 (미완료 시)
- AC-5: 하단 탭바 (홈, 갤러리, 인사이트, 프로필)

## US-07: 회고 갤러리
**As a** 과거를 돌아보는 유저
**I want to** 캘린더 형태로 과거 회고를 보고
**So that** 감정 패턴을 발견할 수 있다

### Acceptance Criteria
- AC-1: 월간 캘린더 뷰 — 각 날짜에 감정 색상 도트
- AC-2: 날짜 탭 → 해당 날 회고 상세 보기
- AC-3: 월별 감정 분포 차트 (간단한 바 차트)
- AC-4: 이전/다음 달 네비게이션

## US-08: 인사이트
**As a** 자기 이해를 원하는 유저
**I want to** 내 감정/에너지 패턴을 분석해서 보고
**So that** 나에 대해 새로운 것을 발견한다

### Acceptance Criteria
- AC-1: 주간 감정+에너지 트렌드 차트
- AC-2: 요일별 평균 에너지 (막대 그래프)
- AC-3: 텍스트 인사이트 1-3개 (예: "화요일에 가장 행복해요")
- AC-4: 데이터 부족 시 "5일 더 기록하면 인사이트가 열려요" 안내

## US-09: 프로필/설정
**As a** 앱 사용자
**I want to** 알림 시간과 설정을 관리하고
**So that** 내 라이프스타일에 맞게 쓸 수 있다

### Acceptance Criteria
- AC-1: 알림 시간 설정 (기본 19:00)
- AC-2: 총 회고 수, 연속 스트릭, 가장 많은 감정 통계
- AC-3: 데이터 초기화 옵션
