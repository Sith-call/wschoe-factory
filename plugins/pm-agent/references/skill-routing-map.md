# PM Skill Routing Map

This reference maps each product lifecycle phase to the specific pm-* skills that should be invoked.

## Phase 1: Ideation & Brainstorming

| Trigger | Skill | Output |
|---------|-------|--------|
| New product idea generation | `pm-product-discovery:brainstorm-ideas-new` | Feature ideas from PM/Designer/Engineer perspectives |
| Existing product feature ideas | `pm-product-discovery:brainstorm-ideas-existing` | Multi-perspective feature ideas |
| Product vision definition | `pm-product-strategy:product-vision` | Inspiring vision statement |
| Value proposition design | `pm-product-strategy:value-proposition` | 6-part JTBD value proposition |

## Phase 2: Market Research & Validation

| Trigger | Skill | Output |
|---------|-------|--------|
| Market size estimation | `pm-market-research:market-sizing` | TAM/SAM/SOM analysis |
| Customer segment identification | `pm-market-research:market-segments` | 3-5 segments with JTBD |
| Competitor landscape analysis | `pm-market-research:competitor-analysis` | Strengths/weaknesses/differentiation |
| SWOT analysis | `pm-product-strategy:swot-analysis` | Actionable SWOT |
| PESTLE macro analysis | `pm-product-strategy:pestle-analysis` | External factor assessment |
| Porter's Five Forces | `pm-product-strategy:porters-five-forces` | Industry dynamics analysis |

## Phase 3: Strategy & Business Model

| Trigger | Skill | Output |
|---------|-------|--------|
| Business model design | `pm-product-strategy:business-model` | Business Model Canvas |
| Lean Canvas | `pm-product-strategy:lean-canvas` | 9-block Lean Canvas |
| Startup Canvas | `pm-product-strategy:startup-canvas` | Combined strategy + business model |
| Product strategy | `pm-product-strategy:product-strategy` | 9-section Strategy Canvas |
| Pricing strategy | `pm-product-strategy:pricing-strategy` | Pricing models and analysis |
| Monetization strategy | `pm-product-strategy:monetization-strategy` | 3-5 revenue model options |
| Ansoff growth matrix | `pm-product-strategy:ansoff-matrix` | Growth strategy mapping |

## Phase 4: Discovery & Validation

| Trigger | Skill | Output |
|---------|-------|--------|
| Assumption identification (new) | `pm-product-discovery:identify-assumptions-new` | Risky assumptions across 8 categories |
| Assumption identification (existing) | `pm-product-discovery:identify-assumptions-existing` | VUFV risk assumptions |
| Assumption prioritization | `pm-product-discovery:prioritize-assumptions` | Impact x Risk matrix |
| Experiment design (new product) | `pm-product-discovery:brainstorm-experiments-new` | Pretotypes and lean experiments |
| Experiment design (existing product) | `pm-product-discovery:brainstorm-experiments-existing` | A/B tests, prototypes, spikes |
| Interview script creation | `pm-product-discovery:interview-script` | JTBD interview guide |
| Interview summarization | `pm-product-discovery:summarize-interview` | Structured interview insights |
| User persona creation | `pm-market-research:user-personas` | 3 personas with JTBD |
| Customer journey mapping | `pm-market-research:customer-journey-map` | End-to-end journey map |
| Opportunity Solution Tree | `pm-product-discovery:opportunity-solution-tree` | OST framework |

## Phase 5: Planning & Execution

| Trigger | Skill | Output |
|---------|-------|--------|
| PRD creation | `pm-execution:create-prd` | 8-section PRD |
| OKR brainstorming | `pm-execution:brainstorm-okrs` | Team-level OKRs |
| User story writing | `pm-execution:user-stories` | Stories with acceptance criteria |
| Job story writing | `pm-execution:job-stories` | JTBD-format stories |
| WWA backlog items | `pm-execution:wwas` | Why-What-Acceptance items |
| Feature prioritization | `pm-product-discovery:prioritize-features` | Ranked feature list |
| Feature request triage | `pm-product-discovery:analyze-feature-requests` | Themed and prioritized requests |
| Sprint planning | `pm-execution:sprint-plan` | Sprint with capacity and dependencies |
| Stakeholder mapping | `pm-execution:stakeholder-map` | Power/Interest grid |
| Test scenarios | `pm-execution:test-scenarios` | Comprehensive test cases |
| Pre-mortem | `pm-execution:pre-mortem` | Risk analysis with Tigers/Paper Tigers/Elephants |
| Metrics dashboard design | `pm-product-discovery:metrics-dashboard` | Dashboard with KPIs and alerts |
| North Star Metric | `pm-marketing-growth:north-star-metric` | NSM with input metrics |

## Phase 6: Go-to-Market

| Trigger | Skill | Output |
|---------|-------|--------|
| Beachhead segment selection | `pm-go-to-market:beachhead-segment` | First target segment |
| ICP definition | `pm-go-to-market:ideal-customer-profile` | Ideal Customer Profile |
| GTM strategy | `pm-go-to-market:gtm-strategy` | Channels, messaging, timeline |
| GTM motions | `pm-go-to-market:gtm-motions` | 7 motion types analysis |
| Growth loops | `pm-go-to-market:growth-loops` | 5 loop types evaluation |
| Competitive battlecard | `pm-go-to-market:competitive-battlecard` | Sales-ready battlecard |
| Product positioning | `pm-marketing-growth:positioning-ideas` | Differentiated positioning |
| Product naming | `pm-marketing-growth:product-name` | 5 name candidates |
| Marketing ideas | `pm-marketing-growth:marketing-ideas` | 5 creative campaign ideas |
| Value prop statements | `pm-marketing-growth:value-prop-statements` | Marketing/sales/onboarding copy |

## Phase 7: Launch & Iterate

| Trigger | Skill | Output |
|---------|-------|--------|
| Release notes | `pm-execution:release-notes` | User-facing release notes |
| Outcome roadmap | `pm-execution:outcome-roadmap` | Outcome-focused roadmap |
| Sprint retrospective | `pm-execution:retro` | Retro with action items |
| Meeting notes | `pm-execution:summarize-meeting` | Structured meeting summary |
| A/B test analysis | `pm-data-analytics:ab-test-analysis` | Statistical analysis + ship decision |
| Cohort analysis | `pm-data-analytics:cohort-analysis` | Retention and adoption trends |
| SQL query generation | `pm-data-analytics:sql-queries` | Database queries |
| Sentiment analysis | `pm-market-research:sentiment-analysis` | Feedback sentiment insights |
| User segmentation | `pm-market-research:user-segmentation` | Behavior-based segments |
